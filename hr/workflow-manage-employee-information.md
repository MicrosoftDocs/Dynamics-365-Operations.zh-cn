# <a name="use-workflows-to-manage-employee-information"></a>使用工作流管理员工信息

本主题说明如何可以使用工作流。可以管理功能由人力资源员工信息。 例如，您可以将工作流与职位和配置启动创建审核工作流，当员工更改其记录时。

人力资源工作流的功能为人力资源管理活动提供许多工作流。 此外，许多选项可用，以便您可以修改特定工作流和将这些上课时间与某报告层次结构。 工作流可以帮助管理针对员工信息在内的多种标准类型的更改。 您可以将工作流与该职位。 然后，如果员工改换其"员工"记录中，要求审核的工作流开始，在新的信息保存。 工作流以下列信息类型预定义高效地帮助您管理更改以及维护您的准确员工的数据：

-   标识号
-   课程
-   教育程度
-   图像
-   借出物品
-   专业经验
-   项目经验
-   技能
-   诚信职位
-   人力资源操作
-   课程登记

在转移或终止雇用员工，后，工作流才能包括审核流程。 此类文档，可以修改工作流作为一部分，或操作的条款可以定义。 在审核流程完成时，文档或操作，完成，并且工作流移到个最终审核步骤。

## <a name="associate-a-workflow-with-a-position-hierarchy"></a>关联工作流与职位层次结构
您可以将工作流与您正在配置的所有层次结构。 例如，如果一职位，与报告层次结构，您可以发送配置特定项目的支出与潜在客户{{而项目不是：instead_of}}员工经理与该职位的工作流。 若要创建新工作流或修改现有工作流，在**人力资源工作流**页，单击新** **。 选择列表中的工作流来启动工作流设计器。 设计器可以使用创建新工作流或更改以现有工作流中的步骤。 当你更改现有工作流时，更改保存为新版本。 因此，如果需要，您可以始终返回以前版本。
