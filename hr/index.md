# <a name="welcome-to-human-resources"></a>欢迎使用人力资源

[!include[banner](includes/banner.md)]

人力资源简化许多例行记录保持任务，并自动化与为组织进行人员配备相关的一系列流程。 它还提供了一个框架，用于人力资源人员管理监督领域。 这些领域包括员工招聘和保留、福利管理、培训、绩效考核和变更管理。

您可以使用“人力资源”模块完成以下任务：

> + 管理组织结构。
> + 维护工作人员从雇用到退休的全面信息。
> + 定义和管理福利计划、招收工作人员、分配依赖型保险和指定受益人。
> + 建立和监督缺勤政策。
> + 实施和跟踪基于配置文件的时间管理，生成导出到工资单系统的报酬信息。
> + 管理工作人员能力。
> + 考核绩效和实现工作人员目标。
> + 设置、提供并分析包括议程、会话和跟踪的培训课程。
> + 招聘工作人员和跟踪申请人。

<a name="develop-a-human-resources-strategy"></a>制定人力资源战略
---------------------------------------------------------

与人力资源部工作时，将决定如何组织您的组织使用元素，如部门、 工作和职位。 这些功能是将配置人力资源中的基础元素。 单个员工被分配到与工作关联的职位。

-   [设置特定于公司的 HR 参数](set-up-company-specific-hr-parameters.md)
-   [设置法人之间的 HR 参数](set-up-hr-parameters-across-legal-entities.md) 

## <a name="recruit-hire-and-motivate-employees"></a>招聘、雇用和激励员工

招聘项目管理空缺职位广告使用的内容，并可可帮助您管理职位空缺应用程序。 您可以使用它们来跟踪对特定招聘项目的职位空缺的回应，或跟踪特定的申请人，更新空缺职位的状态。 此外，还可以雇用一个申请人或管理“大批雇用项目”，以便于雇用多个工作人员，例如，为满足季节性的业务需求。

-   [Manage a recruiting process](manage-recruiting-process.md)
-   [Mass hire projects](mass-hire-projects.md) 

雇佣员工后，您可以有效公平地设置计划来管理薪酬，并管理您的组织出借给员工的物料，如计算机或电话。 您可以创建固定和可变薪酬计划，并定义应用薪酬计划以满足该计划的条件的规则。

-   [薪酬计划](compensation-plans.md)
-   [创建固定薪酬计划](create-fixed-compensation-plans.md)
-   [创建可变薪酬计划](create-variable-compensation-plans.md)

## <a name="develop-and-train-employees"></a>培养和培训员工

以帮助您达到其职业目标而提供关键业务需要您的员工可以设置目标、 创建绩效考核和跟踪反馈，以帮助员工发展所需的技能，您还可以设置教师、 课程类型、 课程、 课程说明、 议程、 轨道，以及会话之前将指导员分配给课程，或注册人的课程。 教师必须已经以工作人员、申请人或联系人身份存在。

-   [性能管理](performance-management-overview.md)
-   [协调员工技能和业务需要](skills.md)
-   [Administer training through courses](courses.md)
