# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>通过从现有配方复制来创建配方

[!include[task guide banner](../../includes/task-guide-banner.md)]

该过程显示如何通过复制现有配方，快速创建新的配方。 您可以使用演示数据公司 USMF、USP2 或 USPI 创建该过程。


## <a name="create-a-new-formula"></a>创建新配方
1. 转到“配方”。
2. 单击“新建”。
3. 在“配方”字段中，键入一个值。
4. 在“名称”字段中，键入一个值。
5. 在“站点”字段中，输入或选择一个值。
6. 在“物料组”字段中，输入或选择一个值。
7. 单击“保存”。

## <a name="copy-from-an-existing-formula"></a>从现有配方中复制
1. 单击“复制”。
2. 在“物料编号”字段中，输入或选择一个值。
3. 在“配方版本”字段中，输入或选择一个值。
4. 单击“确定”。

## <a name="revise-the-copied-formula"></a>修改复制的配方
1. 在列表中，找到并选择所需记录。
2. 单击“删除”。
3. 单击“是”。

## <a name="approve-formula"></a>审核配方
1. 单击“审核配方”。
2. 在“已审核”字段中，输入或选择一个值。
3. 单击“确定”。
4. 关闭该页面。
