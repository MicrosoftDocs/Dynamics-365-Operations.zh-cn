# <a name="commerce-essentials-overview"></a>商务精英概览

此文章提供有关制定的安全港 Essentials 的信息，这是 Microsoft Dynamics 365 的某一零售销售点 (POS) 和电子商务配置选项的工序。 商业要素提供可以帮助零售商使用基于 Microsoft Dynamics 的零售解决方案迅速运营并让他们关注特定于零售的功能的现成零售工作负载。 

<a name="overview"></a>概览
--------

商业软件环境通常包含许多系统，每个系统均执行非常具体的功能。 通常，零售商若要利用零售管理方面工序的 Microsoft Dynamics 365，但是，不存在要替换其财务或人力资源 (HR) 系统。 对于这些零售商，需要特定于零售方面的工作负载。 商务 Essentials Pack 提供这些特定于零售的功能为 Dynamics 365 的一个配置选项的工序。 在完成配置后，解决方案的所有非零售方面将隐藏，以便将精力放在重要方面：管理零售组织。 商业要素首先是作为 Microsoft Dynamics AX 2012 R3 CU8 中的配置选项引入的。 自那以后，已实施许多改进以使该解决方案适用于全方位渠道零售商，而不只是适用于传统组织。

## <a name="configuration"></a>配置
若要配置商务 Essentials，请转到**系统管理** &gt; **设置** &gt; ** **许可证配置，并将零售** **移。 展开**“零售”**节点，然后选中**“商业要素”**的复选框。 单击**“保存”**，查看更改，然后刷新页面。 若要切换回完全的解决方案，转到**商务精英** &gt; **的设置总部** &gt; ** **许可证配置，并将零售** **移。 展开**“零售”**节点，然后清除**“商业要素”**的复选框。 单击**“保存”**，查看更改，然后刷新页面。

## <a name="what-is-included"></a>包含哪些内容？
商务精英包括已创建工序到零售解决方案的核心 Dynamics 365 的所有零售后勤办公室功能，除呼叫中心功能。 所有渠道、工作人员、产品、商店、收银机和设备管理功能都包含在一个导航节点中。 此外，将包含所有零售工作区以提供现用的角色特定的登录页。 针对有砖商务精英和泥或移动情况有利。Retail Modern POS (MPOS) 零售商的和电子商务渠道。 通过使用商业要素，零售商可在一个位置管理这些渠道，从而获得真正的全渠道功能。

## <a name="how-is-commerce-essentials-different"></a>商业要素有哪些不同？
除了提供统一的导航和仅零售页面之外，商业要素将隐藏所有非零售页面元素，例如字段和控件。 结果是一个简化的用户界面，可帮助零售商将精力放在手头的任务上。 具有完整的产品的许可证零售商可以使用商务 Essentials 为的起始点是面向零售用户。 其他元素，例如高级功能，HR 可以添加到商务 Essentials，在需要它们，程序就绪许可协议中的条款。 作为商业要素实现的简化的一部分，会计导出功能可让零售商将商业要素中的应计总量映射到其现有会计系统中的帐号。 在零售系统生成财务数据时，可轻松地将此数据从商业要素导出到零售商的会计系统。 如果需要更高级集成功能，商务精英还包括在工序的 Dynamics 365 包括导出/导入完成的框架。 利用默认情况下包含的数以百计的数据实体，商业要素能够灵活地适应使用了多个专用系统的多层环境。
