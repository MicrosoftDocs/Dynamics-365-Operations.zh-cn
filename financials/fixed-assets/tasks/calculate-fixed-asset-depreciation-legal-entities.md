# <a name="calculate-fixed-asset-depreciation-across-legal-entities"></a>计算各个法人的固定资产折旧

[!include[task guide banner](../../includes/task-guide-banner.md)]

使用此过程更改为其分配了固定资产的固定资产组。 应将固定资产分配给正确的固定资产组。 创建查询和报表，设置新固定资产，以及集成分类帐和将固定资产交易过帐到适当的分类帐科目时，使用固定资产组。

此记录使用 USMF 公司演示。

1. 转到“固定资产”>“固定资产”>“固定资产”。
2. 选择要更改其固定资产组的固定资产。
3. 单击“更改固定资产组”。
4. 在“新组”字段中，输入或选择一个值。
5. 将“新增固定资产的编号”选项设置为“是”将向所选固定资产分配固定资产编号。
    * 如果将“新增固定资产的编号”选项设置为“是”，“固定资产编号”字段将变为可用。   如果为固定资产设置了自动编号，该字段将显示下一个可用的固定资产编号。 您可以更改该数量。   如果为固定资产设置了手动编号，则该字段为空，您必须输入新增固定资产的编号。  
6. 单击“确定”。
7. 单击“是”。
