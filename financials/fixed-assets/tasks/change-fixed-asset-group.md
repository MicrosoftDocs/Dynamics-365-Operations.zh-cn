# <a name="change-a-fixed-asset-group"></a>更改固定资产组

[!include[task guide banner](../../includes/task-guide-banner.md)]

应将固定资产分配给正确的固定资产组。 固定资产组的使用场景：

 - 创建查询和报告

 - 设置新增固定资产

 - 整合分类帐并将固定资产交易记录过帐到合适的会计科目

此指南使用演示公司 USMF。

1. 转到“固定资产”>“固定资产”>“固定资产”。
2. 选择要更改其固定资产组的固定资产。
3. 单击“更改固定资产组”。
4. 在“新组”字段中，输入或选择一个值。
5. 选择此选项将固定资产编号分配给所选固定资产。
    * 如果选择“新增固定资产的编号”选项，则“固定资产编号”字段将可用。   如果为固定资产设置了自动编号，该字段将显示下一个可用的固定资产编号。 您可以更改该数量。   如果设置了手动编号，则该字段为空，您必须输入新增固定资产的编号。     
6. 单击“确定”。
7. 单击“是”。
