财务期间结帐工作区

** **财务期间结帐工作区可以跟踪您的跨公司、面积和人员上财务结转过程。 **根据您的财务期间结帐工作区的以下**，您将看到分配给您的所有任务和状态为结算的计划或任务。 

您必须首先选择一个结算的计划在工作区的顶部。 在工作区显示的数据按所选结转然后计划进行筛选。

### <a name="summary-tiles"></a>汇总磁贴

**汇总**磁贴为您提供流程概览以及帮助您保留跟踪的结转流程的指示器。 由于所有任务相关性和剩余的流程的，您可以查看剩余的到期，是任务为今天的任务，到期当天的任务，但处于锁定状态。 此信息供所选结转计划包括的所有公司。

**在任务和省**部分，总体结转计划的状态将属于各种方式：按公司的按区域中的状态，由负责人员的状态和状态。 您可以查看是在到期当天的所有任务的年末结转，计划通过更改筛选器是在到期卡名称列表顶部的任务，或者任务的状态。 您也可以选择使公司筛选器查看特定公司的状态。 州选项卡按已完成的百分比和剩余信用的任务数源细分。 单击"卡或** **详细信息选择的操作由卡筛选详细任务列表中。

## <a name="financial-period-close-configuration-page"></a>财务期间结帐配置页
在您可以使用财务期间结帐** **之前工作区，通过**财务期间结帐**配置"页面中，您在 Microsoft Dynamics 365 中必须配置过程操作。 (单击**总帐** &gt; **期间关闭** &gt; **财务期间结帐配置**)。

### <a name="resources"></a>资源

** **资源在选项卡上，您定义中的年末结转过程中涉及的人员。 负责某个结算任务的所有员工都必须首先将在此处。 您还必须指定该工作区中员工的视图。 选项如下：

-   **仅分配的任务** – 用户将只看到分配给他或她的任务。
-   **所有任务和状态** – 用户将看到整个流程的所有结算任务和状态。

有权仅查看为其分配的任务的用户不能添加任务到任务列表、不能编辑任务或从任务列表中删除任务。

### <a name="task-areas"></a>任务区域

您使用任务区域将结算任务分组到您的组织中的逻辑所有权区域。 例如，应付帐款、应收帐款或总帐可以使用为任务区域。

### <a name="calendars"></a>日历

使用日历选项卡上，创建和编辑财务结算的日历。  这是你要关闭定义的流程的位置和平时排产结算任务使用。  创建新日历，并指示已为任务计划编制将使用的平时。  因为可以在创建后，编辑创建长的时间的日历，例如一年度或多年数是最佳。  在创建日历后，请单击按钮编辑更新日历天，例如假期。  在值，控制时，将打开结算任务对若干天将计划。  如果结算任务计划不应是特定日，该日期应该有控制值设置为已关闭。

### <a name="templates"></a>模板

您使用财务结算的模板定义是结转流程一部分的所有任务。 一结算任务是作为结算的每个流程的一部分，管理员的完成分配给一个定期工作的工作组。 在"模板，则必须为每个结算任务定义一个相对到期日期。 相对到期日期为天数在定义的期间结束日期之前或之后任务将到期的每一个期间。 由于一时间还分配给每个任务。 到期时间设置。使用您的时区上下文，并且转换为每个用户设置时区。 

您可以将模板中的任务到该任务的应用一个或多个公司。 如果不同的人员在每个公司中将您分配完成该工作精力，找到您可能很有用创建同一个工作组的多个工作任务。 为每个公司创建一个任务。 

**任务链接**菜单项将与工作任务精力，可以转到使用直接从链接任务的关联页在工作区。 例如，一结算任务运行应付帐款的货币升值流程工序中 Microsoft Dynamics 365 中可与关联**外币**重新计算"。 您还可以链接到外部 URL。 

> [! 提示]，如果要链接特定 Management Reporter 报表某一财务期间结帐任务，可以使用报表 URL。 若要访问报表 URL，请在报表设计器中打开一个报表，然后单击**文件** &gt; **视图**打开在报表查看器中查看报表。 然后可以复制浏览器地址栏中的 URL 并将其粘贴到**任务链接** **URL** 字段中。 

您可以定义该模板中的任务相关性。 如果该任务设置由一个或多个任务，该任务不能标记为完成，直到所有相关性已完成。 

您可以创建多个财务结算的模板。 您可以使用不同模板不同期间跟踪类型选结算流程，例如用于月底或年末，或跟踪使用不同的结转流程的公司。 在模板创建之后，您可以将其复制到新模板和进行所需的更改。 您只能将一模板到每个结算的计划。

### <a name="closing-schedules"></a>结帐计划

您使用一结算的计划分配一个财务结算到的模板必须结算特定的财务期间。 从模板的任务自动生成。然后指定的期间，并且新的结转计划添加到工作区。 在您创建某一新结转计划时，**期间结束日期**字段用于基于在财务结算的配置文件分配的相对到期日期确定结算任务的实际到期日期。 

分派日历相应为结算的计划，用于指示任务计划的工作日。 如果未定义特定日历，任务到期日期将利用所有星期几。 

你还必须定义要与结算的计划的公司。 如果模板任务分配给多个公司，单独的任务为中的年末结转计划的每个公司创建和将任务分配到模板。
