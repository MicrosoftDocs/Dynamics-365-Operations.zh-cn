# <a name="accounts-receivable-home-page"></a>应收帐款主页

[!include[banner](../includes/banner.md)]


使用应收帐款可以跟踪客户发票和收入付款。 

您可以创建基于销售订单或装箱单的客户发票。 您还可以输入与销售订单无关的普通发票。 使用若干不同付款类型，您可以接收付款。 其中包括汇票，现金，支票、信用卡和电子付款。 如果您的组织中包括多个法人，则在单个中法人可以使用集中付款代表其他法人记录付款。
