# <a name="cost-accounting-home-page"></a>成本核算主页

为成本核算为可用的此主题提供帮助主题的列表和其他资源。

成本核算模块允许您执行以下操作：

-   定义成本维度要素，导入成本要素维度成员，并且成本维度映射要素。
-   定义成本对象导入成本对象维度和维度成员。
-   定义或导入统计维度。
-   创建统计度量提供程序模板。
-   创建成本核算分类帐。
-   创建成本控制单元。
-   处理总帐条目。
-   处理预算条目。
-   创建和应用成本行为策略。
-   跟踪成本。
-   定义维度层次结构。
-   创建可以在工作区查看的报表。
-   使用 Excel 创建报表。
