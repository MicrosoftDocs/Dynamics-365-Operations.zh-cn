# <a name="configure-tax-integration-for-china"></a>Configure tax integration for China

本主题描述用于配置的税务集成流程中国的。

若要配置中国的税务集成，请完成以下任务。

1.  **创建在增值税发票描述**页面的新增值税发票描述。 例如，您可能需要设置以下参数：
    -   **增值税发票 InvoiceDescID01 ID **描述。
    -   **描述**到详见销售清单
    -   ** **单位到该框中

2.  **上创建税务集成模板**新的页面的税务集成模板。 可以使用发票设置税务集成模板什么时候导入或导出。 在税务集成模板，您可以指定以下信息：
    -   增值税代码
    -   最大发票金额
    -   默认描述和单位金黄税的发票
    -   包含不可扣减的增值税。开票
