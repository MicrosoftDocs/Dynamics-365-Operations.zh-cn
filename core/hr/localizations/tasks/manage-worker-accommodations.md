# <a name="manage-worker-accommodations"></a>管理工作人员便利设施

[!include[task guide banner](../../../includes/task-guide-banner.md)]

此过程介绍设置工作环境便利设施类型的步骤，如符合人体工程学的椅子或定时休息。 此过程还显示如何分配这些便利设施类型给工作人员，以及批准或拒绝便利设施类型。 创建此程序的演示数据公司是 USMF。


## <a name="setup-accommodations"></a>设置便利设施
1. 转到“人力资源”>“设置”>“便利设施类型”。
2. 单击“新建”。
3. 在“便利设施类型”字段中，输入便利设施的名称。 示例：键盘。
4. 在“描述”字段中，输入便利设施类型的描述。 示例：人体工程学键盘。
5. 在“注释”字段中，输入帮助阐明便利设施的所有信息。
6. 单击“保存”。
7. 关闭该页面。

## <a name="assign-accommodations"></a>分配便利设施
1. 转到“人力资源”>“工作人员”>“员工”。
2. 从列表中选择某一位员工。
3. 单击员工的姓名查看请求便利设施的员工的详细信息。
4. 展开“个人信息”快速选项卡。
5. 单击“便利设施”。
6. 单击“新建”。
7. 在“便利设施类型”字段中，选择相应便利设施。 示例：键盘。
8. 在“工作任务”字段中，输入请求便利设施的工作任务。
9. 在“状态”字段中，输入相应状态。 示例：合理
    * 以下状态可供使用。 “已请求”表明已创建便利设施请求，但未审核。 “合理”表明便利设施请求合理，并批准该请求。 “不合理”表明便利设施会对雇主造成不合理的负担，因此拒绝该请求。 在此示例中，由于便利设施请求为最低要求，因此立即审核该请求。  
10. 在“已接受”字段中，选择接受便利设施请求的人员。
11. 单击“选择”。
12. 在“接受日期”字段中，输入接受便利设施请求的日期和时间。
13. 展开“注释”部分。
14. 在“财务”字段中，输入帮助阐明便利设施的影响的任何信息或资源成本。
    * 如果便利设施请求被拒绝，“回复”字段可用于说明为什么请求遭到拒绝。  
15. 单击“保存”。
