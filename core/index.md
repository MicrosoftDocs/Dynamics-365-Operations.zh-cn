# <a name="introduction-to-dynamics-365-for-operations"></a>介绍动态 365 的操作

[!include[banner](includes/banner.md)]

操作 Microsoft Dynamics 365 是微软企业业务应用程序。 它使人们能够通过提供实时的洞察力和数据访问智能用户界面快速做出更明智的决策。 它可以通过变换和 innnovate 公司提供经验证的业务逻辑，使其更快地重新设计其业务流程。 它为企业提供自己的速度在增长和经营几乎任何地方，任何时候，使他们能够扩展其运作全球以满足业务需求的灵活性。 

本用户指南可以帮助您了解如何使动态 365 操作适用于您的业务。
