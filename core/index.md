# <a name="introduction-to-dynamics-365-for-operations"></a>Dynamics 365 for Operations 简介

[!include[banner](includes/banner.md)]

Microsoft Dynamics 365 for Operations 是 Microsoft 面向企业的业务应用程序。 它让用户可以通过智能的用户界面来访问实时见解和数据，从而快速做出更明智的决策。 它让企业能够提供经证实的业务逻辑，加快重新设计业务流程的速度，从而推动变革和创新。 它为企业带来了灵活性，使企业能够按照自己的节奏发展并几乎可随时随地开展业务，让企业根据业务需求在全球范围内拓展运营。 

本用户指南可帮助您了解如何让 Dynamics 365 for Operations 为您的企业效力。
