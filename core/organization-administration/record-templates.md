# <a name="create-record-templates"></a>创建记录模板

本文介绍记录模板的概念并说明如何使用它们创建共享信息的记录。

记录模板。工序的 Microsoft Dynamics 365 中可帮助您迅速创建记录。 您在 Microsoft Dynamics 365 可以创建中只有某些记录的模板类型记录操作。 例如，假如已经****输入在旧金山运营所在的租车费业务的租赁该信息。 由于大多数客户可能来自旧金山地区，因此如果您能够在租赁窗体上的**“州”**、**“国家/地区”**和**“城市”**字段中自动填入值将是非常理想的。 **注释：**只能将模板应用您有权访问的区域工序的 Dynamics 365。 但在您创建新记录时，所有模板标题均对您可见，如果您创建的是将对所有用户可用的模板，则对其他用户也可见。 请确保在命名模板时考虑到这一点。 例如，如果公司某些员工享受与佣金挂钩的薪酬为秘密，请避免使用如“佣金”之类的词语。 当某个特定窗体存在您可以访问的一个或多个模板并且您尝试在该窗体中创建新记录时，将显示**“为以下项选择模板”**页面。 在您从列表中选择某一模板时，将创建新记录，并且该新记录包含基于您选择的模板的默认信息。 如果您不想使用模板，则在您创建新记录时，选中在**请选择模板。**页面的**再次不要求**复选框。 若要再次显示"模板选择"对话框，请右键单击所有记录，请单击** **记录信息，然后单击** **选择查看模板。
