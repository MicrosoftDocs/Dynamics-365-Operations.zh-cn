# <a name="recently-added-editing-features-in-task-recorder"></a>最近添加编辑在任务录制器中并的功能

如果您在使用任务录制器，您可以创建任务指南更高效地编辑您的文件{{在此：in}} wiki 描述使用的功能。

这些功能是可在**设置 &gt; 任务录制 &gt; 器**记录编辑"菜单。

-   没有再录音整个文件插入的步骤。
-   步骤将子"任务"之下，而不再录音整个文件。
-   折叠记录名称和描述字段。

## <a name="insert-steps-without-rerecording-the-entire-file"></a>插入步骤，而不再录音整个文件
您现在可以添加一步骤中的任意位置任务指南，而不播放或整个再录音文件。

1.  选择步骤，在后要插入新的步骤。 确定步骤中突出显示。

为了任务录制器可以插入步骤，您必须打开具有正确的页面。 页面正确的步骤是新的结果页。 任务录制器在确定的机制哪些活动"页为和禁用功能，则正确的页面将不会打开。 

[] (![tg1。/media/tg1.png)](。/media/tg1.png) 


感觉在正确的页面时插入，** **步骤也随之可用。

[] (![tg2。/media/tg2-231x300.png)](。/media/tg2.png)

2. **单击插入**步骤。

在单击插入** **步骤，则任务录制器切换到记录的方式。 在打开执行的所有行动现在将录制并添加了就地为步骤。

3. **单击"停止**。

可以重复流程添加，许多步骤或移动多个子任务根据需要为以下子任务 (请参阅)。

4. 完成任务指南编辑时，请单击**完成的编辑**，然后选择一个选项以保存或发布任务指南。

## <a name="move-steps-under-a-subtask-without-rerecording-the-entire-file"></a>步骤将子"任务"之下，而不再录音整个文件
您可以移动步骤"子任务"之下，而不播放或整个再录音文件。 如果要将各个步骤，现有的基块还可以将子任务步骤或结束子任务步骤。

1.  选择要移动的步骤或子任务步骤。 步骤确保突出显示。
2.  单击"省略号，然后单击**将{{以后：after}} **步骤。

[] (![tg3。/media/tg3.png)](。/media/tg3.png)

3. 选择步骤或子任务步骤后您要移动步骤或子任务步骤。 任务录制器将移动步骤。

4. 若要将结束子任务步骤，则选择它，单击"省略号，单击**步骤将{{以后：after}} **，然后选择步骤，在后要结束步骤是子任务。

如果希望在任务指南的第一步在子任务内，请创建子任务步骤为第二个步骤，然后移到第一步它。 可以添加或移动多个步骤或子任务根据需要。

5. 完成任务指南编辑时，请单击**完成的编辑**，然后选择一个选项以保存或发布任务指南。

## <a name="collapse-recording-name-and-description"></a>折叠记录名称和说明
您可以展开和折叠**记录名称**和** **记录描述字段。 当这些字段折叠，详细步骤要显示在窗格中编辑任务录制器。
