# <a name="personalize-the-user-experience"></a>个性化用户体验

文章此说明如何个性化您可以设置工序的 Microsoft Dynamics 365。

具有个性化的许多类型在工序 365 的 Microsoft Dynamics 中。 一些个性化设置是让您在设置页面中的选项列表中做出选择。 个性化是某些含蓄的，例如，工序的 Dynamics 365 个记录的网格列宽度，则调整并 FastTabs 的扩展/可折叠的状态。 其他个性化设置是显式的。 对于显式个性化设置，您将进入交互式个性化设置模式，通过直接管理元素在页面上显示或作用的方式来修改页面的外观。 

个性化所有，任何类型，用户在工序的 Dynamics 365 进行可仅对该用户，则无论用户交互使用的公司。 用户对页面所做的更改不会影响系统中其他用户。

## <a name="systemwide-options-for-the-current-user"></a>当前用户的系统范围选项
在导航栏中，您将看到一个齿轮图像，称为**“设置”**菜单按钮。 打开**“设置”**菜单将显示一系列选择。 选择**“选项”**将打开用户**“选项”**页面。 您将再此看到四个选项卡：**“可视化”**、**“首选项”**、**“帐户”**和**“工作流”**。

-   **可视化：**用于选择颜色主题和页面上的元素的默认尺寸。
-   **首选项：**此您可以选择默认为，以便在每次打开工序的 Dynamics 365，包括标识的公司、姓名首字母缩写。默认查看/编辑模式 (页面是否用于查看锁定或打开用于编辑您每次打开它。) 您还将看到语言、时区，以及日期、时间和数字格式选项。 最后，此页面包含一系列因版本而异的杂项首选项。
-   **帐户：**用于提供您的用户 ID 和其他与帐户相关的选项。
-   **工作流：**您可以在此选择与工作流相关的选项。

## <a name="implicit-personalizations"></a>隐式个性化设置
隐式个性化设置是只需通过与记得其当前可视化状态的某些控件互动即可执行的个性化设置。 

**网格列：**您可以选中列标题左侧或右侧的大小调整栏并将其向左或向右滑动到所需宽度，从而调整列表中列的宽度。 工序的 Dynamics 365。存储您可以希望，并且以查看具有该列宽度该您每次打开具有该列表的页面的宽度。 

**快速选项卡：**某些页面具有可扩展的分区，成为快速选项卡。 FastTabs 您，您 FastTabs，展开和折叠的工序的 Dynamics 365。存储。 您每次返回该页面时，相同快速选项卡将基于您上次使用的情况展开或折叠。 在本文中，我们将解释如何更改快速选项卡分区的顺序。 在某些情况下，FastTab 折叠可以提高性能，因为工序的 Dynamics 365 不需要检索该 FastTab 的信息，直到 FastTab 展开。 

**速见表：**某些页面有一个成为速见表窗格的分区。 此窗格包含与该页面的当前主题相关的只读信息。 速见表窗格中的每个部分称为一个速见表。 您可以展开或折叠事实框和 Dynamics 365 的工序将存储您的首选项。 在某些情况下，事实框"折叠可以提高性能，因为工序的 Dynamics 365 不需要检索该事实框"的信息，直到事实框"展开。

## <a name="explicit-personalization-adding-a-tile-or-list-to-a-workspace"></a>显式个性化：向工作区添加磁贴或列表
带有列表的某些页面具有额外的个性化功能，可在其位于“选项”选项卡的“个性化”组下的“操作”窗格中找到。 选择**“添加到工作区”**可打开下拉列表，您可以在工作区上以列表或摘要磁贴形式（可用于显示列表中项目的数量）显示当前列表（经过筛选和排序或默认）中的信息。 

[![Add to workspace](./media/personalization-addtoworkspace.png)](./media/personalization-addtoworkspace.png) 

若要将列表添加到工作区，请首先对列表进行排序或筛选您希望在工作区看到的信息，然后选择“添加到工作区”对话框。 然后，选择所需工作区并从“展示”下拉列表中选择**“列表”**。 当您选择**“列表”**时，系统将打开一个对话框，让您挑选您希望在列表中看到的列，和列表在工作区中显示时的标签。 

若要向工作区添加磁贴，首先请对列表进行筛选，以表示您要汇总（或希望快速访问）的数据。 然后，打开“添加到工作区”下拉对话框。 然后，选择所需工作区并从“展示”下拉列表中选择**“磁贴”**。 当您选择**“磁贴”**时，系统将打开一个对话框，让您提供磁贴标签，并决定磁贴是否会显示计数。 将磁贴置于工作区后，可从工作区打开当前页面，并显示与磁贴相关的信息的列表。 

将列表或磁贴添加到工作区后，您可以打开该工作区并在放置列表或磁贴的组中对列表或磁贴重新排序。

## <a name="administration-of-personalization"></a>管理个性化设置
只需导出个性化设置页面，然后请求其他用户导航到个性化页面并导入您已创建的个性化设置文件，即可对页面进行个性化设置并将其与其他用户共享。 如果用户具备管理特权，那么他们也可以在**“个性化设置”**页面中为其他用户管理个性化设置。 导航到 b 页。 在**“个性化设置”**页面中，您将看到两个选项卡，一个标记为**“系统”**，一个标记为**“用户”**。 

**系统：**：您可以在此临时禁用或“关闭”系统中全部个性化设置。 这不会删除个性化设置，而会将所有窗体重置为其默认状态。 您可以在以后重新启用个性化设置，以将所有个性化设置重新应用于每个用户窗体。 您还可以删除所有用户的所有个性化设置。 请注意，如果您删除个性化设置，将无法从系统自动重新启用个性化设置。 在执行此步骤之前，请确保已将您以后可能要导入的个性化设置导出。 

**用户：**您可以在此为每个用户决定他们可以执行隐式个性化设置还是执行显式个性化设置。 您还可以决定每个用户可以对特定窗体执行隐式个性化设置还是显式个性化设置。 最后，您可以为每个用户导入或导出或删个性化设置。 

**注意：**在最初发布时，个性化设置管理仅允许针对不同的用户进行管理。
