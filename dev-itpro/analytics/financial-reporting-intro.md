# <a name="financial-reporting-for-finance-and-operations"></a>Finance and Operations 财务申报

[!include[banner](../includes/banner.md)]


Finance and Operations 财务申报允许财务和业务专业人员创建、维护、部署和查看财务报表。 它超出了传统报告约束，可以帮助您高效地设计各种类型的报表。

财务报告包括维度支持。 因此，帐户段或尺寸是立即可用。 不需要额外的工具或配置步骤。

## <a name="financial-reporting-components"></a>财务报告组件
财务报告的下列组件让创建、查看和计划报表更轻松。

| 组件        | 功能                                                                                                                                                                                                                                                                           | 附加信息                                                                          |
|------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| 报表设计器  | 创建可以组合来定义和生成报表的报表构建基块。 报表向导将通过设计流程指导经验较少的用户。 高级用户可以创建新的报表构建基块或修改现有构建基块以满足其需求。 |                                                                                                 |
| 报表计划 | 计划单个单报表或报表组，以便定期生成报表。                                                                                                                                                                                          | [生成财务报表](generate-financial-report.md) |
