# <a name="report-designer-interface"></a>报表设计器界面

[!include[banner](../includes/banner.md)]


本文介绍如何在报表设计器内导航，以及如何使用各种选项以满足您的特定要求。 

<a name="report-designer-menu-commands"></a>报表设计器菜单命令
-----------------------------

以下表格描述了您在设计财务报表时可使用的菜单命令和选项。 一些菜单命令和选项仅在特定情况下才可用。 例如，用于提升和降低报告单元的命令仅在您修改报告树定义时可用。

### <a name="file-menu"></a>文件菜单

**“文件”**菜单适用于所有用户并包含以下命令。

| 命令                           | 描述                                                                                                                                                                                      |
|-----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 新增                               | 创建新的报表定义、行定义、列定义、报告树定义、报表组定义或文件夹。 另外还提供其他选项，这取决于您的用户角色。 |
| 未完成                              | 打开现有行定义、列定义、报告树定义或报表定义。                                                                                             |
| 结束                             | 关闭当前构建基块。                                                                                                                                                                |
| 全部关闭                         | 关闭全部构建基块。                                                                                                                                                                       |
| 保存                              | 保存当前的行定义、列定义、报告树定义或报表定义。                                                                                             |
| 另存为                           | 在新名称下方保存当前的行定义、列定义、报告树定义或报表定义。                                                                            |
| 属性                        | 打开**“属性”**对话框，您可以在此对话框中更改报表的名称和描述。                                                                                                   |
| 生成                          | 生成当前报表。 报表定义中提供此命令。                                                                                                                 |
| 查看报表                       | 在 Dynamics 365 for Operations 中打开生成的最新版本报表。 如果您已生成至少一个报表，则报表定义中将提供此命令。                                 |
| 最近报表定义         | 显示最近已创建或修改的报表列表。 然后，您可以在此列表中选择一个报表。                                                                                    |
| 最近行定义            | 显示最近已创建或修改的行定义列表。 然后，您可以在此列表中选择一个行定义。                                                                    |
| 最近列定义         | 显示最近已创建或修改的列定义列表。 然后，您可以在此列表中选择一个列定义。                                                              |
| 最近报告树定义 | 显示最近已创建或修改的报告树定义列表。 然后，您可以在此列表中选择一个报告树定义。                                              |
| 退出                              | 退出报表设计器。                                                                                                                                                                            |

### <a name="edit-menu"></a>编辑菜单

**“编辑”**菜单适用于具有**“设计人员”**或**“管理员”**角色的用户。 此菜单包含以下命令。

| 命令                                | 描述                                                                                                                                                                                                        |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 撤消                                   | 撤消上次操作。                                                                                                                                                                                              |
| 重做                                   | 退回上次撤消的操作。                                                                                                                                                                                      |
| 剪切                                    | 删除所选文本，然后将其复制到剪贴板。                                                                                                                                                            |
| 复制                                   | 将所选文本复制到剪贴板。                                                                                                                                                                           |
| 粘贴                                  | 插入最近从剪贴板中已剪切或复制的文本。                                                                                                                                                    |
| 清除                                  | 删除所选构建基块单元的内容。                                                                                                                                                           |
| 查找                                   | 打开**“查找和替换”**对话框，您可以在此对话框中搜索视图窗格中的文本。                                                                                                                              |
| 替换                                | 打开**“查找和替换”**对话框，您可以在此对话框中搜索和替换视图窗格中的文本。                                                                                                                  |
| 从维度插入行            | 打开**“从维度插入行”**对话框，您可以在此对话框中选择要包含在行定义中的维度值。 行定义中提供此命令。                                  |
| 对行重新编号                          | 对所有数字行代码重新编号。 行定义中提供此命令。                                                                                                                                   |
| 行链接                              | 打开**“行链接”**对话框，您可以在此对话框中指定行定义和报告树定义中数据链接的源。 行定义中提供此命令。                            |
| 舍入调整                    | 打开**“舍入调整”**对话框，您可以在此对话框中指定舍入的参数。 行定义中提供此命令。                                                                  |
| 管理维度集                  | 打开**“维度集”**对话框，您可以在此对话框中创建和修改维度集。 行定义和报告树定义中提供此命令。                                              |
| 插入行                             | 将空行插入行定义中，或者将空标题行插入列定义中。 行定义或列定义中提供此命令。                                               |
| 删除行                             | 从行定义中删除所选行，或者从列定义中删除所选标题行。 行定义或列定义中提供此命令。                                       |
| 插入列                          | 将空列插入列定义中。 列定义中提供此命令。                                                                                                             |
| 删除列                          | 从列定义中删除所选列。 列定义中提供此命令。                                                                                                         |
| 从维度插入报告单元 | 打开**“从维度插入报告单元”**对话框，您可以在此对话框中选择要包含在报告树定义中的维度值。 报告树定义中提供此命令。 |
| 导入维度集层次结构         | 打开**“维度集层次结构”**对话框，您可以在此对话框中导入财务数据中的维度集层次结构。 ...\财务维度\基于维度的系统的报告树定义中提供此命令。  |
| 插入报告单元                  | 将空行插入报告树定义中。 报告树定义中提供此命令。                                                                                                |
| 删除报告单元                  | 从报告树定义中删除所选报告单元行。 报告树定义中提供此命令。                                                                             |

### <a name="view-menu"></a>视图菜单

**“视图”**菜单适用于所有用户并包含以下命令。

| 命令         | 描述                                                            |
|-----------------|------------------------------------------------------------------------|
| 导航窗格 | 显示或隐藏导航窗格。                                      |
| 工具栏        | 选择可见的工作栏。                                  |
| 状态栏      | 在**报表设计器**窗口中显示或隐藏状态信息。 |
| 欢迎页    | 打开**“欢迎”**页面。                                             |

### <a name="format-menu"></a>格式菜单

**“格式”**菜单适用于具有**“设计人员”**或**“管理员”**角色的用户。 此菜单包含以下命令。

| 命令               | 描述                                                                                                                                                                                                          |
|-----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 样式和格式设置 | 打开**“样式和格式设置”**对话框，您可以在此对话框中创建和修改行定义和列定义中文本的样式。 行定义或列定义中提供此命令。 |
| 列宽度          | 打开**“列宽度”**对话框，您可以在此对话框中设置所选列的宽度。 行定义、列定义或报告树定义中提供此命令。                      |
| 隐藏                  | 隐藏所选列。 行定义、列定义或报告树定义中提供此命令。                                                                                        |
| 取消隐藏                | 将隐藏的列显示在所选列之间。 行定义、列定义或报告树定义中提供此命令。                                                      |

### <a name="company-menu"></a>公司菜单

**“公司”**菜单适用于具有**“设计人员”**或**“管理员”**角色的用户。 此菜单包含以下命令。

| 命令               | 描述                                                                                                            |
|-----------------------|------------------------------------------------------------------------------------------------------------------------|
| 公司             | 打开**“公司”**对话框，您可以在此对话框中创建和修改公司。                                          |
| 构建基块组 | 打开**“构建基块组”**对话框，您可以在此对话框中创建、修改、导入和导出构建基块组。 |

### <a name="go-menu"></a>转到菜单

**“转到”**菜单适用于所有用户并包含以下命令。 **注意：**这些命令不具有可见影响，除非导航窗格可见。

| 命令                   | 描述                                                                        |
|----------------------------|------------------------------------------------------------------------------------|
| 报表定义         | 在导航窗格中显示报表定义。                                    |
| 行定义            | 在导航窗格中显示行定义。                                       |
| 列定义         | 在导航窗格中显示列定义。                                    |
| 报告树定义 | 在导航窗格中显示报告树定义。                            |
| 安全性                   | 在导航窗格中显示用户、组和公司的安全信息。 |

### <a name="tools-menu"></a>工具菜单

**“工具”**菜单适用于所有用户但某些命令具有有限的可用性。 此菜单包含以下命令。

| 命令                       | 描述                                                                                                                                                                                                       |
|-------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 保护                       | 将密码应用于当前构建基块。 此命令适用于具有**“设计人员”**或**“管理员”**角色的用户。                                                                           |
| 报表队列状态           | 打开**“报表队列状态”**对话框，您可以在此对话框中查看最近生成的所有报表以及每个报表的详细信息。                                                                                    |
| 源系统信息     | 显示 Microsoft Dynamics ERP 系统的设置。 此命令适用于具有**“设计人员”**或**“管理员”**角色的用户。                                                                 |
| 已签出项             | 显示当前打开的行定义、列定义、报告树定义和报表定义。 此命令适用于具有**“设计人员”**或**“管理员”**角色的用户。 |
| 刷新缓存的财务数据 | 更新财务维度列中的数据。                                                                                                                                                               |
| 期权                       | 打开**“选项”**对话框，您可以在此对话框中修改 Report Designer 的用户首选项。                                                                                                                       |

### <a name="window-menu"></a>窗口菜单

**“窗口”**菜单适用于所有用户并包含以下命令。

| 命令              | 描述                                                                                                                                                                                   |
|----------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 水平平铺    | 并排水平显示所有打开的窗口。                                                                                                                                                     |
| 垂直平铺      | 并排垂直显示所有打开的窗口。                                                                                                                                               |
| 级联              | 分层放置所有打开的窗口，以便每个窗口的标题栏可见。                                                                                                                      |
| 水平冻结    | 冻结所选行，以便在您滚动时，行继续在窗口中显示。 此命令适用于具有**“设计人员”**或**“管理员”**角色的用户。       |
| 垂直冻结      | 冻结所选列，以便在您滚动时，列继续在窗口中显示。 此命令适用于具有**“设计人员”**或**“管理员”**角色的用户。 |
| 打开的窗口的列表 | 显示打开的窗口的列表。 选择要将其带至前方的窗口。                                                                                                               |
