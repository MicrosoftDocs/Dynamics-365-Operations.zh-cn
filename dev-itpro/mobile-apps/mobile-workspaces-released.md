# <a name="mobile-workspaces-recently-released"></a>最近发布的移动工作区

[!include[banner](../includes/banner.md)]



此主题列出了 Microsoft Dynamics 365 for Unified Operations 移动应用最近发布的移动工作区。

> [!NOTE]
> 此移动应用以前称为 *Microsoft Dynamics 365 for Finance and Operations*。


## <a name="new-mobile-workspaces-released-in-june-2017"></a>2017 年 6 月发布的新移动工作区

| 移动工作区     | 说明   | 了解更多   |
|----------------------|---------------|--------------|
|公司目录| 让用户能够查看和联系其组织中的其他员工。| [公司目录移动工作区](company-directory-mobile-workspace.md)|    
|我的团队| 让用户能够查看其直接下属和扩展员工，以及向其报告链中的人员发送表扬。| [我的团队移动工作区](manager-self-service-mobile-workspace.md)|     
|发票审核| 提供通过供应商发票抬头工作流程分配给您的发票的列表。|[发票审核移动工作区](invoice-approval-mobile-workspace.md)    |    

## <a name="new-mobile-workspaces-released-in-april-2017"></a>2017 年 4 月发布的新移动工作区

| 移动工作区   | 说明                                                                                                                                                                                                      | 了解更多                                                                                                      |
|--------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| 费用报销管理 | 让用户能够捕获和上载收据，以便以后可以将其附加到支出报表。 移动工作区还允许用户使用附加的收据快速创建支出行。 | [费用管理移动工作区](/dynamics365/unified-operations/financials/expense-management/expense-management-mobile-workspace) |
| 采购订单审核 | 让用户能够查看和使用审核或拒绝等操作响应采购订单。 | [采购订单审核移动工作区](/dynamics365/unified-operations/supply-chain/procurement/purchase-order-mobile-workspace) |

## <a name="new-mobile-workspaces-released-in-march-2017"></a>2017 年 3 月发布的新移动工作区

| 移动工作区   | 说明                                                                                   | 了解更多                                                                                                                                                                              |
|--------------------|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

## <a name="new-mobile-workspaces-released-in-january-2017"></a>2017 年 1 月发布的新移动工作区

| 移动工作区     | 说明                                                                                                                                                                         | 了解更多                                                                                                                                                        |
|----------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 成本控制     | 让成本中心管理人可以随时随地查看成本中心的性能。                                                                                               | [成本控制移动工作区](/dynamics365/unified-operations/financials/cost-accounting/cost-controlling-mobile-workspace)         |
| 现有库存量    | 让用户可随时随地从移动角度洞察预留库存和可用库存。                                                                                                    | [现有库存量移动工作区](/dynamics365/unified-operations/supply-chain/inventory/inventory-on-hand-mobile-workspace)       |
| 销售订单         | 让用户可以随时随地掌握最新的销售订单。                                                                                                                          | [销售订单移动工作区](/dynamics365/unified-operations/supply-chain/sales-marketing/sales-orders-mobile-workspace)                 |
