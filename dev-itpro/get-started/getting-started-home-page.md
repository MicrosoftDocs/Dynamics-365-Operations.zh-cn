### <a name="task-recordings-and-task-guides"></a>任务录制和任务指南

任务录制器是一种 Dynamics 365 for Operations 工具，可用来记录您在用户界面 (UI) 上执行的操作。 在您创建*任务录制*时，在服务器上执行的所有事件将在您在 UI 中执行它们时记录。 这些事件包括添加值、更改设置和删除数据。 *任务指南*是受控的、引导式、交互式的体验，带领您完成任务录制的步骤。 您可以从帮助系统中打开（*播放*）任务指南。
-   [连接帮助系统](help-connect.md)
-   [Dynamics 365 for Operations 中的任务录制器](../user-interface/task-recorder.md)
-   [使用任务录制创建文档或培训](../user-interface/task-recorder.md)
