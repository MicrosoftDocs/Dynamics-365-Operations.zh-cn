# <a name="design-the-relationships-between-organizational-units"></a>设计组织部门之间的关系

[!include[task guide banner](../includes/task-guide-banner.md)]

此过程可以通过设计组织单位关系的方式完成。 您必须在定义组织单位关系之前创建一个新的组织用途，也可以使用现有的组织用途。 使用 USRT 公司演示数据完成此过程。 此任务专门面向管理人员。

1. 转到“组织管理”>“组织”>“组织层次结构”。
2. 单击“新建”。
3. 在“名称”字段中，键入一个值。
4. 单击“分配用途”。
5. 在列表中，找到并选择所需记录。
6. 单击“添加”。
7. 在列表中，找到并选择所需记录。
8. 单击“确定”。
    * 根据您的组织层次结构的需要，您可以选择多个用途。  
9. 在列表中，找到并选择所需记录。
10. 单击“设置”以设置默认组织结构。
11. 关闭该页面。
12. 单击“保存”。
13. 单击“查看”。
14. 单击“编辑”。
15. 单击“插入”。
16. 单击“业务单位”。
17. 在列表中，找到并选择所需记录。
18. 在列表中，单击所选行中的链接。
19. 单击“插入”。
20. 单击“零售渠道”。
21. 在列表中，找到并选择所需记录。
22. 在列表中，单击所选行中的链接。
    * 根据需要，您还可以添加多个组织单位。  
23. 单击“保存”。
24. 单击“关闭”。
25. 单击“发布”以打开下拉对话框。
26. 在“有效日期”字段中输入日期和时间。
27. 在“有效日期”字段中输入日期和时间。
28. 在“变量描述”字段中，输入一个值。
29. 单击“发布”。
30. 单击“关闭”。
