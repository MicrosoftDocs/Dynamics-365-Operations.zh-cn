# <a name="create-financial-dimensions-for-retail-channels-and-configure-dimension-values-on-stores"></a>创建零售渠道的财务维度和配置商店上的维度值

[!INCLUDE [task guide banner](../includes/task-guide-banner.md)]

此程序会逐步演示如何创建带有维度值的零售渠道财务维度，以及配置零售商店的财务维度值的步骤。 该主题不包括其他相关步骤，例如创建维度集和科目结构。 此程序使用 USRT 演示数据公司。

1. 转到“总帐”>“会计科目表”>“维度”>“财务维度”。
2. 单击“新建”。
3. 在“使用以下来源中的值”字段中，选择“零售渠道”。
4. 在“维度名称”字段中，输入一个值。
5. 单击“启用”。
6. 单击“关闭”。
7. 单击“启用”。
8. 单击“维度值”。
9. 关闭该页面。
10. 单击“保存”。
11. 关闭该页面。
12. 转至“零售和商业”>“渠道”>“零售商店”>“所有零售商店”。
13. 在列表中，单击所选行中的链接。
14. 切换“财务维度”部分的展开项。
15. 单击“编辑”。
16. 在“零售渠道”字段中，单击下拉按钮以打开查找。
17. 在列表中，查找并选择正在更新的商店的维度值。
18. 在列表中，单击所选行中的链接。
19. 在“成本中心”字段中，单击下拉按钮以打开查找。
20. 在列表中，找到并选择所需记录。
21. 在列表中，单击所选行中的链接。
22. 在“部门”字段中，单击下拉按钮以打开查找。
23. 在列表中，找到并选择所需记录。
24. 在列表中，单击所选行中的链接。
25. 单击“保存”。
