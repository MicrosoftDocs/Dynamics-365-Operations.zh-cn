# <a name="payment-methods-in-a-call-center"></a>电话中心内的付款方法

[!include[banner](includes/banner.md)]


本主题介绍可以在 Dynamics 365 for Retail 的呼叫中心使用的不同付款方法。

在其他通道中使用的付款方法，例如现金、支票、信用卡和礼品卡，也可以用于电话中心。 在您为电话中心设置付款方法后，它作为一个选项出现在所有电话中心用户**销售订单**页的**付款**部分。 此外，您可设置优惠券为在贵组织的呼叫中心下单的客户提供折扣。 优惠券可为固定折扣金额或物料价格或订单合计的百分比。 例如，当客户话费 750.00 或更多时，基于金额的优惠券提供给客户75.00 的折扣。 可以创建不同类型的优惠券，设置父/子优惠券，复制或取消优惠券。 使用下表中的选项可创建优惠券。

|                           |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **属性**             | 在**兑现率**字段中，以输入以优惠券预期兑换率百分比的价值，然后选择优惠券是否可以一次性使用，是否为自动签发，或是否指定到客户。                                                                                                                                                                                                                                                                                                                                                                                       |
| **有效**                 | 在**开始日期**和**结束日期** 字段中，输入优惠券有效期的开始和结束日期。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **包含/排除规则** | 在**目录**和**物料**字段中，选择优惠券中是否包括或排除任何目录或物料。 如果您选择**包括**或**排除**，单击**设置**，选择**包含/排除目录**或**包含/排除产品**，输入目录或物料的有关信息。 如果您在以下字段中选择**无**，优惠券中包括目录或物料。                                                                                                                                                                                                                          |
| **杂项**         | 如果不能和其他折扣一起使用此优惠券，选择**排除**复选框。 然后，在**来源**字段中，选择可以使用优惠券的位置。 如果此优惠券为制造商的优惠券，请选择**制造商优惠券**复选框。                                                                                                                                                                                                                                                                                                                                                                |
| **未来优惠劵**         | 如果将此优惠关联作为其他优惠券的父优惠券，请选择 **父优惠券** 复选框。 如果此优惠券与现有优惠券关联作为子优惠券，请选择**父优惠券 ID** 字段中的父优惠券。 例如，您为即将发布的春季目录创建一个优惠券。 为春季目录中创建的所有其他优惠券将为春季目录优惠券的子优惠券。 子优惠券可能包括新客户订单的 20% 折扣，最新发布物料的 10% 折扣，金额为 1000.00 或更多的订单折扣为 95%。 |

如果从**销售订单**页提交信用卡付款并收到表明该卡未经授权的消息，则可以手动处理授权。 您可以通过使用**授权管理**页授权、拒绝或提交信用卡交易。 您使用调用中心参数页来配置其他付款处理选项︰

-   支票保留让财务人员可以处理已暂停的订单，这些订单因为将支票用作付款方式，并且支票保留金额超过了阈值金额而暂停。 可以手动释放保留，或在配置期间结束时自动到期。
-   您可以设置必须手动审核通过支票和信用卡发出的退款高出阈值。 超过阈值金额的任何退款均被添加至审核队列。 在审核退款后，可对返还销售订单开具发票。
