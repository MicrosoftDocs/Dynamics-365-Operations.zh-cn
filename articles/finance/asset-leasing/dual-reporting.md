# <a name="dual-reporting"></a>双重申报

[!include [banner](../includes/banner.md)]

本主题通过一个示例指导您如何在资产租赁中同时满足国际财务申报标准 (IFRS) 申报和法定申报的要求。 需要熟悉 Microsoft Dynamics 365 Finance中的过帐层，这一可以让示例更容易理解。

## <a name="example"></a>示例

以下示例使用现金基础法和 IFRS 申报法介绍意大利法定申报下的租赁。 公司必须建立三本帐簿，以同时满足意大利的法定要求和 IFRS 16 的要求。 IFRS 16 需要一本帐簿，法规要求需要一本帐簿，并且还需要一本帐簿来自动撤消法定过帐。 下面的表中显示了这些帐簿的设置。

**IFRS 16 帐簿**

IFRS 16 帐簿设置为遵守 IFRS 16 会计标准。 此帐簿中过帐的所有条目都将过帐到自定义层。

| 姓名                                    | 说明    |
|-----------------------------------------|----------------|
| 帐簿类型                               | IFRS 16        |
| 帐簿描述                        | IFRS 16        |
| 过帐层                           | 自定义层 1 |
| 租赁类型                              | Finance        |
| 会计框架                    | IFRS 16        |
| 租赁期/使用年限设置         | 0.00           |
| 现值/资产的公平价值设置 | 0.00           |
| 短期阈值                    | 12             |
| 低价值阈值                     | 5,000.00       |
| 向供应商付款                           | 不             |

**法定帐簿**

法定帐簿是一种现金基础帐簿，在这种帐簿中，公司将把租赁费用作为每月支付的租金现金金额入帐。 这种帐簿不会产生使用权 (ROU) 资产或租赁负债。

| 姓名                                    | 说明 |
|-----------------------------------------|-------------|
| 帐簿类型                               | 法定   |
| 帐簿描述                        | 本地 GAAP  |
| 过帐层                           | 当前     |
| 租赁类型                              | 自动   |
| 会计框架                    | 现金基数  |
| 租赁期/使用年限设置         | 0.00        |
| 现值/资产的公平价值设置 | 0.00        |
| 短期阈值                    | 0           |
| 低价值阈值                     | 0           |
| 向供应商付款                           | 不          |

**法定冲销帐簿**

法定冲销帐簿的建立方式与法定帐簿相同。

| 姓名                                    | 说明                    |
|-----------------------------------------|--------------------------------|
| 帐簿类型                               | 法定 – 冲销           |
| 帐簿描述                        | 用于冲销法定帐簿的帐簿 |
| 过帐层                           | 自定义层 1                 |
| 租赁类型                              | 自动                      |
| 会计框架                    | 现金基数                     |
| 租赁期/使用年限设置         | 0.00                           |
| 现值/资产的公平价值设置 | 0.00                           |
| 短期阈值                    | 0                              |
| 低价值阈值                     | 0                              |
| 向供应商付款                           | 不                             |

如上表所示，根据法定申报和 IFRS 申报，需要申报三本帐簿。

- 法定帐簿根据当前层下基于现金的记帐记录租赁付款。
- 法定冲销帐簿冲销法定日记帐条目。
- IFRS 16 帐簿创建按照 IFRS 16 所需的日记帐条目。

一次只能输入一个租赁。 然后可以打开 **帐簿** 页以查看与租赁关联的所有帐簿。

> [!NOTE]
> 创建帐簿时，所有三本帐簿都必须与同一个租赁记录关联。

第一个日记帐条目将租赁费用记录在法定帐簿下。 您可以批量创建付款，也可以通过在法定帐簿中选择付款计划来创建付款。

对于此示例，将从付款计划为法定帐簿生成以下日记帐条目。

| 总帐       | 4              | 当前 | 结算科目    |          | 1,000.00 |

如果要使用 IFRS 16 数据来运行同一个试算平衡表，则必须冲销法定会计日记帐条目，并且必须过帐 IFRS 16 日记帐条目。 为了冲销法定日记帐条目，此示例中包括一个法定冲销帐簿，该帐簿与法定帐簿具有相同的设置，但相反的过帐配置文件。 例如，法定帐簿 *借记* 租赁费用科目，但冲销帐簿则 *贷记* 此科目。 这些关系很容易在 **资产租赁参数** 页面（**资产租赁 \> 设置 \> 资产租赁参数**）中的资产租赁过帐科目中定义。

如果对冲销帐簿使用用于法定帐簿的相同流程，将生成以下日记帐条目。 区别在于冲销帐簿记录法定帐簿中的冲销条目。 还将为自定义层创建冲销条目。 在当前层运行试算平衡表时，不包括冲销日记帐条目。

| 总帐       | 1              | 自定义 | 租赁费用       |          | 1,000.00 |
