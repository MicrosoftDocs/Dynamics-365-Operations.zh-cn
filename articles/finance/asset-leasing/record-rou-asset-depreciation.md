# <a name="record-right-of-use-asset-depreciation-preview"></a><span data-ttu-id="3d5cd-103">记录使用权资产折旧（预览）</span><span class="sxs-lookup"><span data-stu-id="3d5cd-103">Record right-of-use asset depreciation (Preview)</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="3d5cd-104">对于在组织资产负债表上确认的租赁，使用权 (ROU) 资产按月摊销。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-104">For leases that are recognized on an organization's balance sheet, the right-of-use (ROU) asset is amortized on a monthly basis.</span></span> <span data-ttu-id="3d5cd-105">本主题说明如何为摊销创建日记帐条目。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-105">This topic explains how to create the journal entry for the amortization.</span></span> <span data-ttu-id="3d5cd-106">摊销根据您的过帐配置文件和租赁类型的设置借记费用分类帐科目和贷记累计折旧分类帐科目。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-106">The amortization debits the expense ledger account and credits the accumulated depreciation ledger account, based on the setup of your posting profile and the lease type.</span></span> <span data-ttu-id="3d5cd-107">可以为每个租赁创建这些条目，也可以使用批量日记帐功能为多个租赁创建它们。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-107">These entries can be created for each lease, or they can be created for multiple leases by using the batch journal functionality.</span></span>

## <a name="asset-depreciation-schedule"></a><span data-ttu-id="3d5cd-108">资产折旧计划</span><span class="sxs-lookup"><span data-stu-id="3d5cd-108">Asset depreciation schedule</span></span>

1. <span data-ttu-id="3d5cd-109">在 **租赁摘要** 页上，选择一个租赁。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-109">On the **Lease summary** page, select a lease.</span></span> <span data-ttu-id="3d5cd-110">然后选择 **帐簿 \> 资产折旧计划** 打开 **资产折旧计划** 页。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-110">Then select **Books \> Asset depreciation schedule** to open the **Asset depreciation schedule** page.</span></span>

    <span data-ttu-id="3d5cd-111">使用权资产折旧费用日记帐条目基于 **折旧费用** 列中的金额。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-111">The ROU asset depreciation expense journal entry is based on the amount in the **Depreciation Expense** column.</span></span> <span data-ttu-id="3d5cd-112">有关会计准则合规性指南的示例，请参阅本主题后面的[计算融资租赁的使用权资产摊销费用](#calculation-of-rou-asset-amortization-expense-for-finance-leases)部分。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-112">For an example of the guidance for accounting standard compliance, see the [Calculation of ROU asset amortization expense for finance leases](#calculation-of-rou-asset-amortization-expense-for-finance-leases) section later in this topic.</span></span>

4. <span data-ttu-id="3d5cd-116">选择日记帐条目，然后选择 **过帐** 将折旧条目记录到总帐中。</span><span class="sxs-lookup"><span data-stu-id="3d5cd-116">Select the journal entry, and then select **Post** to record the depreciation entry to General ledger.</span></span>

## <a name="calculation-of-rou-asset-amortization-expense-for-finance-leases"></a><span data-ttu-id="3d5cd-165">计算金融租赁的使用权资产摊销费用</span><span class="sxs-lookup"><span data-stu-id="3d5cd-165">Calculation of ROU asset amortization expense for finance leases</span></span>
