# <a name="assign-number-sequences"></a>分配编号规则

[!include [banner](../includes/banner.md)]

本主题说明如何为租赁 ID 创建编号规则。 它还说明了如何创建在索引重估过程中使用的唯一 ID。

1. 转到 **资产租赁 \> 设置 \> 资产租赁参数**。
2. 选择 **编号规则** 侧选项卡。
3. 在侧边栏中选择 **编号规则**。
4. 为 **租赁 ID** 引用选择编号规则。 该编号规则将用于为每个租赁生成唯一的标识符。
5. 为 **流程 ID** 引用选择编号规则。 该编号规则将用于为每个索引重估过程生成唯一的标识符。
6. 为 **终止方案 ID** 引用选择编号规则。 该编号规则将用于为每个终止方案生成唯一的标识符。
