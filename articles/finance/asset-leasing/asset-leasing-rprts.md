# <a name="asset-leasing-reports"></a>资产租赁报表

[!include [banner](../includes/banner.md)]

本主题列出并简要描述了资产租赁中的可用报表。 如上所述，大多数报表通过完成这些步骤或非常相似的步骤来显示）。 

- 要查看大多数资产租赁报表，请转到 **资产租赁 > 查询和报表 > 租赁报表**，然后选择要查看的报表。 对于需要选择其他路径的报表，报表的描述中包含用于打开报表的步骤。 
- 选择要打印的报表时，将打开一个参数页面，供您筛选报表中包含的信息。 输入筛选条件，然后选择 **确定** 生成报表。 生成的报表将显示属于您指定的筛选范围的信息。

## <a name="asset-movement"></a>资产变动
资产变动报表充当各租赁的使用权资产余额的后滚报表。 此报表用于查看指定时间段某个租赁的资产交易记录。 该报表中包含以下字段。 

|     报表字段                  |     说明                                                                |
|------------------------------------|--------------------------------------------------------------------------------|
|     开始日期              |     租赁最早版本的开始日期。                     |   
|     租赁期                     |     租赁期的当前版本。                            |
|     短期租赁               |     如果将租赁分类为短期租赁，则将显示为 **是**。         |
|     低价值租赁                |     如果将租赁分类为低价值租赁，则将显示为 **是**。          |
|     初始使用权资产     |     初始确认日记帐条目中的使用权资产的原始值。      |
|     期初余额              |     截止 **开始日期** 的使用权资产的帐面净值。                         |
|     期间折旧支出    |     为报表定义的日期范围内花费的折旧费金额。        |
|     累计折旧       |     截至 **开始日期** 的累计折旧金额。                               |
|     减损                     |     为报表定义的日期范围内减损的资产金额。               |
|     修改                  |     日期参数之间使用权资产的调整金额。                            |
|     帐面净值                 |     使用权资产的期末帐面净值，这是截止 **结束日期** 的累计折旧净值。    |

## <a name="differences-report"></a>差异报表
差异报表显示加载到租赁导入框架中的信息与系统中的当前信息之间的差异。 供您比较在租赁导入框架中调整、更新或添加的内容。  

此报表中的值取决于所选租赁。 此报表仅显示与系统中的当前字段和暂存表中的字段不同的字段。 旧值是系统中的当前值或系统中以前的值，具体取决于是否运行了导入流程。 新值显示暂存表中将取代旧值的值。

## <a name="five-years-undiscounted-payment-forecast"></a>五年无折扣付款预测
五年无折扣付款预测报表显示预测从报表参数中指定的日期开始接下来五年中要支付的无折扣租赁付款。 该报表中包含以下字段。 

|     报表字段         |     说明                                                                                       |
|-------------------------- |---------------------------------------------------------------------------------------------------    |
|     租赁描述     |     租赁标题中的租赁描述。                                                      |
|     租赁 ID              |     唯一租赁 ID。                                                                              |
|     帐簿                  |     关联到帐簿 ID 的租赁帐簿。                                                         |
|     分类        |     租赁的分类。                                                                  |
|     过帐层         |     此租赁过帐到的层。                                                         |
|     货币              |     租赁的货币。                                                                        |
|     当前               |     从报告日期开始接下来 12 个月内所有应付租赁付款的总和。          |
|     13-24 个月          |     从报告日期开始 13 与 24 个月之间所有应付租赁付款的总和。           |
|     25-36 个月          |     从报告日期开始 25 与 36 个月之间所有应付租赁付款的总和。           |
|     37-48 个月          |     从报告日期开始 37 与 48 个月之间所有应付租赁付款的总和。           |
|     49-60 个月          |     从报告日期开始 49 与 60 个月之间所有应付租赁付款的总和。           |
|     后续            |     从报告日期开始 61 个月或之后所有应付租赁付款的总和。              |

## <a name="gaap-cash-flows-report"></a>GAAP 现金流报表
GAAP 披露报表满足 842-20-50-4(g)(1) 中规定的 US GAAP 披露要求。 要查看此报表，请转到 **资产租赁 > 查询和报表 > 披露 > GAAP – 现金流**。 

|     报表字段                                 |     说明                                                                                                                                               |
|------------------------------------------------   |-----------------------------------------------------------------------------  |
|     开始日期 <br> 截止日期                        |     定义用于限制报表中包含的信息的日期范围。      |
|     法人                                  |     与租赁绑定的法人。                                      |
|     租赁类型                                    |     作为财务或运营的租赁归类。           |
|     租赁 ID                                      |     唯一租赁 ID。                                                      |
|     租赁描述                             |     租赁标题中的租赁描述。                              |
|     租赁帐簿                                    |     为其引用行的租赁帐簿。                        |
|     过帐层                                 |     附加到固定资产的每个帐簿都是针对具有整体折旧目标的特定过帐层设置的。                          |
|     租赁组                                   |     租赁绑定到的组。                                     |
|     货币                                      |     所用交易货币的缩写。 所有报表将交易货币转换为申报币种。                      |
|     财务租赁 – 经营性现金流         |     日期参数之间从摊销计划过帐的总过帐可变付款和总利息付款之和。        |
|     财务租赁 – 财务现金流           |     日期参数之间摊销计划中总主体付款之和。       |
|     经营性租赁 – 经营现金流       |     日期参数之间所有过帐租赁付款和过帐可变付款之和。            |

## <a name="lease-balances-forecast"></a>租赁余额预测
租赁余额预测列出了直接来自负债摊销计划和资产折旧计划的信息。 此报表显示一段时间预测租赁负债的预测金额和使用权资产，包括这些租赁的所有预测费用。 该报表中包含以下字段。

|     报表字段                 |     说明                                                                                                                                                                               |
|---------------------------------  |--------------------------------------------------------------------------------------------------------------------   |
|     期初余额             |     包含报表开始日期的期间的租赁摊销计划中的期初余额。            |
|     初始确认           |     如果租赁的开始日期在为报表定义的日期范围内，此列将显示初始确认日记帐条目的负债科目值。      |
|     付款                      |     租赁的付款之和在为报表定义的日期范围内。                               |
|     利息                      |     租赁的利息费用之和在为报表定义的日期范围内。                    |
|     期末余额                |     截止 **结束日期** 的租赁负债余额。                                                             |
|     短期负债          |     截止 **结束日期** 的租赁摊销计划中的短期负债金额。                           |
|     长期负债           |     截止 **结束日期** 的租赁摊销计划中的长期负债金额。                            |
|     期初帐面净值      |     包含报表开始日期的期间的租赁资产折旧计划中的“期初帐面净值”      |
|     初始确认           |     如果租赁的开始日期在报表的日期参数之间，此列将显示初始确认日记帐条目的资产科目值。            |
|     折旧支出          |     租赁的折旧费用之和在为报表定义的日期范围内。                  |
|     期末余额                |     截止 **结束日期** 的租赁资产余额。                                                              |
|     权益调整             |     期初余额减去期初帐面净值。                                                             |

## <a name="lease-commencements-report"></a>租赁开始报表
租赁开始报表显示指定日期范围内开始的所有租赁，包括初始负债和使用权资产余额。 该报表中包含以下字段。 

|     报表字段                 |     说明                                                                                       |
|---------------------------------  |---------------------------------------------------------------------------------------------------    |
|     开始日期             |     为该租赁版本过帐的初始确认日记帐条目的日期。         |
|     初始负债金额      |     初始确认日记帐中的负债金额。                                  |
|     初始资产金额          |     初始确认日记帐中的资产金额。                                      |

## <a name="lease-modification-report"></a>租赁修改报表
租赁修改报表显示指定日期范围内已修改的所有租赁。 此报表还显示调整租赁的用户，以及调整的负债总额。 该报表中包含以下字段。 

|     报表字段                 |     说明           |
|---------------------------------  |-------------------------  |
|     调整者                   |     修改租赁的人员的用户名。                                |
|     调整日期               |     调整日记帐条目的过帐日期。                        |
|     调整                   |     日期参数之间过帐到租赁的负债金额的任何调整日记帐条目的金额。 贷方将显示为正，而借方将为负。       |
|     收益(损失)金额            |     日期参数之间过帐到租赁的收益/损失金额的任何调整日记帐条目的金额。 贷方将显示为正，而借方将为负。       |
|     留存收益             |     日期参数之间过帐到租赁的留存收益金额的任何调整日记帐条目的金额。      |
|     期末负债余额      |     截止租赁的调整日期产生的负债余额。           |

## <a name="lease-movement-report"></a>租赁变动报表
租赁变动报表充当各租赁的租赁负债余额的后滚报表。 此报表供用户查看指定时间段某个租赁的负债交易记录。

|     报表字段             |     说明                                               |
|----------------------------   |-------------------------------------------------------------- |
|     开始日期         |     租赁最早版本的开始日期。    |
|     租赁期                |     租赁最早版本的租赁期。           |
|     剩余付款        |     尚未为租赁过帐的付款数量。                       |
|     期初余额         |     影响报表开始日期之前的租赁负债的所有交易记录之和。             |
|     初始确认       |     为报表定义的日期范围内为租赁过帐的任何初始确认交易记录的金额。     |
|     付款                  |     为报表定义的日期范围内已过帐的租赁付款交易记录之和。 因为付款将导致租赁的负债余额减小，因此此列中的值将显示为负金额。  |
|     利息                  |     为报表定义的日期范围内针对租赁已过帐的应计利息之和。            |
|     调整               |     为报表定义的日期范围内租赁的过帐调整交易记录之和。                               |
|     期末余额            |     截止报表的 **结束日期** 已过帐的所有租赁负债交易记录的余额。                  |

## <a name="lease-transactions-report"></a>租赁交易记录报表
租赁交易记录查询显示资产租赁已生成的所有日记帐条目。 每个日记帐条目与其来源帐簿 ID 链接。 因此您可以轻松地将此日记帐条目与相应租赁关联。 租赁交易记录以类似总帐中的 **凭证交易记录** 页面的方式查询函数。

## <a name="weighted-average-discount-rate-report"></a>加权平均折扣率报表
加权平均折扣率报表满足 ASC 842-20-50-4(g)(4) 中为加权平均折扣率规定的 US GAAP 披露要求。 要查看此报表，请转到 **资产租赁 > 查询和报表 > 披露 > 加权平均折扣率**。 该报表中包含以下字段。 

|     报表字段                     |     说明                                                           |
|------------------------------------   |------------------------------------------------------------------------   |
|     截止日期                        |     此报表中包含 **截止** 日期参数或之前以确认的所有租赁。 应该在要披露的期间的最后一天运行此报表。      |
|     法人                      |     与租赁绑定的法人。                           |
|     租赁 ID                          |     唯一租赁 ID。                                                  |
|     租赁描述                 |     租赁标题中的租赁描述。                          |
|     预订                              |     显示的租赁的特定帐簿类型。                                                                                                                                            |
|     过帐层                     |     附加到固定资产的每个帐簿都是针对具有整体折旧目标的特定过帐层设置的。      |
|     租赁组                       |     租赁所属组。                                 |
|     折扣率                     |     用于折扣租赁付款的比率。                             |
|     货币                          |     所用交易货币的缩写。 所有报表将交易货币转换为申报币种。  |
|     剩余租赁付款          |     从 **截止** 日期开始付款计划中剩余未付款租赁付款的总额。            |
|     剩余加权付款       |     剩余租赁付款乘以所用折扣率。   |
