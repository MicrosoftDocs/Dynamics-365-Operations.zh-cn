# <a name="manage-expense-delegation"></a>管理费用委托

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]


费用委托用户可以代表组织中的另一位员工创建和管理费用报表。

## <a name="configuring-expense-delegation"></a>配置费用委托

要将用户设置为费用委托人，请转到**费用管理 > 设置 > 常规 > 委托人**打开**委托人**页面。 选择**新建**，然后选择将定义为委托人的员工。 输入委托用户的别名以及委托期间的开始日期和结束日期。

## <a name="managing-expense-delegation-on-behalf-of-another-employee"></a>代表另一名员工管理费用委托

如果功能管理键**启用费用委托列表页面**已启用，**委托给我的费用**列表页可通过导航到**费用管理 > 我的费用 > 委托给我的费用**显示。

委托用户可以快速筛选和搜索委托给该用户的现有费用报表。 用户还可以通过单击**新建费用报表**来代表其他用户快速创建新的费用报表。

委托用户还可以通过导航到**费用管理 > 我的费用 > 费用报表**，然后单击**打开其他用户的费用**按钮，来代表其他员工创建和管理费用报表。
