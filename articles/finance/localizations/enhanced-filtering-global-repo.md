# <a name="enhanced-filtering-options-for-finding-configurations-in-the-global-repository"></a>用于在全局知识库中查找配置的增强筛选选项

[!include [banner](../includes/banner.md)]

此主题介绍 Regulatory Configuration Service (RCS) 全局知识库的增强筛选功能，已经改进了这些功能，以便包含以下筛选器： 
- **国家/地区** - 基于 ISO 国家/地区代码  
- **标签** - 适用于功能/特性区域、行业、业务文档类型 

可以单独或成组应用筛选器，以便查找特定或相关配置。 例如，若要查找所有与供应商发票有关的可配置业务文档，可以应用**业务文档类型**筛选器。 

可以通过选择国家/地区代码并单击**应用筛选器**来进一步优化搜索。  

[![全局知识库的“筛选器”部分](media/rcs-enhanced-filter-section.JPG)](./media/rcs-enhanced-filter-section.JPG) 

以下示例显示基于**业务文档类型**进行筛选时的结果。 

[![为业务文档类型应用的筛选器和导入](media/rcs-enhanced-filtering-applied.JPG)](./media/rcs-enhanced-filtering-applied.JPG) 

可以将筛选后的结果单独或成组（通过选择配置组，然后单击**导入**）导入到用户 RCS 或 Dynamics 365 Finance 环境中。
