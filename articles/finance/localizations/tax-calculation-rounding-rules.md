# <a name="tax-calculation-rounding-rules"></a>税款计算舍入规则

[!include [banner](../includes/banner.md)]

本文提供有关税款计算服务税款计算参数中的舍入规则工作方式的信息。

> [!NOTE] 
> 启用税款计算服务后，**销售税代码** 和 **销售税组** 页面上的舍入规则无效。

您可以在 **税款计算参数** 页面（**税务** \> **设置** \> **税务配置** \> **税款计算参数**）的 **常规** 选项卡上 **计算** 快速选项卡的 **销售税舍入规则** 部分中查看税款计算服务的舍入规则。

[![“税务计算参数”页上的舍入规则配置](./media/tax-calculation-parameters-calculation-1.png)](./media/tax-calculation-parameters-calculation-1.png)

**舍入精度** 和 **舍入方法** 字段确定如何舍入税款计算服务的有效负载中的计算金额。

## <a name="rounding-precision"></a>舍入精度

**舍入精度** 字段支持具有最多 6 个小数位的值。 例如，如果将 **舍入精度** 字段设置为 **0.000000**，则计算出的金额将舍入到 6 个小数位，然后发送到 Microsoft Dynamics 365 Finance。 例如，如果使用 **普通** 舍入方法，则金额 **987.1234567** 会舍入为 **987.123457**。

> [!NOTE]
> Finance 根据货币舍入规则来舍入金额。 因此，交易中显示和记录的税额受税款计算舍入规则和货币舍入规则的影响。

## <a name="rounding-method"></a>舍入方法

可以为每个法人配置税款计算的舍入方法。 在 **舍入方法** 字段中，您可以在以下三个选项中选择：**普通**、**舍尾** 和 **进位**。

## <a name="rounding-by"></a>舍入依据 

在 **舍入依据** 字段中，选择应用于税收的舍入原则。 选项如下：

- **税码** - 将每个税码中的税额舍入。
- **税码组合** – 舍入行上税码组合内的税额。

## <a name="calculation-method"></a>计算方法

在 **计算方法** 字段中，选择是为每一行还是为所有行计算发票上的销款。 选项如下：

- **行** - 逐按行计算税额。 每行上的税额不受其他行上的税额的影响。
- **总计** - 计算一个单据上所有行的税额。

### <a name="rounding-calculation-example"></a>舍入计算示例

此示例显示可以针对 **舍入依据** 和 **计算方法** 值的不同组合为一张发票执行的不同计算。 对于此示例，已设置以下设置：

- 发票有四行。
- 有两个税码：**VAT1**（10%）和 **VAT2**（10%）。
- 舍入精度设置为 **0.01**。
- 舍入方法设置为 **进位**。

#### <a name="rounding-by--tax-codes-and-calculation-method--line"></a>舍入依据 = 税码和计算方法 = 行

| 行号 | 净额行 | 已确定的税码 | 舍入前的税额 | 舍入的税额 |
|-------------|-----------------|----------------------|----------------------------|--------------------|
| 1           | 11.11           | VAT1                 | 1.111                      | 1.12               |
| 2           | 22.22           | VAT1；VAT2           | 2.222；2.222               | 2.23；2.23         |
| 3           | 33.33           | VAT1                 | 3.333                      | 3.34               |
| 4           | 44.44           | VAT1；VAT2           | 4.444；4；444               | 4.45；4.45         |

#### <a name="rounding-by--tax-code-combinations-and-calculation-method--line"></a>舍入依据 = 税码组合和计算方法 = 行

| 行号 | 净额行 | 已确定的税码 | 舍入前的税额 | 舍入的税额 |
|-------------|-----------------|----------------------|----------------------------|--------------------|
| 1           | 11.11           | VAT1                 | 1.111                      | 1.12               |
| 2\*         | 22.22           | VAT1；VAT2           | 2.222；2.222               | 2.23；2.22         |
| 3           | 33.33           | VAT1                 | 3.333                      | 3.34               |
| 4\*\*       | 44.44           | VAT1；VAT2           | 4.444；4；444               | 4.45；4.44         |

\* 行 2 = Round\[22.22 × (10% + 10%)\] = 2.23 + 2.22

\*\* 行 4 = Round\[44.44 × (10% + 10%)\] = 4.45 + 4.44

#### <a name="rounding-by--tax-codes-and-calculation-method--total"></a>舍入依据 = 税码和计算方法 = 总计

| 行号 | 净额行 | 已确定的税码 | 舍入前的税额 | 舍入的税额 |
|-------------|-----------------|----------------------|----------------------------|--------------------|
| 1           | 11.11           | VAT1\*               | 1.111                      | 1.12               |
| 2           | 22.22           | VAT1\*；VAT2\*\*     | 2.222；2.222               | 2.22；2.23         |
| 3           | 33.33           | VAT1\*               | 3.333                      | 3.33               |
| 4           | 44.44           | VAT1\*；VAT2\*\*     | 4.444；4；444               | 4.44；4.44         |

\*VAT1（行 1、行 2、行 3、行 4）= Round\[（11.11 + 22.22 + 33.33 + 44.44）× 10%\] = 1.12 + 2.22 + 3.33 + 4.44

\*\*VAT2（行 2，行 4）= Round\[(22.22 + 44.44) × 10%\] = 2.23 + 4.44

#### <a name="rounding-by--tax-code-combinations-and-calculation-method--total"></a>舍入依据 = 税码组合和计算方法 = 总计

| 行号 | 净额行 | 已确定的税码 | 舍入前的税额 | 舍入的税额 |
|-------------|-----------------|----------------------|----------------------------|--------------------|
| 1\*         | 11.11           | VAT1                 | 1.111                      | 1.12               |
| 2\*\*       | 22.22           | VAT1；VAT2           | 2.222；2.222               | 2.23；2.22         |
| 3\*         | 33.33           | VAT1                 | 3.333                      | 3.33               |
| 4\*\*       | 44.44           | VAT1；VAT2           | 4.444；4；444               | 4.44；4.45         |

\* 行 1，行 3 = Round\[(11.11 + 33.33) × 10%\] = 1.12 + 3.33

\*\* 行 2，行 4 = Round\[(22.22 + 44.44) × (10% + 10%)\] = 2.23 + 2.22 + 4.44 + 4.45
