<span data-ttu-id="43ccc-129">通过使用每月平均成本模型运行每月库存结转时，当月的所有收货按照表示该月的收货总量和值的发货交易记录结算。</span><span class="sxs-lookup"><span data-stu-id="43ccc-129">When you run a monthly inventory closing by using the monthly average cost model, all receipts for the month are settled against an issued transaction, which represents the total received quantity and value for that month.</span></span> <span data-ttu-id="43ccc-130">此发货交易记录有一个相应的收货交易记录，该收货交易记录将结算所有库存交易记录发货。</span><span class="sxs-lookup"><span data-stu-id="43ccc-130">This issued transaction has a corresponding receipt transaction, from which all inventory transaction issues will be settled.</span></span> <span data-ttu-id="43ccc-131">通过这种方法，所有发货分配相同的平均成本。</span><span class="sxs-lookup"><span data-stu-id="43ccc-131">In this manner, all issue are allocated the same average cost.</span></span> <span data-ttu-id="43ccc-132">发货交易记录和收货交易记录可以视为转移，称为“加权平均库存结转转移”。</span><span class="sxs-lookup"><span data-stu-id="43ccc-132">The issued transaction and receipt transaction can be seen as a transfer, which is named the weighted average inventory closing transfer.</span></span>
