[!include [banner](../includes/banner.md)]

<span data-ttu-id="fc817-104">本主题说明如何为从源扣缴税款 (TDS) 税务类型设置预缴税金组分组，如 **租金** 和 **承包商**。</span><span class="sxs-lookup"><span data-stu-id="fc817-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="fc817-105">转到 **税 \> 设置 \> 预缴税金 \> 预缴税金组分组**。</span><span class="sxs-lookup"><span data-stu-id="fc817-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="fc817-106">[![预缴税金组分组页面](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="fc817-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="fc817-107">在 **税务类型** 字段中，选择 **TDS** 为 TDS 税务类型设置预缴税金组分组。</span><span class="sxs-lookup"><span data-stu-id="fc817-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="fc817-108">在“操作窗格”中，选择 **新建** 创建一行。</span><span class="sxs-lookup"><span data-stu-id="fc817-108">On the Action Pane, select **New** to create a line.</span></span>

5. <span data-ttu-id="fc817-110">在 **描述** 字段中，输入描述。</span><span class="sxs-lookup"><span data-stu-id="fc817-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="fc817-111">在 **常规** 快速选项卡上，在 **状态** 字段中，选择 TDS 组分组的居民身份。</span><span class="sxs-lookup"><span data-stu-id="fc817-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="fc817-112">选项有 **居民** 和 **非居民**。</span><span class="sxs-lookup"><span data-stu-id="fc817-112">The options are **Resident** and **Non-resident**.</span></span>

8. <span data-ttu-id="fc817-114">关闭该页面。</span><span class="sxs-lookup"><span data-stu-id="fc817-114">Close the page.</span></span>
