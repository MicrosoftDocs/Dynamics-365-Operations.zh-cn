# <a name="deferral-posting-methods"></a>延期过帐方法

本文介绍订阅计费中收入和支出延期的延期过帐方法之间的差异。

在 **收入和费用延期参数** 页面上，递延过帐方法选项包括 **资产负债表** 和 **损益**。 本文中的示例将帮助解释这两种方法之间的差异，以及您可能使用一种方法或另一种方法的原因。

**资产负债表** 方法仅使用两个帐户。 因此，涉及的设置较少。 对于收入、折扣和成本，**损益** 方法有两个额外的帐户（**初始确认** 和 **确认抵销**），具体取决于交易类型。 这些帐户在 **延期默认值** 页面（**订阅计费 \> 收入和支出延期 \> 设置**）上设置。 虽然该示例专注于延期收入，但可以将相同的概念应用于延期折扣和延期成本。

## <a name="example"></a>示例

销售发票 1 总共具有 $3,000，并且具有延期收入。 下表显示过帐此销售发票时的分配。

**资产负债表方法**

| 类型 | 科目 | 借方 | 贷方|
|---|---|---|---|
| 借方 | 应收帐款 | 3,000.00 | |
| 贷方 | 延期收入 | | 3,000.00 |

**损益方法**

| 类型 | 科目 | 借方 | 贷方 |
|---|---|---|---|
| 借方 | 应收帐款 | 3,000.00 | |
| 借方 | 收入确认对方科目 | 3,000.00 | |
| 贷方 | 延期收入 | | 3,000.00 |
| 贷方 | 初始收入确认 | | 3,000.00 |

销售发票 2 总共具有 $2,000，并且没有延期收入。

| 类型 | 科目 | 金额 |
|---|---|---|
| 借方 | 应收帐款 | 3,000.00 |
| 贷方 | 收入 | 3,000.00 |

**合并销售发票 1 和 2 的资产负债表方法合计**

| 科目 | 借方 | 贷方 |
|---|---|---|
| 应收帐款 | 5,000.00 | |
| 收入 | | 2,000.00 |
| 延期收入 | | 3,000.00 |

使用 **资产负债表** 方法时，查看期间总收入不是那么容易。 部分收入过帐到 **递延收入** 帐户。 请记住，由于每个期间都会确认收入，因此 **递延收入** 帐户中有多个借方和贷方。 给定期间的总收入将与收入确认的详情混在一起。

**合并销售发票 1 和 2 的损益方法合计**

| 科目 | 借方 | 贷方 |
|---|---|---|
| 应收帐款 | 5,000.00 | |
| 收入 | | 5,000.00 |
| 收入抵销科目 | 3,000.00 | |
| 延期收入 | | 3,000.00 |

所有收入都将转到损益 **收入** 帐户。 然后，延期收入将从损益对帐单移至资产负债表。 您可以轻松地查看总收入，因为所有内容都已过帐到 **收入** 帐户。 但是，部分总收入已延期。 因此，它已移至 **延期收入** 帐户，并在稍后被确认。

在 **损益** 方法中，您可以查看 **收入** 帐户和 **收入抵销** 帐户，以查看 $5,000 的总收入和 $2,000 的净收入（延期净额）。 虽然 **损益** 方法可以简化报告，但要设置的帐户更多。
