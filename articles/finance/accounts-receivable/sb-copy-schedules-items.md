# <a name="copy-a-billing-schedule"></a>复制计费计划

[!include [banner](../includes/banner.md)]

您可以使用 **复制计划** 页面从现有的计费计划创建新的计费计划。 新的计费计划将具有与原始计费计划相同的设置。 您还可以使用此页面将物料从一个计费计划复制到另一个计费计划。

## <a name="copy-a-billing-schedule"></a>复制计费计划

要复制计费计划，请按照以下步骤操作。

1. 在 **所有计费计划** 或 **有效计费计划** 列表页上，选择现有计费计划。
2. 在 **计费计划** 选项卡上，选择 **复制计划**。
3. 在 **客户帐户** 字段中，选择客户帐户。
4. 根据需要更新计费开始日期和结束日期。
5. 根据需要更新以下可选字段的值：**最终用户帐户**、**合作伙伴帐户**、**客户参考**、**客户申请** 和 **延期开始日期**。
6. 选择 **查看预览** 查看并选择要复制的计费计划行。
7. 选择 **复制**。

复制流程完成后，新的计费计划将出现在 **所有计费计划** 或 **有效计费计划** 列表页的底部，以及 **计费计划** 页上。

## <a name="copy-items-from-one-billing-schedule-to-another"></a>将物料从一个计费计划复制到另一个

1. 在 **所有计费计划** 或 **有效计费计划** 列表页上，选择现有计费计划。
2. 在 **计费计划** 选项卡上，选择 **复制计划**。
3. 在 **客户帐户** 字段中，选择客户帐户。
4. 在 **计费计划编号** 字段中，选择计费计划编号。
4. 更新计费开始日期和结束日期。
5. 根据需要更新以下字段的值：**最终用户帐户**、**合作伙伴帐户**、**客户参考**、**客户申请** 和 **延期开始日期**。
6. 选择 **查看预览** 查看并选择要复制到现有计费计划的计费计划行。
7. 选择 **复制**。

复制流程完成后，查看您将物料复制到的计费计划，确保物料已按预期添加。
