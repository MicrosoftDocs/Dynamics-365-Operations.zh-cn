# <a name="revenue-split-templates-in-subscription-billing"></a>订阅计费中的收入分成模板

使用 **收入分成模板** 页设置收入分成模板。 收入分成由具有子物料的父物料构成。 此类物料通常作为单个物料或捆绑销售物料销售给客户。

例如，可以通过以下方式创建计算机物料：

- **父物料：** 订阅银色
- **行（子）物料：**

    - 支持
    - 维护
    - 许可证

创建父物料时，请记住以下限制：

- 物料只能指定为父物料一次。
- 父物料可在同一模板中选择为子物料。
- 有效的模板至少需要一个子物料。
- 物料可以指定为多个捆绑物料的子物料。
- 每个父子关系必须是唯一的。

## <a name="create-a-parent-item-that-has-child-items"></a>创建具有子物料的父物料

按照以下步骤创建具有子物料的父物料。

1. 在 **收入分成模板** 页面上，选择 **新建**。
1. 在 **父物料** 字段中，选择父物料。 将自动更新 **变型编号** 字段。 您可以根据需要更改此值。
1. 在 **分配方法** 字段中，选择一个分配方法。
1. 在 **组件物料** 列表中，选择 **添加** 以添加子物料。
1. 如果您在 **分配方法** 字段中选择了 **百分比**，请在 **百分比** 字段中指定一个百分比。

    - 如果您在 **分配方法** 字段中选择了 **等额**，则将自动更新 **百分比** 字段，以便每个物料具有相等的百分比。
    - 如果您在 **分配方法** 字段中选择了 **可变金额**、**零父金额** 或 **零金额**，则 **百分比** 字段的值将保持为 **0**（零），并且无法更改。

1. 选择 **保存**。

## <a name="revenue-split-on-a-sales-order"></a>销售订单上的收入分成

若要创建具有为收入分成设置的物料的销售订单，请按照这些步骤操作。

1. 在 **销售订单** 页上，创建一个销售订单。
2. 在为收入分成设置的每个物料的行上，选择 **收入分成** 复选框。 该物料将成为父物料。 如果已设置模板，则子物料将自动显示在列表中。
3. 若要添加更多子物料，请选择 **添加收入分成子项**，然后选择要添加的子物料。
4. 保存订单。

## <a name="revenue-split-with-billing-schedules"></a>具有计费计划的收入分成

若要创建具有为收入分成设置的物料的计费计划，请按照这些步骤操作。

1. 在 **所有/可用计费计划** 页面上，创建计费计划。
2. 在为收入分成设置的每个物料的行上，选择 **收入分成** 复选框。 该物料将成为父物料。 如果已设置模板，则子物料将自动显示在列表中。
3. 若要添加更多子物料，请选择 **添加收入分成子项**，然后选择要添加的子物料。
4. 继续执行用于处理计费计划的步骤。

> [!NOTE]
> 如果在 **定期合同计费参数** 页面上将 **自动创建收入分成** 选项设置为 **是**，则会执行以下操作：
>
> - 如果在收入分成模板中将该行物料设置为父物料，则会自动选中 **收入分成** 复选框。
> - 将自动在销售订单或计费计划行上输入子物料。
>
> 如果 **自动创建收入分成** 选项设置为 **否**，则行为如先前所解释的一样。

## <a name="additional-revenue-split-information"></a>其他收入拆分信息

添加作为收入拆分的一部分的物料时，请注意以下信息： 

- 父金额不能延期。
- 子物料的开始日期、结束日期、数量、单位、站点和仓库值基于父物料。 无法更改子物料的这些值。 必须对父物料进行所有更改。 
- 定价方法为 **固定**，无法更改。
- 可以添加或删除子物料。
- 父物料和子物料必须使用同一物料组。 
- 子物料可以具有以下设置之一：

    - **计费频率** 和 **计费间隔** 字段设置为与父物料相同的相应值。 
    - **计费频率** 字段设置为 **一次**。 在这种情况下，**计费间隔** 字段将自动设置为 **1**。 

- 子物料的净额之和等于父金额。 如果分配方法为 **零金额**，则子物料金额和父金额的总和为 0（零）。 

    > [!NOTE]
    > 如果分配方法是 **零父金额**，则子物料的（非零）总和不等于父金额，即不等于 0（零）。 此分配方法用于内部用途，以便员工可以看到子物料。 但是，客户只能看到父物料。

- 如果销售订单的多元素排列 (MEA) 类型为 **单一**，则在添加父物料和子物料时将创建相应的多元素收入分配交易记录行。 
- 如果收入拆分的分配方法为 **等额**，并且父金额已更改，则将重新计算所有子行的金额。 
- 对于分配方法为 **可变金额** 的收入拆分，将发生以下行为：

    - 父物料的净额显示在 **父金额** 列中。 可以编辑此值。 但是，单位价格、净额和折扣都为 0（零），并且无法编辑。
    - 子物料的单位价格为 0（零）。 您可以编辑单价或净额。 编辑一个值时，将自动更新另一个值。

- 对于分配方法为 **百分比** 的收入拆分，将发生以下行为：

    - 父物料的净额显示在 **父金额** 列中。 可以编辑此值。 但是，单位价格、净额和折扣都为 0（零），并且无法编辑。 
    - 子物料净额的计算公式为：*百分比* &times; *父金额*。

- 对于分配方法为 **等额** 的收入拆分，将发生以下行为：

    - 父物料的净额显示在 **父金额** 列中。 可以编辑此值。 但是，单位价格、净额和折扣都为 0（零），并且无法编辑。 
    - 子物料的净额是通过将父金额平均分配给所有子物料来计算的。 
    - 如果删除或添加子物料，则将重新计算净额和单位价格，以便所有子行都具有同等金额。 
    - 如果不能平均分配父金额，则最后一个子物料的净额和单位价格可能略大于或小于其他子物料的净额和单位价格。 

- 对于分配方法为 **零额** 的收入拆分，将发生以下行为：

    - 可以编辑单价、净额和折扣。 父金额为 0（零），无法编辑。 
    - 子物料的数量、单位、站点和仓库值基于父物料。 您无法更改子物料的这些值。 必须对父物料进行所有更改。 
    - 子物料的单位价格和净价为 0（零），无法编辑。 

- 对于分配方法为 **零父金额** 的收入拆分，将发生以下行为：

    - 父物料的单位价格、父金额和净额为 0（零）。
    - 在计费计划中，子行看起来好像是手动添加的，并且所有值都根据选定的计费计划组进行了更新。 这些值无法编辑。 对于子物料，您可以使用 **查看计费详细信息** 中的 **已输入数量**、**单位价格**、**折扣** 和 **净额** 字段来访问 **呈报和折扣** 和 **高级定价** 选项。 
    - 在销售订单上，子行的折扣和折扣百分比为 0（零）。 
    - 可以更改父物料和子物料的计费频率，并且每个行都可以具有不同的频率。 但是，父物料将自动更新，以便使用子行中的最短频率。 例如，收入拆分包含两个子物料，其中一个使用 **每月** 计费频率，另一个使用 **每年** 计费频率。 在这种情况下，父物料的计费频率将更新为 **每月**。
