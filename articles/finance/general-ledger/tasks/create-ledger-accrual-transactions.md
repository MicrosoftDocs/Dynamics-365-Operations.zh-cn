# <a name="create-ledger-accrual-transactions"></a>创建分类帐应计交易记录

[!include [banner](../../includes/banner.md)]

此任务指南介绍生成基于应计架构的分类帐应计交易记录的步骤

1. 转到“总帐”>“日记帐条目”>“普通日记帐”。
2. 在列表中，查找并选择所需日记帐或创建新日记帐。
3. 单击以访问“日记帐批号”字段中的链接。
4. 在列表中，标记所选的行。
5. 在“帐户”字段中，指定所需值。
    * 在此示例中，我们将定义保险的费用。 这将会是定期费用金额。  
6. 在“描述”字段中，键入一个值。
7. 在“借方”字段中，输入一个数值。
8. 在“抵销帐户”字段中，指定所需值。
9. 单击“功能”。
10. 单击“应计分类帐”。
11. 在“应计标识”字段中，单击下拉按钮以打开查找。
12. 在列表中，查找并选择您想要应用的应计架构。
13. 在列表中，单击所选行中的链接。
14. 在“开始日期”字段中，输入一个日期。
15. 单击“交易记录”。
16. 关闭该页面。
17. 单击“确定”。
18. 单击“过帐”。
