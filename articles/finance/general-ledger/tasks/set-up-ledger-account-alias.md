# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="37d4a-103">设置会计科目别名</span><span class="sxs-lookup"><span data-stu-id="37d4a-103">Set up a ledger account alias</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="37d4a-104">该过程显示如何创建可提供科目编号快捷输入方式的科目别名。</span><span class="sxs-lookup"><span data-stu-id="37d4a-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="37d4a-105">该过程使用演示数据公司 USMF。</span><span class="sxs-lookup"><span data-stu-id="37d4a-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="37d4a-106">转到“总帐”>“会计科目表”>“科目”>“会计科目别名”。</span><span class="sxs-lookup"><span data-stu-id="37d4a-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="37d4a-107">单击“新建”。</span><span class="sxs-lookup"><span data-stu-id="37d4a-107">Click New.</span></span>
3. <span data-ttu-id="37d4a-108">在“会计科目别名”字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="37d4a-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="37d4a-109">在“科目结构”字段中，选择科目和维度所属的结构。</span><span class="sxs-lookup"><span data-stu-id="37d4a-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="37d4a-110">在“公司”字段中，单击下拉按钮以打开查找。</span><span class="sxs-lookup"><span data-stu-id="37d4a-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="37d4a-111">在列表中，找到并选择别名适用的公司。</span><span class="sxs-lookup"><span data-stu-id="37d4a-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="37d4a-112">在列表中，单击所选行中的链接。</span><span class="sxs-lookup"><span data-stu-id="37d4a-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="37d4a-113">在“会计科目别名定义”字段中，指定科目和维度。</span><span class="sxs-lookup"><span data-stu-id="37d4a-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="37d4a-114">在使用快捷方式时，将会填充科目和维度。</span><span class="sxs-lookup"><span data-stu-id="37d4a-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="37d4a-115">在“初始焦点”字段中，选择使用别名时将会出现焦点的维度。</span><span class="sxs-lookup"><span data-stu-id="37d4a-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="37d4a-116">在输入快捷方式，以及填充科目和维度后，“初始焦点”字段是光标或焦点的移动目标。</span><span class="sxs-lookup"><span data-stu-id="37d4a-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
