# <a name="cost-rollup-policy-and-overhead-calculation"></a><span data-ttu-id="6abd5-103">成本累积政策和开销计算</span><span class="sxs-lookup"><span data-stu-id="6abd5-103">Cost rollup policy and overhead calculation</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="6abd5-104">成本核算让您能够了解成本流与在组织中交付的产品和服务之间的关联。</span><span class="sxs-lookup"><span data-stu-id="6abd5-104">Cost accounting lets you gain insight into how the cost flow relates to the products and services that are delivered within an organization.</span></span> <span data-ttu-id="6abd5-105">若要查看成本的透明度，基于适当的分配基础实现成本对象间的成本分配十分重要。</span><span class="sxs-lookup"><span data-stu-id="6abd5-105">To see the cost transparency, it's crucial to achieve cost allocation between cost objects based on an appropriate allocation basis.</span></span> <span data-ttu-id="6abd5-106">默认情况下，为主要成本元素实现成本分配，这在有些情况下有利，但它有一些应该考虑的影响。</span><span class="sxs-lookup"><span data-stu-id="6abd5-106">By default, the cost allocation is achieved for the primary cost element, which is desired in some situations, but it has a few implications that should be considered.</span></span>

-   <span data-ttu-id="6abd5-107">在开销计算后，辅助成本对象将以主要成本元素零余额结束。</span><span class="sxs-lookup"><span data-stu-id="6abd5-107">Auxiliary cost objects will end with a zero balance for the primary cost element after overhead calculation.</span></span>

<span data-ttu-id="6abd5-110">要避免这些影响，成本核算让您能够配置成本分配以适应您的组织的管理报告要求。</span><span class="sxs-lookup"><span data-stu-id="6abd5-110">To avoid these implications, Cost accounting lets you configure cost allocation to fit into your organization managerial reporting requirements.</span></span> <span data-ttu-id="6abd5-111">本主题讨论您可以如何确定正确级别的辅助成本元素和创建成本累积规则以适应组织报告和成本可跟踪性。</span><span class="sxs-lookup"><span data-stu-id="6abd5-111">This topic discusses how you can determine the correct level of secondary cost elements and create cost rollup rules that fit into organization reporting and cost traceability.</span></span>

> [!NOTE]
> <span data-ttu-id="6abd5-112">如果报告要求发生更改，您可以更改配置。</span><span class="sxs-lookup"><span data-stu-id="6abd5-112">You can change configurations if reporting requirements change.</span></span>
