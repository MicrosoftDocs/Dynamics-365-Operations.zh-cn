# <a name="import-letter-of-credit"></a>导入信用证

[!include [task guide banner](../../includes/task-guide-banner.md)]

该过程介绍了导入信用证的流程。 在完成该过程前必须完成以下设置：银行贷款、过帐模板、银行贷款协议和供应商银行详细信息。

该程序适用于 USMF 演示公司。


## <a name="create-a-purchase-order-with-letter-of-credit"></a>创建一个“信用证采购订单”
1. 转到“应付账款”>“采购订单”>“所有采购订单”。
2. 单击“新建”。
3. 在“供应商帐户”字段中，输入或选择一个值。
4. 在列表中，找到并选择所需记录。
5. 在列表中，单击所选行中的链接。
6. 展开“常规”部分。
7. 在“站点”字段中，输入或选择一个值。
8. 在列表中，单击所选行中的链接。
9. 在“仓库”字段中，输入或选择一个值。
10. 在列表中，单击所选行中的链接。
11. 在“会计结算日期”字段中，输入一个日期。
12. 在“交货日期”字段中，输入一个日期。
    * 说明：在“银行单据类型”字段中应选择相关值“信用证”。  
13. 单击“确定”。
14. 在“物料编号”字段中，输入或选择一个值。
15. 在列表中，找到并选择所需记录。
16. 在列表中，单击所选行中的链接。
17. 展开“行详细信息”部分。
18. 单击“交货”选项卡。
19. 在“交货日期”字段中，输入一个日期。
20. 在“确认交货日期”字段中，输入一个日期。
21. 在“单位价格”字段中，输入一个数字。
    * 定义信用证的详细信息。  
22. 在“操作窗格”上单击“管理”。
23. 单击“信用证/导入收款单”。
24. 在“申请日期”字段中，输入一个日期和时间。
    * 核实“银行帐户”字段具有基于申请日期默认有效的银行帐户。  
25. 在“银行单据编号”字段中，键入一个值。
26. 在“收到日期”字段中，输入一个日期和时间。
27. 展开“银行单据”部分。
28. 在“到期日期”字段中，输入日期和时间。
29. 展开“银行详细信息”部分。
30. 在“通知银行”字段中，输入或选择一个值。
31. 在列表中，单击所选行中的链接。
32. 单击“保存”。
33. 单击“提取采购订单装运”。
34. 关闭该页面。
35. 在操作窗格上，单击“采购”。
36. 单击“确认”。
37. 在“操作窗格”上单击“管理”。
38. 单击“信用证/导入收款单”。
39. 单击“流程”。
40. 单击“确认”。
    * 核实该融资余额除去了采购订单金额。  在此示例中，采购金额 = 500.00，融资限制 = 10000.00，因此，融资余额 = 9500.00。  
41. 关闭该页面。
42. 在“单位价格”字段中，输入一个数字。
43. 单击“保存”。
44. 在操作窗格上，单击“采购”。
45. 单击“确认”。
    * 根据“单价”的改变修正“信用证”。  
46. 在“操作窗格”上单击“管理”。
47. 单击“信用证/导入收款单”。
    * 根据“采购单价”的改变修正信用证。  
48. 单击“流程”。
49. 单击“修正”。
50. 单击“移除”。
51. 单击“是”。
52. 单击“提取采购订单装运”。
53. 单击“流程”。
54. 单击“确认”。
    * 核实该融资余额除去了采购订单金额。  在此示例中，已编辑的采购金额 = 600.00，融资限制 = 10000.00，因此融资余额 = 9400.00。  
55. 关闭该页面。

## <a name="post-packing-slip"></a>过帐“装箱单”
1. 在操作窗格上，单击“接收”。
2. 单击“产品收据”。
3. 在“PurchParm 表格编号”字段中，键入一个值。
    * 选择“装运数量”创建相关“信用证”。  
4. 在列表中，单击所选行中的链接。
5. 在“产品收货日期”字段中，输入一个日期。
6. 单击“确定”。
7. 关闭该页面。
8. 关闭该页面。

## <a name="verify-import-letter-of-credit-status"></a>核实“导入信用证”状态
1. 转到现金和银行管理>信用证>导出信用证和导入催款
2. 在列表中，找到并选择所需记录。
3. 在列表中，单击所选行中的链接。
    * 核实“导入信用证”状态。     
4. 关闭该页面。
5. 关闭该页面。

## <a name="post-purchase-invoice"></a>过帐采购发票
1. 转到“应付账款”>“采购订单”>“所有采购订单”。
    * 选择此信用证创建的采购订单。  
2. 在列表中，找到并选择所需记录。
3. 在列表中，单击所选行中的链接。
4. 在操作窗格上，单击“发票”。
5. 单击“发票”。
6. 在“编号”字段中，输入一个值。
7. 在“装运数量”字段中，输入或选择一个值。
8. 在列表中，单击所选行中的链接。
9. 在“发票日期”字段中，输入日期。
10. 单击“更新匹配状态”。
11. 单击“过帐”。
12. 关闭该页面。
13. 关闭该页面。

## <a name="verify-import-letter-of-credit-status"></a>核实“导入信用证”状态
1. 转到现金和银行管理>信用证>导出信用证和导入催款
2. 在列表中，找到并选择所需记录。
3. 在列表中，单击所选行中的链接。
    * 核实“导入信用证2”。  
    * 验证：装运状态 = 开具发票的银行融资的详细信息  
4. 单击“查看”。
5. 单击“打印申请”。
6. 单击“确定”。
    * 核实该“信用证”申请已打印。  
7. 关闭该页面。
8. 关闭该页面。
9. 关闭该页面。

## <a name="post-vendor-payment-journal-for-the-created-purchase-invoice-with-letter-of-credit"></a>为创建信用证采购发票过帐“供应商”付款日记帐
1. 转至“应付账款”>“付款”>“付款日记帐”。
2. 单击“新建”。
3. 在“名称”字段中，输入或选择一个值。
4. 在列表中，单击所选行中的链接。
5. 单击“行”。
6. 在“日期”字段中，输入一个日期。
7. 在“帐户”字段中，指定所需值。
8. 单击“结算交易记录”。
9. 展开“总计”部分。
10. 在“显示”字段中，选择一个选项。
    * 核实“银行单据编号和装运数量”字段内容已更新。  
11. 选择“标记”复选框。
12. 单击“确定”。
13. 单击“付款”选项卡。
    * 核实“银行单据编号和装运数量”字段内容已更新。  
14. 单击“过帐”。
15. 关闭该页面。
16. 关闭该页面。

## <a name="verify-import-letter-of-credit-status-after-invoice-paid"></a>核实“导入信用证”状态为“发票支付”
1. 转到现金和银行管理>信用证>导出信用证和导入催款
2. 在列表中，找到并选择所需记录。
3. 在列表中，单击所选行中的链接。
    * 核实“导入信用证”状态。   
4. 关闭该页面。

## <a name="verify-the-bank-facility-limit-and-utilization-report"></a>核实“银行融资限制与利用率报告”
1. 转到现金和银行管理>查询与报告>信用证或担保人
2. 扩展“要包括的记录”部分。
3. 单击“筛选器”。
    * 用所需银行账户定义“条件”字段。  
4. 在“标准”字段中，输入或选择一个值。
5. 在列表中，单击所选行中的链接。
6. 单击“确定”。
7. 单击“确定”。
    * 核实列出交易记录的报表。  
    * 核实该报告列出的交易记录，包括“银行单据编号”、“融资限制”、已使用金额和融资余额。  
8. 关闭该页面。
