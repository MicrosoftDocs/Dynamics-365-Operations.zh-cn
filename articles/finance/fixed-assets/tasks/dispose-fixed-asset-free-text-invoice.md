[!include [banner](../../includes/banner.md)]

<span data-ttu-id="07858-104">此主题介绍如何使用普通发票处置固定资产。</span><span class="sxs-lookup"><span data-stu-id="07858-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="07858-105">在导航窗格中，转到**模块 > 应收帐款 > 发票 > 所有普通发票**。</span><span class="sxs-lookup"><span data-stu-id="07858-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="07858-106">选择**新建**。</span><span class="sxs-lookup"><span data-stu-id="07858-106">Select **New**.</span></span>
3. <span data-ttu-id="07858-107">在**客户帐户**字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="07858-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="07858-108">验证默认**发票**日期并编辑（如果适用）。</span><span class="sxs-lookup"><span data-stu-id="07858-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="07858-109">验证其余默认标题字段（如**币种**）并编辑（如果适用）。</span><span class="sxs-lookup"><span data-stu-id="07858-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="07858-110">在**发票行**部分中，验证**说明**和**主科目**字段。</span><span class="sxs-lookup"><span data-stu-id="07858-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="07858-111">验证默认**销售税组**和**销售税(物料)** 组。</span><span class="sxs-lookup"><span data-stu-id="07858-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="07858-112">输入固定资产的销售**单价**或**金额**。</span><span class="sxs-lookup"><span data-stu-id="07858-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="07858-113">选择**行详细信息**部分。</span><span class="sxs-lookup"><span data-stu-id="07858-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="07858-114">在**固定资产**字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="07858-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="07858-115">选择**过帐**。</span><span class="sxs-lookup"><span data-stu-id="07858-115">Select **Post**.</span></span>
