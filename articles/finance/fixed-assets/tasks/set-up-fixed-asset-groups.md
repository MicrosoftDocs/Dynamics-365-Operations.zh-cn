# <a name="set-up-fixed-asset-groups"></a>设置固定资产组

[!include [banner](../../includes/banner.md)]

此主题介绍如何创建新的固定资产组。 它为 USMF 法人实体使用会计角色和演示数据。

1. 在导航窗格中，转到**模块 > 固定资产 > 设置 > 固定资产组**。
2. 选择**新建**。
3. 在**固定资产组**字段中，键入一个值。
4. 在**名称**字段中，键入一个值。 **固定资产**组为固定资产自动生成编号及编号规则代码将覆盖固定资产参数上的设置。 如果固定资产组中的资产具有与其他组不同的编号，您可以在此进行更改。  
5. 选择**帐簿**。
6. 在**帐簿**字段中，输入或选择一个值。 如果将**计算折旧**字段设置为**是**，资产帐簿将包含于折旧方案中。 如果**计算折旧**设置为**否**，资产将不自动折旧。  
7. 设置资产的使用年限，用年表示。 请注意，在设置使用年限后，计算**折旧期间**字段值。  
8. 在**折旧惯例**字段中，选择一个选项。
9. 关闭该页面。
