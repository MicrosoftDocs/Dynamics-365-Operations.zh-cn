# <a name="set-up-value-models"></a>设置价值模型

[!include [banner](../../includes/banner.md)]
[!include [preview banner](../../includes/preview-banner.md)]

此过程显示如何创建新固定资产帐簿并将其与固定资产组关联。

## <a name="create-a-book"></a>创建帐簿
1. 转到 **固定资产 \> 设置 \> 帐簿**。
2. 选择 **新建**。
3. 在 **帐簿** 字段中，输入一个值。
4. 在 **描述** 字段中，输入一个值。
5. 将 **计算折旧** 选项设置为 **是**。

    如果将 **计算折旧** 选项设置为 **是**，关联的资产帐簿将包含于折旧方案中。 如果设置为 **否**，资产帐簿将不自动折旧。

6. 在 **折旧模板** 字段中，输入或选择一个值。

    * 备选折旧模板也称作折旧转换方法。 当备选模板计算等于或大于默认折旧模板的折旧金额时，折旧方案将切换到此模板。
    * 异常折旧模板用于异常情况下资产的额外折旧。 例如，您可以使用此来记录自然灾害导致的折旧。
    * 如果您选择 **使用基础调整创建折旧调整**，在资产价值更新时将自动创建折旧调整。 否则，更新后的资产价值只会影响未来的折旧计算。

7. 将 **使用基本调整创建折旧调整** 选项设置为 **是**。

    * 默认情况下，固定资产帐簿交易记录将过帐到总帐。 不过，您可以通过将 **过帐到总帐** 选项设置为 **否**，为帐簿禁用过帐到总帐。 不过帐到总帐的帐簿通常用于报税。 此选项使您可以更灵活地删除资产帐簿的历史交易记录，因为交易记录尚未提交到总帐。
    * 默认情况下，如果帐簿过帐到总帐，**过帐层** 字段设置为 **当前层**，如果帐簿未过帐到总帐，则设置为 **无**。 如果此帐簿的交易记录应该过帐到不同层，则更新 **过帐层** 字段的值。

8. 计算正折旧。

    * 默认情况下，**计算正折旧** 选项设置为 **否**。 此设置表示折旧将贷记选定的资产帐簿。 此外，**允许帐面净值高于购置价格** 和 **允许负帐面净值** 选项都设置为 **否**，这些设置可以独立更改。 
    * 要计算正折旧，将 **计算正折旧** 选项设置为 **是**。 此设置表示折旧将借记固定资产帐簿。 当 **计算正折旧** 选项设置为 **是** 时，**允许帐面净值高于购置价格** 和 **允许负帐面净值** 选项将自动设置为 **是**，并且将被锁定。 此锁定有助于确保正折旧仅应用于购置的具有负帐面价值（贷记）的固定资产。 

10. 在 **日历** 字段中，输入或选择一个值。

    派生帐簿将把交易同时过帐到不同帐簿。 可以使用主要帐簿创建交易，并在过帐期间将一模一样的交易副本过帐到衍生帐簿。 不对衍生帐簿交易进行重新计算，所以不应将其用于折旧交易。

## <a name="associate-the-book-with-a-fixed-asset-group"></a>将帐簿与固定资产组关联

1. 选择 **固定资产组**。
2. 在 **固定资产组** 字段中，输入或选择一个值。
3. 在 **使用年限** 字段中，输入一个数字。

    * 在输入资产的使用年限后计算了折旧期。
    * 您可以为税务目的需要来设置折扣惯例。
    * 对于与租赁关联的固定资产，**使用年限** 字段的值将被资产帐簿中的租赁期和资产的使用年限这二者中的较小值所替代。 如果租赁帐簿的 **所有权转移** 选项设置为 **是**，则 **使用年限** 字段的值将始终是资产的使用年限。
