<span data-ttu-id="ed28d-109">以下过程说明如何设置资源的某些能力。</span><span class="sxs-lookup"><span data-stu-id="ed28d-109">The following procedures explain how to set up some of the competencies for a resource.</span></span>

<span data-ttu-id="ed28d-110">若要设置工作人员的能力，您可以使用“人力资源”中的**工作人员**列表页或“项目管理与核算”中的**资源**列表页。</span><span class="sxs-lookup"><span data-stu-id="ed28d-110">To set up competencies for a worker, you can use either the **Workers** list page in Human resources or the **Resources** list page in Project management and accounting.</span></span> <span data-ttu-id="ed28d-111">对于以下过程，将使用“人力资源”中的**工作人员**列表页。</span><span class="sxs-lookup"><span data-stu-id="ed28d-111">For the following procedures, the **Workers** list page in Human resources is used.</span></span>

## <a name="set-up-competencies-certificates"></a><span data-ttu-id="ed28d-112">设置能力：证书</span><span class="sxs-lookup"><span data-stu-id="ed28d-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="ed28d-113">在**工作人员**列表页上，选择您要为其添加证书信息的工作人员的行。</span><span class="sxs-lookup"><span data-stu-id="ed28d-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="ed28d-114">在操作窗格的**工作人员**选项卡的**能力**组中，选择**证书**。</span><span class="sxs-lookup"><span data-stu-id="ed28d-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>

4. <span data-ttu-id="ed28d-116">在**开始日期**字段中，选择 **10/1/2015**，并选择**保存**。</span><span class="sxs-lookup"><span data-stu-id="ed28d-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="ed28d-117">设置能力：技能</span><span class="sxs-lookup"><span data-stu-id="ed28d-117">Set up competencies: Skills</span></span>

2. <span data-ttu-id="ed28d-120">选择**新建**。</span><span class="sxs-lookup"><span data-stu-id="ed28d-120">Select **New**.</span></span>
3. <span data-ttu-id="ed28d-121">在**技能**字段中，选择**项目管理**。</span><span class="sxs-lookup"><span data-stu-id="ed28d-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="ed28d-122">在**级别**字段中选择 **5 专家**。</span><span class="sxs-lookup"><span data-stu-id="ed28d-122">In the **Level** field, select **5 Expert**.</span></span>

6. <span data-ttu-id="ed28d-124">在**年资**字段中，输入 **10**。</span><span class="sxs-lookup"><span data-stu-id="ed28d-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="ed28d-125">选择**保存**，然后关闭页面。</span><span class="sxs-lookup"><span data-stu-id="ed28d-125">Select **Save**, and then close the page.</span></span>
