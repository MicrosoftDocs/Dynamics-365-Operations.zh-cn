# <a name="change-the-banner-or-logo"></a><span data-ttu-id="42dec-103">更改横幅或徽标</span><span class="sxs-lookup"><span data-stu-id="42dec-103">Change the banner or logo</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="42dec-104">系统管理员可以用以下过程列出的步骤更新法人展示所用的横幅或形象标识。</span><span class="sxs-lookup"><span data-stu-id="42dec-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="42dec-105">创建此程序的演示数据公司是 USMF。</span><span class="sxs-lookup"><span data-stu-id="42dec-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="42dec-106">转至“组织管理”>“组织”>“法人”。</span><span class="sxs-lookup"><span data-stu-id="42dec-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="42dec-107">在该列表的左侧，选择需要更新横幅或形象标识的法人。</span><span class="sxs-lookup"><span data-stu-id="42dec-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="42dec-108">如果已经选好，则转到下一步。</span><span class="sxs-lookup"><span data-stu-id="42dec-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="42dec-109">展开“图像”部分。</span><span class="sxs-lookup"><span data-stu-id="42dec-109">Expand the Images section.</span></span> <span data-ttu-id="42dec-110">如果已经展开，则转到下一步。</span><span class="sxs-lookup"><span data-stu-id="42dec-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="42dec-111">单击“编辑”。</span><span class="sxs-lookup"><span data-stu-id="42dec-111">Click Edit.</span></span>
5. <span data-ttu-id="42dec-112">在“仪表板公司图像类型”字段中，选择“横幅”。</span><span class="sxs-lookup"><span data-stu-id="42dec-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="42dec-113">单击“更改”。</span><span class="sxs-lookup"><span data-stu-id="42dec-113">Click Change.</span></span>
    * <span data-ttu-id="42dec-114">提示：横幅图像的理想方案是 1920 x 281 像素。</span><span class="sxs-lookup"><span data-stu-id="42dec-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="42dec-115">形象标识的理想宽度为 350 像素。</span><span class="sxs-lookup"><span data-stu-id="42dec-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="42dec-116">上传成功后，您可以在“仪表板图像”部分查看新的横幅或形象标识。</span><span class="sxs-lookup"><span data-stu-id="42dec-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="42dec-117">单击“保存”。</span><span class="sxs-lookup"><span data-stu-id="42dec-117">Click Save.</span></span>
8. <span data-ttu-id="42dec-118">转到“通用”>“通用”>“默认仪表板”。</span><span class="sxs-lookup"><span data-stu-id="42dec-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="42dec-119">您可以在仪表板上查看新的横幅或形象标识。</span><span class="sxs-lookup"><span data-stu-id="42dec-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="42dec-120">您可能需要使用公司选择器把您公司的横幅改为您所上传的图像。</span><span class="sxs-lookup"><span data-stu-id="42dec-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
