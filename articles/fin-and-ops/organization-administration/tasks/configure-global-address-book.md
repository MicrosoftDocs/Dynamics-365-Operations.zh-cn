# <a name="configure-the-global-address-book"></a>配置全球通讯簿

[!include [task guide banner](../../includes/task-guide-banner.md)]

使用该过程设置全球通讯簿的默认值和安全策略。 

创建此任务的演示数据公司是 USMF。 此任务专门面向“规划和配置团队”。

1. 转到“组织管理”>“全球通讯簿”>“全球通讯簿参数”。
2. 在“名称序列”字段中，选择应如何显示名称。
3. 选择是否删除未被分配角色的当事方。
4. 选择是否检查重复的记录。
5. 选择是否在地址上显示 DUNS 号码。
6. 选择是否检查唯一的 DUNS 号码。
7. 在“当事方”字段中，选择一个选项。
8. 在“客户”字段中，选择一个选项。
9. 在“供应商”字段中，选择一个选项。
10. 在“目标客户”字段中，选择一个选项。
11. 在“竞争对手”字段中，选择一个选项。
12. 单击“专用位置安全”选项卡。
13. 在列表中，找到并选择所需记录。
    * 按下 SHIFT 键选择多个角色，以添加到“选定角色”窗格中，然后单击箭头添加选定的角色。  
14. 单击“保存”。
