# <a name="set-up-i-9-document-types"></a>设置 I-9 文档类型

[!include[task guide banner](../../../includes/task-guide-banner.md)]

此过程显示如何查看和设置 I-9 表格验证所使用的文件类型。 在设置 I-9 文件类型前，您还必须设置签发机构和标识类型。 使用 USMF 公司演示数据创建此过程，该演示数据包括完成此过程所需的签发机构和标识类型的示例。

1. 转到“人力资源”>“工作人员”>“I-9”>“I-9 文件类型”。
2. 单击“新建”。
3. 在“I-9 文件类型”字段中，键入一个值。
    * 示例：学校 ID。  
4. 选择标识类型。  示例：学校 ID
    * 标识类型的示例包括社会保险号 (SSN)、签证号、护照 ID 或驾照。  
5. 选择文件类型使用的 I-9 文件列表。
    * 作为 I-9 流程的一部分，员工必须出示证明其身份和雇用授权的文件。 美国公民及移民服务局网站含有可接受文件以及文件所属列表的信息。  http://www.uscis.gov  
6. 在“表格”字段中，输入该文件类型的官方表格编号。 示例：学校 ID。
    * 官方表格编号可在美国公民及移民服务局网站查看。  http://www.uscis.gov  
7. 选择或取消选择“有效”复选框。
8. 在“到期日期”字段中，设置日期为“2019-10-27”。
    * 到期日期为可选项。  
9. 选择签发该文件类型的机构。 示例：省/直辖市/自治区/区域
10. 单击“保存”。
