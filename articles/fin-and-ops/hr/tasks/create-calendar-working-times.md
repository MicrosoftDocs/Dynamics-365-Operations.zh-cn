[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="41539-104">日历描述运营资源的产能和工作时间。</span><span class="sxs-lookup"><span data-stu-id="41539-104">Calendars describe the capacity and working times of operations resources.</span></span> <span data-ttu-id="41539-105">该过程将帮助您定义基于工作时间模板的工作日历。</span><span class="sxs-lookup"><span data-stu-id="41539-105">This procedure will help you define a work calendar based on a working time template.</span></span> <span data-ttu-id="41539-106">您可以使用演示数据公司 USMF 或您自己的数据浏览此程序。</span><span class="sxs-lookup"><span data-stu-id="41539-106">You can walk through this procedure in demo data company USMF, or using your own data.</span></span>

1. <span data-ttu-id="41539-107">转到“所有工作区”>“资源周期管理”。</span><span class="sxs-lookup"><span data-stu-id="41539-107">Go to All workspaces > Resource lifecycle management.</span></span>
2. <span data-ttu-id="41539-108">单击“日历”。</span><span class="sxs-lookup"><span data-stu-id="41539-108">Click Calendars.</span></span>
3. <span data-ttu-id="41539-109">单击“新建”。</span><span class="sxs-lookup"><span data-stu-id="41539-109">Click New.</span></span>
4. <span data-ttu-id="41539-110">在“日历”字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="41539-110">In the Calendar field, type a value.</span></span>

5. <span data-ttu-id="41539-112">在“名称”字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="41539-112">In the Name field, type a value.</span></span>

7. <span data-ttu-id="41539-114">在列表中，标记所选的行。</span><span class="sxs-lookup"><span data-stu-id="41539-114">In the list, mark the selected row.</span></span>

    * <span data-ttu-id="41539-117">生成您希望能够安排工作的每日工时。</span><span class="sxs-lookup"><span data-stu-id="41539-117">Generate working hours for each day in the period where you want to be able to schedule work.</span></span> <span data-ttu-id="41539-118">随着时间的推移，您可以生成其他期间的工作时间。</span><span class="sxs-lookup"><span data-stu-id="41539-118">As time goes by, you can generate working times for additional periods.</span></span>  
10. <span data-ttu-id="41539-119">在“开始日期”字段中输入日期。</span><span class="sxs-lookup"><span data-stu-id="41539-119">In the From date field, enter a date.</span></span>
    * <span data-ttu-id="41539-120">这是日历必须打开的第一天。</span><span class="sxs-lookup"><span data-stu-id="41539-120">This is the first day that this calendar must be open.</span></span>  
11. <span data-ttu-id="41539-121">在“结束日期”字段中输入日期。</span><span class="sxs-lookup"><span data-stu-id="41539-121">In the To date field, enter a date.</span></span>
    * <span data-ttu-id="41539-122">这是日历打开的最后一天。</span><span class="sxs-lookup"><span data-stu-id="41539-122">This is the last day that this calendar is open.</span></span>

13. <span data-ttu-id="41539-125">单击“确定”。</span><span class="sxs-lookup"><span data-stu-id="41539-125">Click OK.</span></span>
14. <span data-ttu-id="41539-126">关闭该页面。</span><span class="sxs-lookup"><span data-stu-id="41539-126">Close the page.</span></span>
