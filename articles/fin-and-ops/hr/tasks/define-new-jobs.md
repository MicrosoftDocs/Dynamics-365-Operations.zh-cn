# <a name="define-new-jobs"></a>定义新工作

[!include [task guide banner](../../includes/task-guide-banner.md)]

此任务的重点是创建新工作。 将通过手动输入和复制模板功能完成创建。 创建此任务的演示数据公司是 USMF。 此任务专门面向人力资源专员。

1. 转到“人力资源”>“作业”>“作业”。
2. 单击“新建”。
3. 在“作业”字段中，输入一个值。
4. 在“描述”字段中，键入一个值。
5. 选择将用于该作业的标准标题。 
    * 标题上输入的值将用作默认为新的职位分配给作业。  
6. 在列表中，找到并选择所需记录。
7. 在列表中，单击所选行中的链接。
8. 在最大职位分组数中，选择“最大职位数”
9. 输入最大职位数。 
    * 职位分配数不得高于此处输入的最大值。  
10. 切换“描述”部分的扩展。
11. 在“注释”字段中，输入该作业的标准描述。
    * 该描述适用于分配至此处所定义作业的所有职位。  
12. 切换“作业分类”部分的扩展。
13. 选择将用于人力资源中的标准报表的工作职能。
14. 在列表中，找到并选择所需记录。
15. 在列表中，单击所选行中的链接。
16. 选择作业类型，以进一步将您的组织中的作业分类。 
    * 作业类型可在整个人力资源报告过程中使用。  
17. 在列表中，找到并选择所需记录。
18. 在列表中，单击所选行中的链接。
19. 切换“薪酬”部分的扩展。
20. 选择薪酬级别。
    * 薪酬水平将定义员工就工作可拥有的最少和最多薪酬。 薪酬规则将确定员工的薪酬是否超出范围之外。  
21. 在列表中，找到并选择所需记录。
22. 在列表中，单击所选行中的链接。
23. 切换“技能”部分的扩展。
    * 请注意，技能、教育和证书的快速选项卡字段为空。 在从工作模板复制信息后，这些字段的信息将在单步操作中输入。   
24. 单击“保存”。
25. 单击“复制自”。
26. 单击“从模板复制”。
27. 在“作业模板”字段中，单击下拉按钮以打开查找。
28. 在列表中，找到并选择所需记录。
29. 在列表中，单击所选行中的链接。
30. 单击“注释”选项。
    * 只有您选择的物料才会复制到所选作业中。    
31. 单击“确定”。
32. 切换“证书”部分的扩展。
33. 切换“教育”部分的扩展。
34. 单击“保存”。
