# <a name="create-a-new-benefit"></a>创建新福利

[!include[task guide banner](../../includes/task-guide-banner.md)]

该任务将向您显示，如何创建在创建新福利时需要使用的福利元素。 创建此任务的演示数据公司是 USMF。 此任务专门面向薪资和福利经理。


## <a name="create-benefit-elements"></a>创建福利元素
1. 转到“人力资源”>“福利”>“设置”>“福利元素”。
2. 单击“新建”。
3. 在“类型”字段中，输入您创建的福利类型的名称。
4. 在“描述”字段中，键入一个值。
5. 在“同时注册”字段中，选择一个选项。
    * 为限制员工的登记多个医疗计划的可能，每一种类型选择一个登记。  
6. 在“工资类别”字段中，选择一个选项。
7. 单击“计划”选项卡。
8. 单击“新建”。
9. 在“计划”字段中，输入一个值。
10. 在“描述”字段中，键入一个值。
11. 在“类型”字段中，输入或选择一个值。
12. 在“工资影响”字段中，选择一个选项。
13. 单击“保存”。

## <a name="create-a-benefit"></a>创建福利
1. 关闭该页面。
2. 转到“人力资源”>“福利”>“福利”。
3. 单击“新建”，以打开对话框。
4. 在“计划”字段中，输入或选择一个值。
5. 在“选项”字段中，输入或选择一个值。
6. 在"有效日期"字段中，输入日期和时间。
7. 单击“创建福利”。
