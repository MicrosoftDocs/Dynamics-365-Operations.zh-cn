# <a name="create-a-hiring-team-by-using-dynamics-365-talent---onboard"></a><span data-ttu-id="31261-103">使用 Dynamics 365 Talent - Onboard 创建招聘团队</span><span class="sxs-lookup"><span data-stu-id="31261-103">Create a hiring team by using Dynamics 365 Talent - Onboard</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="31261-104">在 Microsoft Dynamics 365 Talent: Onboard 中，可以创建招聘团队。</span><span class="sxs-lookup"><span data-stu-id="31261-104">In Microsoft Dynamics 365 Talent: Onboard, you can create hiring teams.</span></span> <span data-ttu-id="31261-105">然后，您可以为每个团队分配入职指南和模板。</span><span class="sxs-lookup"><span data-stu-id="31261-105">You can then assign onboarding guides and templates to each team.</span></span>

## <a name="create-a-hiring-team"></a><span data-ttu-id="31261-106">创建招聘团队</span><span class="sxs-lookup"><span data-stu-id="31261-106">Create a hiring team</span></span>

1. <span data-ttu-id="31261-107">在左侧菜单中，选择**团队**。</span><span class="sxs-lookup"><span data-stu-id="31261-107">On the left menu, select **Teams**.</span></span>
2. <span data-ttu-id="31261-108">在**团队**下，选择**添加**（加号 \[**+**\]）磁贴。</span><span class="sxs-lookup"><span data-stu-id="31261-108">Under **Teams**, select the **Add** (plus sign \[**+**\]) tile.</span></span>
3. <span data-ttu-id="31261-109">在**创建新团队**对话框中，在**团队名称**下，输入招聘团队的名称。</span><span class="sxs-lookup"><span data-stu-id="31261-109">In the **Create a new team** dialog box, under **Team name**, enter a name for the hiring team.</span></span>

    ![[<span data-ttu-id="31261-110">在 Onboard 中创建新团队</span><span class="sxs-lookup"><span data-stu-id="31261-110">Creating a new team in Onboard</span></span>](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. <span data-ttu-id="31261-111">在**选择团队成员**下，输入每个团队成员的姓名或电子邮件地址。</span><span class="sxs-lookup"><span data-stu-id="31261-111">Under **Choose team members**, enter the name or email address of each team member.</span></span>

    <span data-ttu-id="31261-112">要删除团队成员，请选择框下方其姓名旁边显示的 **X**。</span><span class="sxs-lookup"><span data-stu-id="31261-112">To remove team members, select the **X** that appears next to their name under the box.</span></span>

5. <span data-ttu-id="31261-113">要向新团队成员发送电子邮件，请打开**向新成员发送电子邮件**选项。</span><span class="sxs-lookup"><span data-stu-id="31261-113">To email new team members, turn on the **Send an email to the new members** option.</span></span>
6. <span data-ttu-id="31261-114">选择**创建**。</span><span class="sxs-lookup"><span data-stu-id="31261-114">Select **Create**.</span></span>
7. <span data-ttu-id="31261-115">如果您指示要向新团队成员发送电子邮件，请在下一个对话框中编辑电子邮件，然后选择**完成**。</span><span class="sxs-lookup"><span data-stu-id="31261-115">If you indicated that you want to email new team members, edit the email in the next dialog box, and then select **Done**.</span></span>

2. <span data-ttu-id="31261-118">选择团队。</span><span class="sxs-lookup"><span data-stu-id="31261-118">Select the team.</span></span>

    ![[<span data-ttu-id="31261-120">为团队添加入职指南</span><span class="sxs-lookup"><span data-stu-id="31261-120">Adding onboarding guides to a team</span></span>](./media/onboard-add-guides-to-team.png)](./media/onboard-add-guides-to-team.png)

4. <span data-ttu-id="31261-121">选中要分配给团队的每个入职指南的复选框，然后选择**添加**。</span><span class="sxs-lookup"><span data-stu-id="31261-121">Select the check box for each onboarding guide that you want to assign to the team, and then select **Add**.</span></span>

    ![[<span data-ttu-id="31261-122">选择要添加到团队的入职指南</span><span class="sxs-lookup"><span data-stu-id="31261-122">Selecting the onboarding guides to add to the team</span></span>](./media/onboard-select-guides.png)](./media/onboard-select-guides.png)
