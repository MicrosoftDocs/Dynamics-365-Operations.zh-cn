# <a name="create-a-hiring-team"></a>创建招聘团队

[!include [banner](includes/banner.md)]

在 Microsoft Dynamics 365 Talent: Onboard 中，可以创建招聘团队。 然后，您可以为每个团队分配入职指南和模板。

## <a name="create-a-hiring-team"></a>创建招聘团队

1. 在左侧菜单中，选择**团队**。
2. 在**团队**下，选择**添加**（加号 \[**+**\]）磁贴。
3. 在**创建新团队**对话框中，在**团队名称**下，输入招聘团队的名称。

    ![[在 Onboard 中创建新团队](./media/onboard-create-team.png)](./media/onboard-create-team.png)

4. 在**选择团队成员**下，输入每个团队成员的姓名或电子邮件地址。

    要删除团队成员，请选择框下方其姓名旁边显示的 **X**。

5. 要向新团队成员发送电子邮件，请打开**向新成员发送电子邮件**选项。
6. 选择**创建**。
7. 如果您指示要向新团队成员发送电子邮件，请在下一个对话框中编辑电子邮件，然后选择**完成**。

## <a name="assign-onboarding-guides-to-a-hiring-team"></a>为招聘团队分配入职指南

1. 在左侧菜单中，选择**团队**。
2. 选择团队。
3. 在**指南**选项卡上，选择**添加指南**。

    ![[为团队添加入职指南](./media/onboard-add-guides-to-team.png)](./media/onboard-add-guides-to-team.png)

4. 选中要分配给团队的每个入职指南的复选框，然后选择**添加**。

    ![[选择要添加到团队的入职指南](./media/onboard-select-guides.png)](./media/onboard-select-guides.png)

## <a name="assign-onboarding-templates-to-a-hiring-team"></a>将入职模板分配给招聘团队

1. 在左侧菜单中，选择**团队**。
2. 选择团队。
3. 在**模板**选项卡上，选择**添加模板**。

    ![[为团队添加模板](./media/onboard-add-templates-to-team.png)](./media/onboard-add-templates-to-team.png)

4. 选中要分配给团队的每个模板的复选框，然后选择**添加**。

    ![[选择要添加到团队的模板](./media/onboard-select-templates.png)](./media/onboard-select-templates.png)
