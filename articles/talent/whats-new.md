# <a name="whats-new-or-changed-in-dynamics-365-talent"></a>Dynamics 365 Talent 中的新增功能或更改 

此页提供有关如何查找 Microsoft Dynamics Talent HR Core 最新版本的信息的详细信息。

每周都会推出 Talent 的新版本。 这些版本在 docs.microsoft.com 站点 ([docs.microsoft.com/dynamics365/unified-operations](../index.md)) 的目录中列出。 您可以扫描目录或使用目录上方的筛选器控制来查找您要查找的最新主题。 主题标题将以“Dynamics 365 Talent 中的新增功能或更改的功能…”开头。

目录将继续每周更新，请返回查找 Talent 最新版本的所有新功能和更改。
