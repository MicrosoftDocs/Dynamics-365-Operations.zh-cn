# <a name="view-the-status-of-onboarding-employees-and-your-tasks"></a>查看入职员工和您的任务的状态

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard 允许您查看新雇员在其入职活动中的进度。 您还可以查看从入职指南分配给您的任何任务。

## <a name="view-the-status-of-onboarding-employees"></a>查看入职员工的状态

1. 在左侧菜单中，选择 **指南**。 您会看到您已发送的所有入职指南的进度概览。

    ![[查看入职指南的状态](./media/onboard-guide-status.png)](./media/onboard-guide-status.png)

2. 要向下钻取，请选择要查看的指南。
3. 选择 **活动** 选项卡，查看新雇员已完成的活动。

    ![[查看新雇员已完成的活动](./media/onboard-status-activities.png)](./media/onboard-status-activities.png)

## <a name="view-onboarding-tasks-that-are-assigned-to-you"></a>查看分配给您的入职任务

1. 在左侧菜单中，选择 **任务**。 您可以从已发送给员工的入职指南中看到已分配给您的所有入职任务的概览。
2. 要向下钻取，请选择所需的任务。
3. 根据需要跟进新雇员。
4. 完成任务后，通过向圆圈添加复选标记将其标记为已完成。
