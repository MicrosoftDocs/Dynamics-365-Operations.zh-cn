# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-september-24-2018"></a>Dynamics 365 for Talent Core HR（2018 年 9 月 24 日）中的新增功能或更改的功能

[!include [banner](includes/banner.md)]

**内部版本 8.1.1015.0**

此主题介绍了 Core HR 中的新增功能和更改的功能。

## <a name="currency-added-to-benefits"></a>为福利添加的币种

福利计划已经过更新，现在包含福利币种。 工作人员登记福利中也有这个新字段。 这个新字段是“维护权益”和“查看权益列表”安全权限的一部分。

## <a name="update-proration-process--leave-and-absence"></a>更新按比例分配过程 – 休假和缺勤

组织根据员工加入和离开公司的时间以不同的方式奖励休假。 有些离开组织的员工可能需要在解除雇用日期结束奖励，而其他用户则根据最后工作日停止累积过程。 这些更改让组织可以在组织解除雇用过程中选择何时结束登记。 这些选项是经理自助服务中“解除雇用工作人员”权限的一部分。 

## <a name="other-changes"></a>其他更改

此版本中还包含若干缺陷修复。

## <a name="known-issue"></a>已知问题

-   **问题：** 向工作人员添加新附件时，**新建**和**编辑**按钮灰显。**解决方法：** 打开附件页面之前，确保已关闭**工作人员**页面中的速见表。 如果加载**工作人员**页面时速见表已关闭，将启用附件按钮。 （下一个平台更新中将解决这个问题。）
