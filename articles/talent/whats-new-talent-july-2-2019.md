# <a name="whats-new-or-changed-in-dynamics-365-talent-july-02-2019"></a>Dynamics 365 Talent（2019 年 7 月 2 日）新增功能或更改

此主题介绍了 Microsoft Dynamics 365 Talent 中的新增功能和更改的功能。

## <a name="changes-in-attract"></a>Attract 中的更改

### <a name="use-organization-service-accounts-to-send-e-mails"></a>使用组织的服务帐户发送电子邮件

现在可配置电子邮件设置，以便从组织的电子邮件服务帐户将电子邮件发给应聘者、面试者、招聘人员和招聘经理。 有关更多信息，请参阅[连接电子邮件服务帐户](./attract-configure-email-settings.md#connect-an-email-service-account)。

### <a name="offer-email-templates"></a>聘约电子邮件模板 

现在可针对准备、审核和接受聘约期间的所有阶段配置组织的电子邮件模板。 发送电子邮件之前，用户可使用默认模板或对其进行自定义。 有关详细信息，请参阅[配置电子邮件模板设置](./attract-configure-email-settings.md#configure-email-template-settings)。

### <a name="candidate-consent-on-application"></a>申请表中的用户同意

下载可启用贵组织的“条款和条件”链接。 启用此设置之后，应聘者必须先同意《条款和条件》，才能提交工作申请表。 

### <a name="coming-soon-in-attract"></a>Attract 中即将推出
#### <a name="job-approvals-appear-on-the-home-page"></a>主页中显示工作审核

仪表板的 **审核** 部分中将显示审核。 审核人可以在 **已分配给您**（其中显示工作 ID、工作头衔、其他审核人和作业的分配日期）下查看自己的审核。 提交待审核工作的用户可以在 **您请求的**（其中显示仍然必须审核提交的工作的审核者）下查看自己的工作。

## <a name="changes-in-onboard"></a>Onboard 中的更改

本版本中包含 Dynamics 365 Talent: Onboard 的小缺陷修复。

## <a name="changes-in-core-hr"></a>Core HR 中的更改

本版本中包含 Dynamics 365 Talent: Core HR 的小缺陷修复。
