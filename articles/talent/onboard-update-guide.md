# <a name="update-onboarding-guides"></a><span data-ttu-id="655cd-103">更新入职指南</span><span class="sxs-lookup"><span data-stu-id="655cd-103">Update onboarding guides</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="655cd-104">如果您必须更改 Microsoft Dynamics 365 Talent: Onboard 中的入职指南，即使您已经发送了指南，也可以更新它们并推送更改。</span><span class="sxs-lookup"><span data-stu-id="655cd-104">If you must make changes to onboarding guides in Microsoft Dynamics 365 Talent: Onboard, you can update them and push the changes, even if you've already sent the guides.</span></span> <span data-ttu-id="655cd-105">您有两种更新入职指南的选项：</span><span class="sxs-lookup"><span data-stu-id="655cd-105">You have two options for updating an onboarding guide:</span></span>

## <a name="edit-an-onboarding-guide-directly"></a><span data-ttu-id="655cd-108">直接编辑入职指南</span><span class="sxs-lookup"><span data-stu-id="655cd-108">Edit an onboarding guide directly</span></span>

1. <span data-ttu-id="655cd-109">在左侧菜单中，选择**指南**。</span><span class="sxs-lookup"><span data-stu-id="655cd-109">On the left menu, select **Guides**.</span></span>

    ![[<span data-ttu-id="655cd-112">保存对入职指南的更改</span><span class="sxs-lookup"><span data-stu-id="655cd-112">Saving changes to an onboarding guide</span></span>](./media/onboard-save.png)](./media/onboard-save.png)

<span data-ttu-id="655cd-113">Onboard 会自动向新雇员发送一封电子邮件，指明更改内容。</span><span class="sxs-lookup"><span data-stu-id="655cd-113">Onboard will automatically send the new hire an email that indicates what the changes are.</span></span> <span data-ttu-id="655cd-114">为便于识别，每项更改旁边都会显示一个红色的**新**标签。</span><span class="sxs-lookup"><span data-stu-id="655cd-114">For easy identification, a red **New** tag will appear next to each change.</span></span>
