# <a name="attract-overview"></a>Attract 概述
Attract 提供用于管理新人才雇用流程的功能。 可以将应聘者添加到空缺职位并逐级提升，直到发出聘约或申请被拒绝。 在每个阶段，应聘者经历筛选电话、评估或面试安排等活动。 提交申请后，应聘者将收到一封电子邮件，邀请其注册 Attract，他们可以在此处跟进工作状态、申请表并响应分配给他们的任何任务。

Attract 提供下列功能。

### <a name="attract-dashboard"></a>Attract 仪表板
招聘团队成员可以在此处查看他们参加的工作的详细信息。 招聘经理或招聘人员也可以在此处向 Attract 添加新的工作，方法是一次创建一个工作，或一次性从 Excel 电子表格中导入多个工作。

### <a name="jobs"></a>工作
在**工作**页上，可以查看和编辑工作描述和工作地点等详细信息。 **工作**页可以将招聘团队集合在一起，您也可以在此添加或更改团队中的招聘人员和面试官。 还可以在**工作**页上添加新应聘者。

### <a name="candidates"></a>应聘者
可以将应聘者分别添加到工作中，也可以从 Excel 电子表格大量上传。 在任何工作中，所有应聘者自动添加到招聘流程的第一个阶段，即申请阶段。 在此处可以查看和管理所有应聘者的申请表信息。 可以上传和查看其他文档，以及联系人信息，如电子邮件地址、电话号码和他们的社交媒体帐户链接。 如果应聘者提供了其 LinkedIn 个人资料的链接，且设置了 LinkedIn 集成（需要 LinkedIn 招聘人员许可证），则会显示该应聘者的个人资料。

### <a name="stages"></a>阶段
Attract 提供预定义的招聘流程，包含四个基本阶段：申请、筛选、面试和聘约。 新应聘者从申请阶段开始，在此阶段将审核他们的简历、求职信、档案和其他申请文档。 在此处可以将应聘者提升到筛选阶段，邀请应聘者参加筛选电话或 Skype 视频面试。 招聘经理和招聘人员可以提供关于应聘者的反馈并将其提升到下一个阶段。 然后与招聘团队的空闲成员安排当面面试，并且可以自定义为一对一或不同时长的小组面试。 面试结束后，捕获并审核面试团队的反馈。 选择要进入到最终阶段的应聘者，在该阶段将提供聘约并准备审批。 

### <a name="candidate-app"></a>应聘者应用
应聘者被添加到工作后，将收到一封电子邮件，该电子邮件中包含获取该应用的链接。 此 Web 应用为应聘者提供工作详细信息，并且是与招聘团队进行沟通的工具。 应聘者还可以提交他们可接受面试的空闲时间。
