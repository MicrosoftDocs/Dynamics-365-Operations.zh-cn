# <a name="whats-new-or-changed-in-dynamics-365-talent-core-hr-january-17-2019"></a>Dynamics 365 Talent: Core HR（2019 年 1 月 17 日）中的新增功能或更改

[!include [banner](includes/banner.md)]

**内部版本 8.1.2107**

此主题介绍了 Core HR 中的新增功能和更改的功能。

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>此版本中包括的细微更改和缺陷修复

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>更正了新的职位分配开始日期/时间默认值
在转移员工时，系统将默认此日期/时间为用户所在时区的所选日期和一天开始的时间。

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>所有员工在退出的工作人员任务管理列表中有相同的职位描述
在**任务管理**工作区中进行了更新以显示最近退出公司的所有员工的正确职位描述。

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>“操作请求者”字段现在在工作人员操作页填充
通过此更改，**操作请求者**字段的默认值现在是请求更改的用户。

### <a name="ideas-portal-updated"></a>想法门户已更新
用于提交 Talent 想法的应用内链接已更新为新的想法门户。 在旧门户中输入的所有想法已迁移，因此，您所有的好想法仍在那里，并可由其他人投票。
