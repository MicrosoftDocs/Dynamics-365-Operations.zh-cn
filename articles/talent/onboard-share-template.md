# <a name="share-content-with-other-contributors"></a>与其他参与者共享内容

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard 允许您与其他参与者共享入职指南和模板，以便他们可以查看文档并进行更改。

## <a name="share-an-onboarding-guide-with-other-contributors"></a>与其他参与者共享入职指南

1. 在左侧菜单，选择 **指南**，然后选择要共享的指南。
2. 选择页面右上角的 **更多** 按钮（三个垂直点），然后选择 **共享**。

    ![[与其他参与者共享入职指南](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. 在 **共享** 下，如果接收人应该能够修改指南，请将 **这些用户能否进行编辑?** 选项设置为 **开**。 如果接收人应通过电子邮件收到您已与他们共享指南的通知，请将 **向接收人发送电子邮件通知** 选项设置为 **开**。

    ![[为入职指南设置访问权限和电子邮件首选项](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. 输入名称或电子邮件地址，然后在下拉列表中选择预期接收人。
5. 在 **管理访问权限** 选项卡上，您可以查看或更改哪些接收人可以处理指南。 若要更改接收人的访问权限，在接收人姓名旁边的下拉列表中，根据需要选择 **可以查看**、**可以编辑** 或 **删除**。
6. 在您完成添加或修改接收人时，请选择 **更新**。

## <a name="share-an-onboarding-template-with-other-contributors"></a>与其他参与者共享入职模板

1. 在左侧菜单中，选择 **模板**。
2. 在 **我的模板** 下，在您要共享的模板的右上角选择 **更多** 按钮（省略号 \[**...**\]），然后选择 **共享**。

    [![与其他参与者共享模板](./media/onboard-share-template.png)](./media/onboard-share-template.png)

3. 在 **接收人** 下，如果接收人应该能够修改模板，请将 **这些用户能否进行编辑?** 选项设置为 **开**。 如果接收人应通过电子邮件收到您已与他们共享模板的通知，请将 **向接收人发送电子邮件通知** 选项设置为 **开**。

    [![为模板设置访问权限和电子邮件首选项](./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)

4. 输入名称或电子邮件地址，然后在下拉列表中选择预期接收人。
5. 在 **管理访问权限** 选项卡上，您可以查看或更改哪些接收人可以处理模板。 若要更改接收人的访问权限，在接收人姓名旁边的下拉列表中，根据需要选择 **可以查看**、**可以编辑** 或 **删除**。
6. 在您完成添加或修改接收人时，请选择 **更新**。
