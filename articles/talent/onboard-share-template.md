[!include [banner](includes/banner.md)]

<span data-ttu-id="495d1-104">Microsoft Dynamics 365 Talent: Onboard 允许您与其他参与者共享入职指南和模板，以便他们可以查看文档并进行更改。</span><span class="sxs-lookup"><span data-stu-id="495d1-104">Microsoft Dynamics 365 Talent: Onboard lets you share onboarding guides and templates with other contributors so that they can review the documents and make changes.</span></span>

## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="495d1-105">与其他参与者共享入职指南</span><span class="sxs-lookup"><span data-stu-id="495d1-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="495d1-106">在左侧菜单，选择**指南**，然后选择要共享的指南。</span><span class="sxs-lookup"><span data-stu-id="495d1-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="495d1-107">选择页面右上角的**更多**按钮（三个垂直点），然后选择**共享**。</span><span class="sxs-lookup"><span data-stu-id="495d1-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="495d1-108">与其他参与者共享入职指南</span><span class="sxs-lookup"><span data-stu-id="495d1-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="495d1-109">在**共享**下，如果接收人应该能够修改指南，请将**这些用户能否进行编辑?** 选项设置为**开**。</span><span class="sxs-lookup"><span data-stu-id="495d1-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="495d1-110">如果接收人应通过电子邮件收到您已与他们共享指南的通知，请将**向接收人发送电子邮件通知**选项设置为**开**。</span><span class="sxs-lookup"><span data-stu-id="495d1-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="495d1-111">为入职指南设置访问权限和电子邮件首选项</span><span class="sxs-lookup"><span data-stu-id="495d1-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="495d1-112">输入名称或电子邮件地址，然后在下拉列表中选择预期接收人。</span><span class="sxs-lookup"><span data-stu-id="495d1-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="495d1-113">在**管理访问权限**选项卡上，您可以查看或更改哪些接收人可以处理指南。</span><span class="sxs-lookup"><span data-stu-id="495d1-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="495d1-114">若要更改接收人的访问权限，在接收人姓名旁边的下拉列表中，根据需要选择**可以查看**、**可以编辑**或**删除**。</span><span class="sxs-lookup"><span data-stu-id="495d1-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="495d1-115">在您完成添加或修改接收人时，请选择**更新**。</span><span class="sxs-lookup"><span data-stu-id="495d1-115">When you've finished adding or modifying recipients, select **Update**.</span></span>
