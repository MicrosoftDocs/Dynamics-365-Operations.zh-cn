# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-july-2018"></a>Dynamics 365 for Talent Core HR（2018 年 7 月）中的新增功能或更改

[!include [banner](includes/banner.md)]

此主题介绍了 Microsoft Dynamics 365 for Talent Core HR 中的新增功能和更改的功能。

## <a name="powerapps-personalization"></a>PowerApps 个性化设置

Microsoft Dynamics 365 for Talent 支持与 Microsoft PowerApps 服务集成。 开发人员和非技术用户都可以使用 PowerApps 在不编写代码的情况下构建适用于移动设备、平板电脑和 Web 的自定义业务应用程序。 而您、您的组织或者更广泛的生态系统使用 PowerApps 开发的应用程序则可以嵌入到 Talent 客户端以细分产品的功能。 例如，您可以构建应用程序以使用从其他系统检索的信息补充 Talent。

有关详细信息，请参阅[嵌入 PowerApps 应用程序](../fin-and-ops/get-started/embed-power-apps.md)。

## <a name="ceridian-payroll-integration"></a>Ceridian 工资单集成

美国、加拿大和墨西哥现在支持 Talent 与 Ceridian Dayforce 集成。 此集成使用各种数据类别。 下面举了一些示例加以说明：

- 人力资源数据
- 薪酬数据
- 工资单数据，如付薪周期、付薪期间和收入代码
- 工作人员数据

有关详细信息，请参阅 [配置工资单集成](configure-payroll-integration.md)。

## <a name="worker-tax-regions-have-been-expanded-beyond-the-us"></a>工作人员税区已扩展到了美国以外

已增加了对美国以外税区的支持。 税区分配给工作人员后，可以推动纳税计算和用于与外部工资单的集成。

## <a name="the-title-field-has-been-expanded-in-talent"></a>Talent 中已扩展了标题字段

此更新中已扩展了标题字段。 该字段现在有 65 个字符。 已在选择了标题的所有位置（如工作人员、职位和工作）实施了此项更改。

## <a name="benefit-enrollment-status-report"></a>福利登记状态报表

有关福利的开放登记的内置报告功能可以帮助您轻松了解员工在开放登记流程的哪个阶段。 您可以了解多少位员工已完成了该流程，多少位员工正在完成，多少位员工尚未开始。 此外，您可以迅速查看员工登记期间出现的任何问题和所有员工提交情况的完整日志。 因此，您可以轻松验证和审核员工提交的信息。
