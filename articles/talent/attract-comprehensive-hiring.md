# <a name="attract-comprehensive-hiring-add-on-capabilities"></a>Attract 全面的雇用加载项功能

[!include[banner](../includes/banner.md)]

Attract 内不同的组织值具有不同的功能集。 因此，有两个“级别”的 Attract 产品。 有针对较小团队或组织的 Attract 产品，其中招聘经理是招聘流程的主要用户。 还有可以添加到 Attract 产品的综合招聘加载项（额外付费），针对大中团队或组织，其中招聘人员或人才专家是招聘流程的主要用户。
有关定价和许可的详细信息，请参阅最新的[许可指南](https://mbs.microsoft.com/Files/public/365/Dynamics365LicensingGuide.pdf)。

Attract 的免费试用版在试用期间自动包括综合招聘加载项。 如果您的组织需要综合招聘加载项，请找到[本地 Dynamics 365 for Talent 合作伙伴](https://dynamics.microsoft.com/partners/find-a-partner/)帮助您满足需求。
