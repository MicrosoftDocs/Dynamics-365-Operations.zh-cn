# <a name="register-driver-check-in-and-check-out-for-an-appointment"></a>登记约会的驾驶员签入和签出

[!include [banner](../../includes/banner.md)]

此过程显示如何登记驾驶员签入和驾驶员签出。这通常由运输协调员完成。 您可以在 USMF 演示数据公司中使用此过程。 首先必须为负荷设置预约。 若要创建预约，可以提前运行“为负荷设置预约”过程。


## <a name="select-an-appointment"></a>选择预约
1. 转到“运输管理”>“计划”>“码头预约计划”>“驾驶员签入和签出”。
2. 选择预约。

## <a name="register-driver-check-in"></a>登记驾驶员签入
1. 单击“驾驶员签入”。
2. 在“拖车编号”字段中，键入一个值。
3. 在“驾驶员名称”字段中，键入一个值。
4. 在“驾照”字段中，键入一个值。
5. 单击“确定”。

## <a name="register-driver-check-out"></a>登记驾驶员签出
1. 单击“驾驶员签出”。
2. 单击“确定”。
