# <a name="set-up-an-appointment-for-a-load"></a>为负荷设置约会

[!include [banner](../../includes/banner.md)]

此过程显示如何为负荷设置和计划码头预约。 这通常由运输协调员完成。 您可以在 USMF 演示数据公司中使用此过程。


## <a name="select-the-load"></a>选择负荷。
1. 转到“运输管理”>“计划”>“装载计划工作台”。
2. 清除“隐藏已装运和已接收”复选框。
3. 在列表中，选择状态为“已装运”的负荷。
4. 单击“运输”。
5. 单击“预约计划”。

## <a name="create-an-appointment"></a>创建预约
1. 单击“新建”。
2. 在“预约规则”字段中，输入或选择一个值。
3. 单击“保存”。
4. 单击“更新状态”。
5. 单击“确认”。
6. 单击“保存”。
7. 关闭该页面。
