# <a name="set-up-transportation-constraints-for-an-item"></a>设置物料的运输约束

[!include [banner](../../includes/banner.md)]

此过程将设置运输约束以阻止所选物料通过所选中心运输。 此任务通常由运输协调员完成。 您可以在 USMF 演示数据公司，也可在您自己的数据中运行该过程。


## <a name="create-an-item-constaint"></a>创建物料约束
1. 转到“约束”。
2. 单击“新建”。
3. 在“物料约束”字段中，输入一个值。
4. 在“名称”字段中，键入一个值。
5. 在“站点”字段中，输入或选择一个值。
6. 在“仓库”字段中，输入或选择一个值。
7. 在“物料编号”字段中，输入或选择一个值。
8. 在“中心”字段中，输入或选择一个值。
9. 在“约束操作”字段中，选择一个选项。
10. 单击“保存”。
11. 关闭该页面。
