# <a name="move-service-order-lines"></a><span data-ttu-id="e9ef4-103">移动服务订单行</span><span class="sxs-lookup"><span data-stu-id="e9ef4-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="e9ef4-104">单击 **服务管理** \> **通用** \> **服务订单** \> **服务订单**。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="e9ef4-105">双击包含您要移到其他服务订单的行的服务订单。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="e9ef4-106">在 **服务订单** 窗体中的操作窗格上，单击 **移动行** 按钮。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="e9ef4-107">在 **移动服务订单行** 窗体上的 **服务订单** 字段中，选择要将行移到的服务订单。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="e9ef4-108">如果要删除从中移走了服务订单行的服务订单，则选中 **删除空服务订单** 复选框。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="e9ef4-109">单击 **确定**。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="e9ef4-110">您只能在附加到相同服务协议的服务订单之间移动服务订单行。</span><span class="sxs-lookup"><span data-stu-id="e9ef4-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
