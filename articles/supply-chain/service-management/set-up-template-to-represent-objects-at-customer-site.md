# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="f45d3-103">设置表示客户站点中对象的模板</span><span class="sxs-lookup"><span data-stu-id="f45d3-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="f45d3-104">使用此过程设置表示客户站点上的对象的模板。</span><span class="sxs-lookup"><span data-stu-id="f45d3-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="f45d3-105">单击**服务管理** \> **常用** \> **服务协议** \> **服务协议**。</span><span class="sxs-lookup"><span data-stu-id="f45d3-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="f45d3-106">双击某个服务协议，或单击**服务协议**以创建一个新的服务协议。</span><span class="sxs-lookup"><span data-stu-id="f45d3-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="f45d3-107">单击**设置**选项卡。</span><span class="sxs-lookup"><span data-stu-id="f45d3-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="f45d3-108">单击**服务对象**以将模板物料清单 (BOM) 附加到该服务协议。</span><span class="sxs-lookup"><span data-stu-id="f45d3-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="f45d3-109">在**服务对象**窗体中，选择一个服务对象，然后单击**功能** \> **附加物料清单模板**。</span><span class="sxs-lookup"><span data-stu-id="f45d3-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="f45d3-110">在**附加物料清单模板**窗体中，在**材料清单模板**字段中选择材料清单模板，然后单击**确定**。</span><span class="sxs-lookup"><span data-stu-id="f45d3-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="f45d3-111">在**服务对象**窗体中，单击**设计器**以修改物料清单模板，以便该物料清单表示您想要的对象。</span><span class="sxs-lookup"><span data-stu-id="f45d3-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="f45d3-112">关闭窗体以保存您所做更改。</span><span class="sxs-lookup"><span data-stu-id="f45d3-112">Close the forms to save your changes.</span></span>
