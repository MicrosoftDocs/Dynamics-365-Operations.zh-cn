# <a name="manage-template-boms-on-object-relations"></a>管理针对对象关系的物料清单模板 

[!include [banner](../includes/banner.md)]


## <a name="attach-a-template-bom-to-a-service-object"></a>将物料清单模板附加到某一服务对象

1.  单击 **服务管理** \> **常用** \> **服务协议** \> **服务协议**。

2.  双击您要将物料清单模板附加到某一对象关系的服务协议。

3.  单击 **设置** \> **服务对象**。

4.  在 **服务对象** 窗体中，选择要将物料清单模板附加到的对象，然后单击 **功能** \> **附加物料清单模板**。

5.  在 **选择物料清单模板** 对话框中，选择某一物料清单模板，然后单击 **确定**。

6.  关闭窗体以保存您所做更改。

## <a name="delete-a-service-bom-from-a-service-object"></a>从服务对象删除服务物料清单

1.  单击 **服务管理** \> **常用** \> **服务协议** \> **服务协议**。

2.  双击您要从某一对象关系删除服务物料清单的服务协议。

3.  单击 **设置** \> **服务对象**。

4.  在 **服务对象** 窗体中，选择具有您要删除的服务物料清单的对象，单击 **功能** \> **删除服务物料清单**。

5.  关闭窗体以保存您所做更改。

## <a name="move-the-service-bom-history-from-one-service-agreement-to-another"></a>将服务物料清单历史记录从一个服务协议移到另一个服务协议

1.  单击 **服务管理** \> **常用** \> **服务协议** \> **服务协议**。

2.  双击从协议服务移动的服务物料清单。

3.  单击 **设置** \> **服务对象**。

4.  在 **服务对象** 窗体中，选择具有您要移动的服务物料清单的对象，单击 **功能** \> **移动服务物料清单**。

5.  在 **选择服务对象关系** 窗体中，选择将服务物料清单移动到的服务对象关系，然后单击 **确定**。

6.  关闭窗体以保存您所做更改。

## <a name="modify-the-information-displayed-for-a-bom-line"></a>修改为物料清单行显示的信息

1.  单击 **服务管理** \> **设置** \> **服务对象** \> **物料清单模板**。

2.  选择某一物料清单模板，然后单击 **设计器**。

3.  单击 **设置** 选项卡。在 **物料清单** 区域，请选中信息复选框以显示您的服务物料清单和物料清单模板的物料清单行。

4.  关闭窗体以保存您所做更改。
