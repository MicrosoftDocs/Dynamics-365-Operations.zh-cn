[!include [banner](../includes/banner.md)]

2.  <span data-ttu-id="e18a7-105">在**所有服务预订**列表页中，选择要为其创建预订费用交易记录的预订。</span><span class="sxs-lookup"><span data-stu-id="e18a7-105">On the **All service subscriptions** list page, select the subscription for which you want to create a subscription fee transaction.</span></span>

5.  <span data-ttu-id="e18a7-108">单击**确定**。</span><span class="sxs-lookup"><span data-stu-id="e18a7-108">Click **OK**.</span></span>
