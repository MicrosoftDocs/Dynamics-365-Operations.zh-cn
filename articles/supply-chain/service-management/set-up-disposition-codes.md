[!include [banner](../includes/banner.md)]


<span data-ttu-id="a3215-104">您可以设置处置代码指定如何处理客户退回的物料。</span><span class="sxs-lookup"><span data-stu-id="a3215-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="a3215-105">例如，创建称作 **修复并返回** 的处置代码指示将修复退回物料然后返回给客户。</span><span class="sxs-lookup"><span data-stu-id="a3215-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="a3215-106">有关通常用于客户退回物料的处置代码的详细示例，请参阅[指定如何处置退回物料](specify-how-to-dispose-of-returned-items.md)。</span><span class="sxs-lookup"><span data-stu-id="a3215-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="a3215-107">还可以设置原因代码帮助解释为何退货。</span><span class="sxs-lookup"><span data-stu-id="a3215-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="a3215-108">有关原因代码的详细信息，请参阅[设置退货原因代码](set-up-return-reason-code.md)。</span><span class="sxs-lookup"><span data-stu-id="a3215-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="a3215-109">单击 **销售和市场营销** \> **设置** \> **销售订单** \> **退货** \> **处置代码**。</span><span class="sxs-lookup"><span data-stu-id="a3215-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="a3215-110">单击 **新建** 或按 CTRL+N 创建新的处置代码。</span><span class="sxs-lookup"><span data-stu-id="a3215-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="a3215-111">输入唯一的描述性名称，选择行动，然后输入处置代码的描述。</span><span class="sxs-lookup"><span data-stu-id="a3215-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="a3215-112">如果要将任何客户费用与此处置代码关联，请单击 **费用** 按钮以打开 **设置费用** 窗体。</span><span class="sxs-lookup"><span data-stu-id="a3215-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="a3215-113">如果要定义与公司自己的处置代码匹配的任何外部代码，请单击 **外部代码** 按钮以打开 **外部代码** 窗体。</span><span class="sxs-lookup"><span data-stu-id="a3215-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>
