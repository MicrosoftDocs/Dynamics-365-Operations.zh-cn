# <a name="set-up-service-activity-types"></a>设置服务活动类型 

[!include [banner](../includes/banner.md)]


1.  单击**销售和市场营销** \> **设置** \> **活动** \> **活动类型**。
    
    –或–
    
    单击**组织管理** \> **设置** \> **活动** \> **活动类型**。

2.  在**活动类型**窗体中，按 CTRL+N 添加新的活动类型行。

3.  在**类型**字段中，输入活动类型的名称。

4.  在**说明**字段中，输入描述此活动类型的文本。

5.  关闭窗体以保存您所做更改。
