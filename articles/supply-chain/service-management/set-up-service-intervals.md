# <a name="set-up-service-intervals"></a><span data-ttu-id="478d7-103">设置服务间隔</span><span class="sxs-lookup"><span data-stu-id="478d7-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="478d7-104">服务间隔指示在您创建服务订单时为服务协议行创建服务订单行所采用的频率。</span><span class="sxs-lookup"><span data-stu-id="478d7-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="478d7-105">单击 **服务管理** \> **设置** \> **服务协议** \> **服务间隔**。</span><span class="sxs-lookup"><span data-stu-id="478d7-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="478d7-106">创建新的服务间隔。</span><span class="sxs-lookup"><span data-stu-id="478d7-106">Create a new service interval.</span></span>
3. <span data-ttu-id="478d7-107">输入 ID 和服务间隔的描述。</span><span class="sxs-lookup"><span data-stu-id="478d7-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="478d7-108">在 **范围** 字段中，选择范围。</span><span class="sxs-lookup"><span data-stu-id="478d7-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="478d7-109">在 **频率** 字段中，输入频率。</span><span class="sxs-lookup"><span data-stu-id="478d7-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="478d7-110">必须将该频率作为系数与范围相乘，以获取服务协议间隔。</span><span class="sxs-lookup"><span data-stu-id="478d7-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="478d7-111">按 **Alt+S** 保存服务间隔。</span><span class="sxs-lookup"><span data-stu-id="478d7-111">Press **Alt+S** to save the service interval.</span></span>
