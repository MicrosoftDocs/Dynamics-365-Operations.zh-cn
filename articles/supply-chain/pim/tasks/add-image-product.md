# <a name="add-an-image-to-a-product"></a>向产品添加图像

[!include [banner](../../includes/banner.md)]

此过程显示如何向产品添加图像。用于创建此任务的演示数据公司为 USMF。

1. 转到“产品信息管理”>“产品”>“已发布产品”。
2. 展开“产品图像”速见表。
3. 在列表中，找到并选择所需记录。
    * 例如，选择物料编号 C0002。  
4. 单击“更改图像”。
5. 单击“新建”。
    * 单击浏览。 找到要使用的图像，选择，然后关闭对话框。
