# <a name="create-a-bar-code-for-a-product"></a>为产品创建条码

[!include [banner](../../includes/banner.md)]

本主题显示以物料编号 M0001 为例如何手动创建条码。 创建此程序的演示数据公司是 USMF。

1. 在主页中选择**已发布产品的维护**。
2. 转到**链接**部分下的**产品 > 已发布产品**。
3. 在列表中，找到并选择所需记录。 在本示例中，选择物料编号 **M0001**。
4. 在操作窗格上，选择**库存管理**。
5. 选择**条码**。
6. 选择**新建**。
7. 标记下面的列表中创建的所选行。
8. 在**条码设置**字段中，输入或选择一个值。
9. 在**条码**字段中，输入或选择一个值。
10. 在**条码**字段中，输入一个值。  
11. 在**数量**字段中，输入一个数字。
12. 选择**保存**。
13. 关闭该页面。
