# <a name="create-product-model-components"></a>创建产品模型组件

[!include [task guide banner](../../includes/task-guide-banner.md)]

基于约束的产品配置模型使用组件作为基本结构块以建立产品模型结构。 这些组件必须在可构建产品模型前创建。 创建此程序的演示数据公司是 USMF。

1. 单击“产品变型模型定义”。
2. 单击“产品配置模型”。
3. 在“操作窗格”上单击“模型”。
4. 单击“组件”。
5. 单击“新建”。
6. 在“名称”字段中，键入一个值。
7. 在“描述”字段中，键入一个值。
8. 关闭该页面。
