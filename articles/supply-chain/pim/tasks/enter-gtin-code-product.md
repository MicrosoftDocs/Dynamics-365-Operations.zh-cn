# <a name="enter-the-gtin-code-for-a-product"></a>输入产品的 GTIN 代码

[!include [banner](../../includes/banner.md)]

GTIN 代码由 GS1 组织分配。 在此示例中，您将为演示数据公司 USMF 中的产品编号 M0001 输入代码。

1. 单击“已发布产品的维护”。
2. 单击“已发布产品”。
3. 展开“产品图像”速见表。
4. 在列表中，找到并选择所需记录。
5. 在“操作窗格”上，单击“库存管理”。
6. 单击“GTIN 代码”。
7. 单击“新建”。
8. 在列表中，标记所选的行。
9. 在“单位”字段中，输入或选择一个值。
10. 在“GTIN 代码”字段中，选择一个选项。
11. 在“GTIN”字段中，键入一个值。
12. 关闭该页面。
