# <a name="create-a-released-product-for-a-single-company"></a>创建单个公司的已发布产品

[!include [banner](../../includes/banner.md)]

此过程介绍如何在单个法定单位背景下创建单个已发布产品。 在创建已发布产品后，其仅立即在该单位中可用。 您可以使用 USMF 公司演示数据浏览此过程。 此任务通常由产品设计者执行。


## <a name="create-a-released-product"></a>创建已发布产品
1. 转至“产品发布”。
2. 单击“新建”。
3. 在“产品编号”字段中，键入一个值。
    * 如果没有在“产品编号”字段中自动输入产品编号，则键入唯一的产品编号。 该步骤只有在还未设置产品编号的编号顺序时才需要。  
4. 在“产品名称”字段中，键入一个值。
    * 产品名称默认为搜索名称。 如果需要，您可以选择更改搜索名称。  
5. 在“物料模型组”字段中，单击下拉按钮以打开查找。
    * 此物料模型组包含确定在进行物料收货和发货时如何控制和处理物料的设置。 该设置还确定如何计算物料消耗量。  
6. 在列表中，找到并选择所需记录。
7. 在列表中，单击所选行中的链接。
8. 在“物料组”字段中，单击下拉按钮以打开查找。
    * 物料组基于物料特征将库存物料分成不同的组，可用于管理库存。 例如，若要管理信息如何过帐到主科目，您可以创建与特定的主科目关联的一系列不同物料组。 这让您能够在不同阶段跟踪物料的库存价值。  
9. 在列表中，找到并选择所需记录。
10. 在列表中，单击所选行中的链接。
11. 在“存储维度组”字段中，单击下拉按钮以打开查找。
    * 存储维度帮助您控制物料存储方式和库存提取方式。 例如，存储维度可以是站点和仓库。  
12. 在列表中，找到并选择所需记录。
13. 在列表中，单击所选行中的链接。
14. 在“跟踪维度组”字段中，单击下拉按钮以打开查找。
    * 跟踪维度组确定可以添加到产品中的跟踪维度。 例如，批号和序列号用于跟踪库存物料。  
15. 在列表中，找到并选择所需记录。
16. 在列表中，单击所选行中的链接。
17. 在“库存单位”字段中，单击下拉按钮以打开查找。
    * 库存单位确定产品在存于库存时如何量化产品。  
18. 在列表中，找到并选择所需记录。
19. 在列表中，单击所选行中的链接。
20. 在“采购单位”字段中，单击下拉按钮以打开查找。
    * 采购单位确定从供应商购买产品时如何量化产品。  
21. 在列表中，找到并选择所需记录。
22. 在列表中，单击所选行中的链接。
23. 在“销售单位”字段中，单击下拉按钮以打开查找。
    * 销售单位确定向客户销售产品时如何量化产品。  
24. 在列表中，找到并选择所需记录。
25. 在列表中，单击所选行中的链接。
26. 在“物料清单单位”字段中，单击下拉按钮以打开查找。
    * 物料清单单位确定在添加产品到物料清单时如何量化产品。  
27. 在列表中，找到并选择所需记录。
28. 在列表中，单击所选行中的链接。
29. 在“物料销售税组”字段中，单击下拉按钮以打开查找。
    * 销售税务组中的物料销售税组确定如何计算每个物料的销售税。  
30. 在列表中，找到并选择所需记录。
31. 在列表中，单击所选行中的链接。
32. 在“物料销售税组”字段中，单击下拉按钮以打开查找。
    * 采购税务组中的物料销售税组确定如何计算每个物料的采购税。  
33. 在列表中，找到并选择所需记录。
34. 在列表中，单击所选行中的链接。
35. 单击“确定”。

## <a name="add-product-characteristics"></a>添加产品特征。
1. 展开或折叠“管理库存”部分。
2. 在“净重”字段中，输入一个数字。
3. 在“皮重”字段中，输入一个数字。
4. 在“总深度”字段中，输入一个数字。
5. 在“总宽度”字段中，输入一个数字。
6. 在“总高度”字段中，输入一个数字。
7. 在“体积”字段中，输入一个数字。

## <a name="add-financial-dimensions"></a>添加财务维度
1. 展开或折叠“财务维度”部分。
2. 在“业务单位”字段中，单击下拉按钮以打开查找。
3. 在列表中，找到并选择所需记录。
4. 在列表中，单击所选行中的链接。
5. 在“成本中心”字段中，单击下拉按钮以打开查找。
6. 在列表中，找到并选择所需记录。
7. 在列表中，单击所选行中的链接。
8. 在“部门”字段中，单击下拉按钮以打开查找。
9. 在列表中，找到并选择所需记录。
10. 在列表中，单击所选行中的链接。
11. 在“物料组”字段中，单击下拉按钮以打开查找。
12. 在列表中，找到并选择所需记录。
13. 在列表中，单击所选行中的链接。
