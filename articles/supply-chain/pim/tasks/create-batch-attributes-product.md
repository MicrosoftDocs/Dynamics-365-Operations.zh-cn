# <a name="create-batch-attributes-for-a-product"></a>创建产品的批属性

[!include [banner](../../includes/banner.md)]

此过程显示如何创建批属性，分配默认值范围和在组中加入属性。 使用 USP2 公司演示数据创建此过程。

1. 转到“库存管理”>“设置”>“批次”>“批属性”。
2. 单击“新建”。
3. 在“属性”字段中，键入一个值。
4. 在“描述”字段中，键入一个值。
5. 在“属性类型”字段中，选择“分数”。
    * 此过程使用“分数”类型以启用小数值。 您可以选择其他属性类型。 如果您选择列举类型，则您必须在列举表中输入数值才能在“目标”字段输入数值。  
6. 在“最小值”字段中，输入一个数字。
7. 在“最大值”字段中，输入一个数字。
8. 在“增量”字段中，输入一个数字。
9. 在“目标”字段中，键入一个值。
10. 单击“保存”。
11. 关闭该页面。
12. 转到“库存管理”>“设置”>“批次”>“批属性组”。
13. 单击“新建”。
14. 在“属性组”字段中，键入一个值。
15. 在“描述”字段中，键入一个值。
16. 单击“保存”。
17. 单击“组属性”。
18. 单击“新建”。
19. 在“属性”字段中，单击下拉按钮以打开查找。
20. 在列表中，找到并选择所需记录。
21. 在列表中，单击所选行中的链接。
    * 属性可包含在任何组中。  
22. 单击“保存”。
23. 关闭该页面。
