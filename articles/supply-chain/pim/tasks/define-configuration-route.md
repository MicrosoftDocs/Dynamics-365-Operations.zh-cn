# <a name="define-configuration-routes"></a>定义配置工艺路线

[!include[task guide banner](../../includes/task-guide-banner.md)]

该过程专注于定义配置流程，以确定配置组将会显示的序列。 创建此程序的演示数据公司是 USMF。 这是八个过程中第六个说明如何构建基于维度的配置组合的过程。

1. 转到“产品信息管理”>“物料清单和配方”>“物料清单”。
2. 在列表中，找到并选择所需记录。
3. 在“操作窗格”上，单击“选项”。
4. 单击“更改”视图。
5. 单击“标题”视图。
6. 展开或折叠“配置流程”部分。
7. 单击“添加”。
8. 在列表中，标记所选的行。
9. 在“配置组”字段中，单击下拉按钮以打开查找。
10. 在列表中，单击所选行中的链接。
11. 单击“添加”。
12. 在列表中，标记所选的行。
13. 在“配置组”字段中，单击下拉按钮以打开查找。
14. 在列表中，找到并选择所需记录。
15. 在列表中，单击所选行中的链接。
16. 单击“保存”。
