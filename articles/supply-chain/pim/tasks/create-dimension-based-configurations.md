# <a name="create-dimension-based-configurations"></a>创建基于维度的配置

[!include [banner](../../includes/banner.md)]

此过程显示如何定义基于维度的产品的配置。 这是此系列中最后一个说明如何构建基于维度的配置组合的过程。 此过程的执行取决于在前七个记录中创建的数据。 创建此程序的演示数据公司是 USMF。


## <a name="find-the-dimension-based-product-master"></a>查找基于维度的基础产品
1. 单击“已发布产品的维护”。
2. 单击“已发布产品”。
3. 在列表中，标记所选的行。
    * 选择您以 8 个记录的此顺序在第一个记录中创建的基于维度的基础产品。  

## <a name="create-configurations"></a>创建配置
1. 在“工程操作窗格”上单击“维护配置”。
2. 单击“配置”。
3. 在列表中，标记所选的行。
4. 在“物料编号”字段中，输入或选择一个值。
    * 选择第一个配置组中的任意物料。  
5. 在列表中，找到并选择所需记录。
6. 在“物料编号”字段中，输入或选择一个值。
    * 从第二个配置组中选择任意物料。  
7. 单击“确定”。
8. 在列表中，标记所选的行。
9. 在“配置”字段中，键入一个值。
    * 输入将便于标识配置的名称。  
10. 在“描述”字段中，键入一个值。
    * 输入说明配置包含的内容的配置描述。  
11. 单击“确定”。
