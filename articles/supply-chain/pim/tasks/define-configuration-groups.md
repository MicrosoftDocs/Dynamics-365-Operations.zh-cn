# <a name="define-configuration-groups"></a>定义配置组

[!include[task guide banner](../../includes/task-guide-banner.md)]

该过程会显示如何通过在演示数据公司 USMF 公司中创建两个组，定义产品配置组。 这是八个过程中第四个说明如何构建基于维度的配置组合的过程。

1. 转到“产品信息管理”>“设置”>“物料清单和配方”>“配置组”。
2. 单击“新建”。
3. 在“配置组”字段中，键入一个值。
    * 键入将用于物料清单行的配置组的名称。  
4. 在“名称”字段中，键入一个值。
5. 单击“新建”。
6. 在“配置组”字段中，键入一个值。
    * 键入将用于物料清单行的配置组的名称。  
7. 在“名称”字段中，键入一个值。
8. 关闭该页面。
