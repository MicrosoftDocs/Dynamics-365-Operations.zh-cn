# <a name="hazardous-materials-in-products-orders-shipments-and-loads"></a>产品、订单、装运和负荷中的危险物料

[!include [banner](../includes/banner.md)]

本文说明如何为已发布产品设置危险物料属性，如何为危险物料设置库存限制，以及如何在销售订单、装运或负荷中包括危险物料。

## <a name="set-hazardous-material-specifications-for-products"></a>为产品设置危险物料规范

在按照[设置危险物料](hazmat-setup.md)中的说明定义了危险物料法规并设置了相关的参考代码之后，可以将此信息与已发布物料关联。 装运文档的装运文本将从已发布物料的危险物料信息中提取。

在将已发布物料与危险物料关联的过程中，必须指定法规代码和物料。 根据运输方式，可能将不同的法规代码与物料关联，您可以将多个法规和物料代码与每个物料关联。

要将已发布产品设置为危险物料，请按照以下步骤操作。

1. 转到 **产品信息管理 \> 产品 \> 已发布产品**。
1. 选择或创建一个产品以打开其 **已发放产品详细信息** 页。
1. 在 **管理库存** 快速选项卡上，将 **危险物料** 选项设置为 **是**。 此设置将物料标识为危险货物，在打印装运文档时使用。
1. 在操作窗格上 **管理库存** 选项卡的 **合规性** 组中，选择 **危险物料**。
1. 通过使用以下小节中介绍的字段，填充所选物料的 **危险物料** 页面。

### <a name="item-hazardous-materials-header"></a>危险物料标题

下表描述了 **危险物料** 页面顶部的可用字段。

| 字段 | 说明 |
|---|---|
| 物料编号 | 您正在处理的已发布产品。 |
| 监管代码 | 选择将应用于产品的危险物料法规。 法规定义如何为物料创建打印装运文本以及关联的交货方式。 分配后，将无法在此页面上编辑代码。 不过，您可以通过选择 **新** 来分配新的法规代码。 |
| 物料代码 | （可选）选择将应用于产品的危险物料代码。 物料代码提供包含此页面上很多其他字段的默认值的模板。 选择代码时，其所有危险物料规范将被复制到当前产品。 但是，系统会首先提示您确认应该从物料代码填充物料数据。 |
| 组代码 | （可选）选择将应用于产品的危险物料分类组代码。 对于 **物料代码** 字段，当您选择代码时，其所有危险物料规范将被复制到当前产品。 但是，系统会首先提示您确认应该从组代码填充物料类组数据。 |

### <a name="descriptions-fasttab"></a><a name="hazmat-description"></a>“描述”快速选项卡

下表说明了 **描述** 快速选项卡上可用的字段。

| 字段 | 说明 |
|---|---|
| 正确装运名称 | 输入适用法规指定的物料的标准描述。 您可以在 **物料装运文本翻译** 上为此值提供翻译，如下一节所述。 |
| 技术名称 | 选择物料的公用名称或通用名称。 此名称可能是您公司内部使用的物料名称。 |
| 未另行说明 | 选中此复选框指示 **正确装运名称** 值是物料的未另行说明 (N.O.S.) 装运名称。 未另行说明 装运名称用于具有特定最终用途，但可能在特定法规的危险品表中未按名称列出的相似化学品和物料的组。 |

### <a name="item-ship-text-translation-fasttab"></a>“物料装运文本翻译”快速选项卡

**物料装运文本翻译** 快速选项卡包含一个网格，其中显示在 **描述** 快速选项卡上为主要语言定义的 **正确装运名称** 值的翻译。 这些翻译可用于一种或多种其他语言的装运打印文本。

下表说明了 **物料装运文本翻译** 快速选项卡上可用的字段。


| 字段 | 说明 |
|---|---|
| 语言 | 该行使用的语言代码。 例如，**pt-br** 表示巴西葡萄牙语。 |
| 装运打印文本 | 以该行使用的语言显示的翻译后的 **正确装运名称** 值。 |

要添加或编辑翻译，请选择网格上方的 **翻译** 打开 **文本翻译** 页面。 然后按照以下步骤之一操作：

- 要添加新翻译，在操作窗格上，选择 **添加**。 选择要添加的语言，然后在 **翻译文本** 字段中，输入翻译后的文本。
- 要编辑现有翻译，请根据需要在 **语言** 字段中选择目标语言，然后在 **翻译文本** 字段中编辑翻译后的文本。

### <a name="material-management-fasttab"></a><a name="material-management"></a>“物料管理”快速选项卡

下表说明了 **物料管理** 快速选项卡上可用的字段。

| 字段 | 说明 |
|---|---|
| 分类 | 选择产品所属的危险物料类，类由您所遵循的法规定义。 您必须为每个包含危险物料的产品分配分部和类。 |
| 说明 | 为在 **类** 字段中选择的类定义的描述。 此字段是只读的。 |
| 分部 | 选择产品所属的危险物料分部，分部由您所遵循的法规定义。 分部是类的子集。 您必须为每个包含危险物料的产品分配分部和类。 |
| 身份证明 | 选择危险物料标识代码。 通常，此代码基于联合国 (UN) 标准。 |
| 包装组 | 选择将应用于当前物料的包装组。 |
| 说明 | 为在 **包装组** 字段中选择的组定义的描述。 此字段是只读的。 |
| 包装描述 | 选择适用的包装描述代码。 此代码引用指示产品必须采用的包装方式的描述。 |
| 危险物料标签 | 选择引用应该应用于产品的适用危险货物标签的代码。 |
| 数量有限 | 将此选项设置为 **是** 将报告每个负荷和每个负荷行中包含的产品总重量。 |
| 数量 | 以指定单位输入产品中危险物料的数量。 此值将用于计算包含产品的负荷和装运的危险物料总分数。 |
| 乘数 | 输入为包含产品的每个负荷行计算危险物料分数时所应用的乘数。 此值由适用法规根据产品中包含的危险物料的类型指定。 |
| 单位 | 选择将应用于产品中危险物料数量的度量单位，度量单位在 **数量** 字段中指定。 此值将用于计算包含产品的负荷和装运的危险物料总分数。 |

#### <a name="how-the-hazardous-material-score-is-calculated"></a>危险物料分数如何计算

在产品的 **物料管理** 快速选项卡上指定的几个值用于计算包括该产品的每个负荷行的 *危险物料分数*。 此分数使用以下公式计算：

危险物料分数 = *&lt;LineQty&gt;* × *&lt;HazmatQty&gt;* × *&lt;UnitConversion&gt;* × *&lt;乘数&gt;*

以下是公式的关键：

- *&lt;LineQty&gt;* 是为负荷行指定的产品数量。
- *&lt;HazmatQty&gt;* 是在 **物料管理** 快速选项卡的 **数量** 字段中为产品指定的危险物料数量。
- *&lt;UnitConversion&gt;* 是在用于负荷行数量的单位与在 **物料管理** 快速选项卡的 **单位** 字段中为产品指定的单位之间进行转换的换算系数。
- *&lt;乘数&gt;* 是在 **物料管理** 快速选项卡的 **乘数** 字段中为产品指定的乘数。

对于包含指定了这些值的产品的每个负荷行，都将报告此分数。 有关详细信息，请参阅本文后面的[包含危险物料的装运](#hazmat-shipments)和[包含危险物料的负荷](#hazmat-loads)两节。

#### <a name="how-the-hazardous-material-weight-is-calculated"></a>危险物料重量如何计算

包含 **物料管理** 快速选项卡上的 **限制数量** 选项设置为 **是** 的产品的负荷和负荷行，将显示危险物料的总重量，如本文后面的[包含危险物料的装运](#hazmat-shipments)和[包含危险物料的负荷](#hazmat-loads)两节中所述。 危险物料重量使用以下公式计算：

危险物料重量 = *&lt;LineQty&gt;* × *&lt;ProductWeight&gt;* × *&lt;UnitConversion&gt;*

以下是公式的关键：

- *&lt;LineQty&gt;* 是为负荷行指定的产品数量。
- *&lt;ProductWeight&gt;* 是为产品指定的净重，以为产品指定的库存单位为单位。
- *&lt;UnitConversion&gt;* 是在用于负荷行数量的单位与用于 *&lt;ProductWeight&gt;* 的库存单位之间转换的换算系数。

## <a name="set-stock-limits-for-hazardous-products"></a><a name="stock-limits"></a>设置危险产品的存货限制

出于安全原因，您可能必须限制可以在一个位置贮存的给定产品的总量。 若要为已发布产品设置存货限制，请执行以下步骤。

1. 转到 **产品信息管理 \> 产品 \> 已发布产品**。
1. 选择一个产品以打开其 **已发布产品详细信息** 页。
1. 在操作窗格上 **管理库存** 选项卡的 **合规性** 组中，选择 **报告详细信息**。
1. 在 **危险存货限制** 和 **危险警告限制** 字段中，为所选产品设置适当的值。

**危险物料存货限制** 报表可让您监视仓库位置中危险物料的库存级别，以确保它们保持在此处建立的安全限制内。 有关详细信息，请参阅[危险物料存货限制报表](hazmat-reports.md#stock-limit-report)。

## <a name="sales-order-transactions-that-include-hazardous-materials"></a>包含危险物料的销售订单交易

要在销售订单中包含被分类为危险物料的产品，您必须将相关的装运承运人与销售订单关联。 打开销售订单，然后在 **交货** 快速选项卡上，根据需要设置 **装运承运人** 和 **承运人服务** 字段。

装运承运人还与交货方式关联。 因此，您必须确保此信息与危险物料法规保持一致。 换言之，危险物料法规中指定的交货方式必须与销售订单抬头上的规范相匹配。 这样，法规、装运承运人和服务就连接到了销售订单上使用的装运行。

在完成销售订单并准备好装运后，可以将其发布到仓库以指示销售和仓库操作之间的转移。

## <a name="shipments-that-include-hazardous-materials"></a><a name="hazmat-shipments"></a>包含危险物料的装运

### <a name="view-hazardous-material-scores-for-each-shipment-line"></a>查看每个装运行的危险物料分数

装运的 **装运详细信息** 页显示已为该装运中包含的每个负荷行计算出的危险物料总重量和分数值。 要查看分数和重量，请按照下列步骤操作。

1. 转到 **仓库管理 \> 装运 \> 所有装运**。
1. 选择一个装运以打开其 **装运详细信息** 页。
1. 在 **负荷行** 快速选项卡上，检查各个行。 对于每一行，您将在以下字段中看到危险物料的计算：

    - **危险物料分数** – 此字段显示负荷行的危险物料分数。 此值根据为适用法规和已发布产品设置中建立的规则和值计算。 此计算采用负荷行上的数量，引用已发布产品的[物料管理设置](#material-management)中的乘数。
    - **限制数量净重** – 对于由于其危险物料物质而被标记为限量产品的产品，此字段显示负荷行上包含的危险物质的总净重。 此计算基于已发布产品设置中标记为危险物质的产品。 如果某物料被标记为限量物料，计算将采用每个负荷行上的数量，引用已发布产品的[物料管理设置](#material-management)中的重量。

### <a name="check-for-compatibility-among-hazardous-materials-that-are-included-in-a-shipment"></a>检查装运中包含的危险物料之间的兼容性

系统可以评估装运中包含的所有危险物料是否适合一起装运。 为评估兼容性，系统将检查分配给装运中包含的每个产品的兼容性组。 有关详细信息，请参阅[危险物料兼容性组](hazmat-setup.md#compatibility-groups)。

要运行兼容性检查，请按照下列步骤操作。

1. 转到 **仓库管理 \> 装运 \> 所有装运**。
1. 选择一个装运以打开其 **装运详细信息** 页。
1. 在操作窗格上，在 **装运** 选项卡上的 **操作** 组中，选择 **兼容性检查**。

您会收到一条通知您检查结果的消息。

## <a name="loads-that-include-hazardous-materials"></a><a name="hazmat-loads"></a>包含危险物料的负荷

### <a name="view-hazardous-material-scores-for-each-load-line"></a>查看每个负荷行的危险物料分数

负荷的 **负荷详细信息** 页显示已为该负荷和每个负荷行计算出的危险物料总重量和分数值。 要查看分数和重量，请按照下列步骤操作。

1. 转到 **仓库管理 \> 装运 \> 所有负荷**。
1. 选择一个负荷以打开其 **负荷详细信息** 页。 （您也可以通过从相关装运选择链接来打开负荷详细信息。）
1. 在 **负荷** 快速选项卡上，您可以通过检查以下字段来查看整个负荷的危险物料总分数和重量：

    - **危险物料分数** – 此字段显示负荷的危险物料分数。 此值根据为适用法规和已发布产品设置中建立的规则和值计算。 此计算采用负荷中包含的数量，引用已发布产品的[物料管理设置](#material-management)中的乘数。
    - **限制数量净重** – 对于由于其危险物料物质而被标记为限量产品的产品，此字段显示负荷中包含的危险物质的总净重。 此计算基于已发布产品设置中标记为危险物质的产品。 如果某物料被标记为限量物料，计算将采用每个负荷中的数量，引用已发布产品的[物料管理设置](#material-management)中的重量。

1. 要查看各个行的分数和重量，请选择 **负荷行** 快速选项卡。 如上一步中所述，为每一行提供的值类似于为整个负荷提供的值。

### <a name="check-for-compatibility-among-hazardous-materials-that-are-included-in-a-load"></a>检查负荷中包含的危险物料之间的兼容性

系统可以评估负荷中包含的所有危险物料是否适合一起装运。 为评估兼容性，系统将检查分配给负荷中包含的每个产品的兼容性组。 有关详细信息，请参阅[危险物料兼容性组](hazmat-setup.md#compatibility-groups)。

要运行兼容性检查，请按照下列步骤操作。

1. 转到 **仓库管理 \> 装运 \> 所有负荷**。
1. 选择一个装运以打开其 **负荷详细信息** 页。 （您也可以通过从相关装运选择链接来打开负荷详细信息。）
1. 在操作窗格上，在 **负荷** 选项卡上的 **操作** 组中，选择 **兼容性检查**。

您会收到一条通知您检查结果的消息。
