# <a name="landed-cost-entities-overview"></a>到岸成本实体概览

[!include [banner](../includes/banner.md)]

**到岸成本** 模块通过提供从制造商到仓库的进口货运的完全财务和物流控制，帮助企业简化入站装运操作。 到岸成本的交易数据实体使外部数据源（如货运转运商服务）能够创建航行和成本，并更新集装箱跟踪记录。

这是一些支持的操作：

- 创建和更新现有入站订单行的航行、装运集装箱和帐页记录。
- 创建和更新每个成本区域的估计成本。
- 创建和更新装运集装箱活动记录以捕获集装箱旅程每一支线的开始日期、估计结束日期和实际结束日期。
- 创建供应商成本发票分配。
