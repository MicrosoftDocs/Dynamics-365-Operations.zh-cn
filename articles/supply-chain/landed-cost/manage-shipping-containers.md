# <a name="manage-shipping-containers"></a>管理装运集装箱

[!include [banner](../../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

装运集装箱用于将实际分组在一起的货物分组在一起。 它们还用于成本只能在这些货物之间共享（通常是因为它们实际分组在一起）的情况。

要通过装运集装箱页面查看和处理货物，请转到 **登陆成本 \> 装运集装箱 \> 所有装运集装箱**。 **所有装运集装箱** 页显示所有可用装运集装箱的列表。 您可以使用操作窗格上的按钮创建、删除和使用装运集装箱。 在列表中选择任意装运集装箱可以在 **装运集装箱** 页上查看它的详细信息。

装运集装箱详细信息页的上部显示装运集装箱和成本计算信息。 **行** 部分显示附加到集装箱的帐页、物料和采购订单或转移单。

## <a name="action-pane"></a>操作窗格

**所有装运集装箱** 和 **装运集装箱** 页上的操作窗格提供让您可以使用选定装运集装箱的按钮。 每个按钮执行一个操作。 操作窗格还包含选项卡，每个选项卡又提供一组相关按钮。 除非另有说明，否则以下小节中所述的所有按钮和选项卡在列表视图（即 **所有装运集装箱** 页）和详细视图（即 **装运集装箱** 页）中均可用。

### <a name="buttons-on-the-manage-tab"></a>“管理”选项卡上的按钮

下表描述了操作窗格的 **管理** 选项卡上的可用按钮。

| 按钮 | 描述 |
|---|---|
| 将收货清单过帐 | 过帐收货清单，或查看装运集装箱中所有采购订单行的产品收货清单。 如果使用多公司装运，将为每个公司打开一个新的收货清单过帐对话框。 |
| 将物料收货过帐 | 为装运集装箱中的所有采购订单行过帐产品收货。 |
| 将账单过帐 | 为装运集装箱中的所有采购订单行过帐发票。 如果使用多公司装运，将为每个公司打开一个新的发票过帐对话框。 |
| 装运转移单 | 为装运集装箱中的所有转移单行过帐转移单装运。 对话框中将仅显示装运集装箱中属于转移单类型的那些行。 |
| 接收转移单 | 为装运集装箱中的所有转移单行过帐转移单接收。 接收对话框是在装运集装箱或航行中接收货物的最简单方法，是三个可用选项之一。 您还可以通过到达日记帐或移动设备处理接收。 |
| 创建到达日记帐 | 您可以使用高级仓库功能为组织生成到达日记帐。 选项有 _初始化数量_（建议），以及 _从在途货物创建_ 或 _从采购订单创建_。 最后选择的两个选项取决于是否使用了在途货物处理。 |
| 重命名 | 打开一个对话框，您可以在其中重命名选定的装运集装箱。 |
| 更改行程模板 | 更改行程模板。 更改行程模板后，您可能必须选择 **查找自动成本** 或再次手动添加成本，因为装运成本将被删除。 |
| 转换为租赁 | 将选定的装运集装箱转换为租赁装运集装箱。 |

### <a name="buttons-on-the-general-tab"></a>“常规”选项卡上的按钮

下表描述了操作窗格的 **常规** 选项卡上的可用按钮。

| 按钮 | 描述 |
|---|---|
| 收货清单 | 为装运集装箱中的所有采购订单行过帐收货清单。 如果使用多公司航行，将为每个公司打开一个新的收货清单过帐对话框。 |
| 物料收货 | 查看产品收货记录（如果使用）。 仅当货物未使用在途货物功能时，才使用产品收货流程。 |
| 物料到达 | 查看装运集装箱的物料到达日记帐（如果使用了该日记帐）。 |
| 支线 | 支线用于标识行程的各个单独部分。 提前期可以与每个支线关联来帮助跟踪装运。 有关详细信息，请参阅[多支线行程设置](multi-leg-journey-setup.md)。 |
| 跟踪 | 查看或更新装运跟踪。 |
| 在途货物订单 | 您可以直接从集装箱打开 **在途货物** 页。 该页面仅显示所选装运集装箱的在途货物记录。 |

## <a name="header-view"></a>标头视图

要打开 **标头** 视图，打开帐页，然后选择帐页标题右上角的 **标头** 选项卡。

### <a name="settings-on-the-general-fasttab"></a>常规快速选项卡上的设置

下表描述了装运集装箱的 **标头** 视图的 **常规** 快速选项卡上的可用设置。

| 字段 | 说明 |
|---|---|
| 装运集装箱 | 装运集装箱的名称。 |
| 航行 | 与装运集装箱关联的航行。 |
| 装运集装箱类型 | 输入装运集装箱类型。 必须设置此字段。 您可以使用它，通过选择与装运集装箱类型关联的自动成本等方法来确定运费成本。 |
| 船舶 | 输入或选择船只。 如果船只未作为值列出，可以作为自由文本输入船只 ID。 在这种情况下，主表不会更新，以可以在以后在此字段中选择船只 ID。 有关详细信息，请参阅[船只](shipping-information-setup.md#vessels)。 |
| 单位类型 | 单位类型用作分组和标识装运集装箱的额外方法。 它们在装运集装箱页面上显示和选择。 有关详细信息，请参阅[设置单位类型](shipping-container-setup.md#unit-types)。 |
| 冷冻类型 | 冷冻类型用作分组和标识装运集装箱（通常是冷藏集装箱）的额外方法。 它们在装运集装箱页面上显示和选择。 有关详细信息，请参阅[设置冷冻类型](shipping-container-setup.md#refrigeration-types)。 |
| 量化指标 | 此字段支持在 **登陆成本** 模块中指定度量。 不知道商品单个体积或重量，但需要比金额或数量提供的更准确分摊的组织经常使用度量。 货运转运商将提供重量（千克）或体积度量，您可以将其放在物料或采购订单级别。 如果选择了参数，它可以自动更新，也可以手动输入。 |
| 度量单位 | 将应用于 **度量** 字段中的数字的度量单位。 |
| 实际重量 | 您可以记录纸箱或集装箱的实际重量。 此值可用于对照装运集装箱设置中允许的最大重量进行验证。 |
| 纸箱数量 | 如果选择了此参数，纸箱数量会自动更新。 |
| 货物描述 | 可以在装运集装箱或帐页标头上选择货物的描述。 它用于帮助识别货物的航行、装运集装箱或帐页。 有关详细信息，请参阅[货物的描述](shipping-information-setup.md#description-of-goods)。 |
| 内部航空运单/提单 | 您可以为装运集装箱指定内部航空运单或提单。 |
| 注解 | 与装运集装箱相关的其他信息。 |
| 可退 | 一个指示是否可以在航行之后退还装运集装箱的值。 |
| 航行状态 | 与装运集装箱关联的行程的状态。 |
| 采购订单状态 | 与装运集装箱关联的采购订单的状态。 |

### <a name="settings-on-the-delivery-fasttab"></a>交货快速选项卡上的设置

下表描述了装运集装箱的 **标头** 视图的 **交货** 快速选项卡上的可用设置。

| 字段 | 说明 |
|---|---|
| 创建日期和时间 | 创建集装箱的日期和时间。 |
| 出厂日期 | 此日期通常提供给工厂/供应商，指示您期望货物何时离开他们的场所。 当您与亚洲的工厂合作时，通常需要此日期，而不是您期望的交货截止日期。 （相比之下，对于当地交货，需要您期望的交货截止日期。）可以从装运集装箱列表中的采购订单行填写此字段。 您也可以在此处手动输入。 |
| 装运日期 | 此日期可以打印在采购订单单据上。 它通常通知工厂/供应商应将货物交付到港口的截止日期。 此字段仅供参考。 它不用于估计装运集装箱中货物的预期交货日期。 可以将此字段设置为在更新跟踪控制页面时自动更新的字段。 |
| 到店日期 | 链接到航行的采购订单中的货物可供销售的最早日期。|
| 预计交货日期 | 通常，是货物应到达仓库的日期。 此字段仅供参考。 它不用于计算装运集装箱中采购订单行上的主计划。 采购订单行上的预期交货日期通过跟踪控制更新。 可以设置此字段，让它在更新跟踪控制页面时更新。 |
| 出发日期 | 通常是飞机或船只实际离开海外港口的日期。 |
| 装运港口 ETA | 目标港口（“到达”港口）的估计到达日期。 |
| 已收到原始单据 | 可选：更新收到原始单据的日期。 |
| 已建议代理 | 可选：更新通知代理的日期。 |
| 已发送原始提单 | 可选：更新发送原始提单的日期。 |
| 已发放货物 | 可选：更新发放货物的日期。 |
| 客户预约 | 可选：更新客户预约日期。 |
| 已在仓库交货 | 可选：更新货物交付到仓库的日期。 |
| 验证日期 | 可选：更新验证日期。 |
| 交货说明 | 可选：更新交货说明的日期。 |
| 发运港口 | 发运物料的港口。 |
| 目标港口 | 物料将送达的港口。 |
| 本地转运商 | 此字段仅供参考。 当地转运商应该可以从供应商表中选择。 |
| 本地运输日期 | 输入预订当地运输的日期。 此字段可以帮助仓库进行计划。 |
| 本地运输时间 | 输入时隙。 此字段可以帮助仓库进行计划。 |
| 行程模板 | 为航行指定的行程模板。 行程模板提供“到达”和“发运”港口的详细信息，以及与装运集装箱的跟踪控制关联的提前期。 |

## <a name="lines-view"></a>行视图

打开 **行** 视图，打开装运集装箱，然后选择装运集装箱标题右上角的 **行** 选项卡。

### <a name="information-on-the-shipping-container-fasttab"></a>装运集装箱快速选项卡上的信息

**行** 视图中的 **装运集装箱** 快速选项卡显示有关帐页的信息。 这些信息中的大多数还会显示在 **标头** 视图中，如本主题前面所述。

### <a name="information-and-buttons-on-the-lines-fasttab"></a>行快速选项卡上的信息和按钮

**行** 视图中的 **行** 快速选项卡显示有关当前装运集装箱中包含的每个完整或部分采购订单行的详细信息。

下表描述了 **行** 视图中的 **行** 快速选项卡上的可用按钮。

| 按钮 | 说明 |
|---|---|
| 移动 | 从航行中删除所选的采购订单行。 |
| 库存 \> 交易记录 | 查看所选采购订单行的库存交易记录。 请注意，如果您使用的是在途货物，还会显示原始订单和在途货物订单。 |
| 库存 \> 显示维度 | 打开一个对话框，您可以在其中选择针对您查看的交易记录显示的库存维度。 |
| 刷新 | 更新与所选采购订单行的行金额、重量或体积相关的信息。 |

### <a name="information-on-the-lines-details-fasttab"></a>行详细信息快速选项卡上的信息

**行** 视图中的 **行详细信息** 快速选项卡显示有关当前在 **行** 快速选项卡上选择的采购订单行的详细信息。
