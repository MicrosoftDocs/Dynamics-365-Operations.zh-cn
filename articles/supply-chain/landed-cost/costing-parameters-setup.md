# <a name="costing-parameter-values-setup"></a>成本计算参数值设置

[!include [banner](../../includes/banner.md)]

当您设置 **登陆成本** 时，您可以定义几组通用值以及每个值的相关设置。 这些值将在您在应用的其他部分选择特定类型的成本计算参数值时可用。 本主题说明如何设置这些值集。

## <a name="set-up-cost-type-codes"></a>设置成本类型代码

成本类型代码确定将商品放入仓库时产生的成本类型，或航行的登陆成本。 尽管它们通常会增加商品的价值，但也可以用于将金额计入分类帐。 当随着时间的推移或在一系列航行期间产生成本并在一个交易中冲销时，将使用分类帐调整。

> [!NOTE]
> 如果成本类型表在法人之间共享，会计科目表也必须在法人之间共享。 否则，过帐交易将无法正确运行。

要设置成本类型代码，请转到 **登陆成本 \> 成本计算设置 \> 成本类型代码**。 您可以使用操作窗格上的按钮来创建新的成本类型代码、编辑现有代码或删除选定的成本类型。

下表说明了可用于每个成本类型代码的字段。

| 字段 | 说明 |
|---|---|
| 成本类型代码 | 为成本类型代码输入名称。 |
| 说明 | 输入成本类型代码的描述。 |
| 使用装运汇率 | 如果必须使用航行汇率（有时称为管理比率）来计算这些成本的值，则将此选项设置为 *是*。 在这种情况下，将使用装运费用代替标准默认或即期汇率来兑换外币发票。 |
| 申报类别 | 此字段为成本类型设定报告类别。 可以按报告类别或成本类型打印报表。 |
| 借方类型 | 选择成本类型是应将物料、会计科目还是供应商记入借方。 |
| 借方过帐 | 如果 **借方类型** 字段设置为 *会计科目*，选择过帐描述。 |
| 借方帐户 | 如果 **借方类型** 字段设置为 *会计科目*，选择要使用的会计科目。 |
| 信贷类型 | 选择此成本类型是应将物料、会计科目还是供应商记入贷方。 |
| 贷方过帐 | 如果 **贷方类型** 字段设置为 *会计科目*，选择过帐描述。 |
| 贷方帐户 | 如果 **贷方类型** 字段设置为 *会计科目*，选择要使用的会计科目。 |
| 结算帐户 | 选择成本类型的结算帐户。 我们建议您为每个成本类型指定一个单独的结算帐户，以帮助进行对帐。 |
| 标准成本过帐类型 | 如果您使用标准成本计算，选择过帐描述。 |
| 标准成本差异帐户 | <p>如果您使用标准成本计算，此处指定的帐户将用于过帐任何差额。 此帐户使用 **物料价格** 页上的登陆成本明细。 此明细通过使用定期例程更新价格来创建。</p><p>例如，某个物料的标准成本是 15.00 美元，FOB 是 13.00 美元，运费是 2.00 美元。 当收到存货的发票时，将以 15.00 美元接收物料，但是物料存在 2.00 美元的标准价格差，因为实际 FOB 是 13.00 美元。 此差额将过帐到在物料过帐模板中设置的标准价格差异科目。 因为估计运费是 2.00 美元，所以过帐存货发票时没有差异。 但是，接收运费发票时，运费为每单位 2.50 美元。 因此，0.50 美元差额将过帐到指定成本。 |
| 移动平均差异帐户 | <p>如果您使用移动平均成本计算，此处指定的帐户将用于过帐任何差额。</p><p>例如，估计运费是 2.00 美元。 但是，接收运费发票时，运费为每单位 2.50 美元。 因此，必须过帐 0.50 美元差额。</p><p>当在 **登陆成本参数** 页上将 **作为差额过帐调整** 选项设置为 *是* 时，所有估计与实际装运成本之间的所有差额将被过帐到此处指定的移动平均差异科目。 当 **作为差额过帐调整** 选项设置为 *否* 时，将使用标准功能，差额将应用于库存或此处指定的移动平均差异科目，具体取决于现有存货。</p> |
| 费用应计帐户 | 选择过帐采购发票时用于计入成本估计的帐户。 仅当 **登陆成本参数** 页 **常规** 选项卡上 **成本计算** 快速选项卡上的 **使用成本类型的费用应计帐户** 选项设置为 *是* 时，才使用此字段。 |
| 费用帐户 | 选择用于捕获供应商已开票的入站运输成本的帐户。 金额将作为借方过帐。 对方帐户是存货变动帐户。 仅当 **应付帐款参数** 页上的 **过帐到分类帐中的费用帐户** 选项设置为 *是* 时，才使用此过帐。 |
| 差异帐户 | 过帐采购发票时将用于冲销应计费用的帐户。 仅当 **登陆成本参数** 页 **常规** 选项卡上 **成本计算** 快速选项卡上的 **使用成本类型的费用应计帐户** 选项设置为 *是* 时，才使用此字段。 |

## <a name="vendor-cost-type-groups"></a>供应商成本类型组

供应商成本类型组帮助确定如何找到 *自动成本* 费用并将其应用于航行。 具有相似进口成本的供应商将链接在一起。 例如，来自新兴市场的所有供应商对从成熟市场购买的相同类型的产品支付相同的关税百分比。

您可以转到 **登陆成本 \> 成本计算设置 \> 供应商成本类型组** 维护供应商成本类型组。 **供应商成本类型组** 页提供一个网格，其中列出所有现有的供应商成本类型组。 您可以使用操作窗格上的按钮添加、删除和编辑网格中的行。

下表描述了网格中每一行上可用的字段。

| 字段 | 说明 |
|---|---|
| 供应商成本类型组 | 为供应商成本类型组输入唯一名称（如 *新兴市场*）。 |
| 说明 | 输入供应商成本类型组的描述。 此描述可以提供有关与供应商组关联的费用级别或类型的详细信息。 |

## <a name="item-cost-type-groups"></a>物料成本类型组

物料成本类型组帮助确定如何找到 *自动成本* 费用并将其应用于航行。 相似的物料链接在一起。 例如，所有关税税率为 5% 的物料可能属于一个特定的成本类型组。

您可以转到 **登陆成本 \> 成本计算设置 \> 物料成本类型组** 维护物料成本类型组。 **物料成本类型组** 页提供一个网格，其中列出所有现有的物料成本类型组。 您可以使用操作窗格上的按钮添加、删除和编辑网格中的行。

下表描述了网格中每一行上可用的字段。

| 字段 | 说明 |
|---|---|
| 物料成本类型组 | 为物料成本类型组输入唯一名称（如 *关税 5%*）。 |
| 说明 | 输入物料成本类型组的描述。 此描述可以提供有关与物料组关联的费用级别或类型的详细信息。 |

> [!NOTE]
> 物料成本类型通过物料的 **已发布产品** 页 **采购** 快速选项卡上的 **成本类型组** 字段链接到物料。

下表描述了网格中每一行上可用的设置。

| 字段 | 说明 |
|---|---|

## <a name="cost-templates"></a>成本模板

您使用成本模板为获取成本估计的用户可能不知道的设置设置默认值。 成本模板可以通过最小化用户为获取准确的估计而必须进行的选择来帮助降低估计流程的复杂性。

要使用成本模板，请转到 **登陆成本 \> 成本计算设置 \> 成本模板**。 在 **成本模板** 页上，左侧的列表窗格将显示所有当前成本模板。 您可以使用操作窗格上的按钮添加、删除和编辑模板。

下表说明了可用于每个模板的设置。

| 字段 | 说明 |
|---|---|
| 成本模板 | 为成本模板输入唯一名称。 此名称通常描述模板的系数或成本倍数。 |
| 说明 | 输入成本模板的描述。 |
| 装运公司 | 选择要与成本模板关联的装运公司的供应商帐户。 |
| 交货模式 | 选择在计算物料的估计成本时成本模板应使用的交货方式。 此字段将帮助确定成本估计中与商品关联的自动成本。 |
| 装运集装箱类型 | 选择要与成本模板关联的装运集装箱类型的类型。 此字段将帮助确定成本估计中与商品关联的自动成本。 |
| 关税代理 | 选择要与成本模板关联的关税代理（供应商）。 此字段将帮助确定与成本估计关联的自动成本。 |
| 系数 | 输入要应用于商品最终成本估计的系数。 例如，要在计算出的成本估计中增加 10%，输入 *1.10*。 |

## <a name="volumetric-divisors"></a>容量除数

容量除数用于计算容重。 每个装运/货运公司都会用公式表示自己的容量除数。 另外，公司的除数通常会随交货方式发生变化。 例如，空运和海运通常有非常不同的除数。 公司还可以根据发运地来让规则更加复杂。

例如，通过空运发送的包装的体积为 3 立方米 (m³)。 公司按容重收费，使用容量除数 6。 此除数乘以体积可确定容重。 因此，此示例的容重为 3 × 6 = 18 千克 (kg)。

要设置容量除数，请转到 **登陆成本 \> 成本计算设置 \> 容量除数**。 **容量除数** 页提供一个列出所有现有容量除数的网格。 您可以使用操作窗格上的按钮添加、删除和编辑网格中的行。

下表描述了网格中每一行上可用的字段。

| 字段 | 说明 |
|---|---|
| 装运公司 | 选择与容量除数关联的装运公司的供应商帐户。 |
| 成本类型代码 | 选择与容量除数关联的成本类型代码。 使用此字段可将成本类型放入报告时段。 可以按报告类别或成本类型打印报表。 |
| 发运港口 | 选择容量除数将应用的“发运”港口。 |
| 容量除数 | 输入将应用于行的容量除数值。 您输入的值将 *乘以* 每个包装的体积，来确定该包装的容重。 |
