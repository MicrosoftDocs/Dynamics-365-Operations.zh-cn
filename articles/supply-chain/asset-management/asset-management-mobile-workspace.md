# <a name="asset-management-mobile-workspace"></a>资产管理移动工作区

[!include [banner](../../includes/banner.md)]


此主题提供有关资产管理移动工作区的信息。 用户可使用此工作区查看和创建维护请求和工作订单。 用户也可以使用日历视图或列表视图查看分配的工作订单作业。 还可以查看和搜索资产和功能位置。


## <a name="overview"></a>概览

资产管理是 Dynamics 365 Supply Chain Management 中的一个高级模块，用于管理资产和工作订单作业。 **资产管理**移动工作区使用户可以在他们选择的移动设备上快速查看分配的工作订单作业。 用户还可以使用其移动设备创建和管理维护请求、更新生命周期状态以及查看资产和功能位置的详细信息。

具体来说，**资产管理**移动工作区让用户能够执行以下任务：

- 创建、查看和编辑维护请求，拍摄照片或将现有图像附加到维护请求，更改维护请求的生命周期状态。 
- 创建、查看和编辑工作订单，拍摄照片或将现有图像附加到工作订单，更改工作订单的生命周期状态，查看工作订单作业。
- 在日历视图中查看分配的工作订单作业。
- 创建、查看和编辑工作订单作业，更新资产计数器，查看维护清单，查看和编辑工单作业注释，查看工作订单作业所需的工具。
- 查看或搜索特定资产或功能位置。


## <a name="prerequisites"></a>先决条件

先决条件根据为您的组织部署的 Dynamics 365 Supply Chain Management 版本而异。

### <a name="prerequisites-if-you-use-microsoft-dynamics-365-supply-chain-management"></a>使用 Microsoft Dynamics 365 Supply Chain Management 的先决条件 
如果已经为您的组织部署 Microsoft Dynamics 365 Supply Chain Management，系统管理员必须发布**资产管理**移动工作区。 有关说明，请查阅 [发布移动工作区](../../dev-itpro/mobile-apps/publish-mobile-workspace.md)。

## <a name="download-and-install-the-mobile-app"></a>下载并安装移动应用
下载并安装 Dynamics 365 for Unified Operations 移动应用：

- [适用于 Android 手机](https://go.microsoft.com/fwlink/?linkid=850662)
- [适用于 iPhones](https://go.microsoft.com/fwlink/?linkid=850663)

## <a name="sign-in-to-the-mobile-app"></a>登录到移动应用
1. 在移动设备上启动应用程序。

2. 输入您的 Dynamics 365 URL。

3. 首次登录时，将提示您输入您的用户名和密码。 输入凭据。

4. 登录后，将显示您的公司的可用工作区。 请注意，如果您的系统管理员以后发布新工作区，您必须刷新移动工作区列表。

![图 1](media/am-mobile-01.png)


## <a name="view-assigned-work-order-jobs-in-calendar-view"></a>在日历视图中查看分配的工作订单作业

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**我的工作订单作业日历**。

3. 选择要查看工作订单作业的日期。 在列表中，您将看到每个工作订单作业的资产 ID 和功能位置 ID。

4. 在列表中选择一个工单作业以查看作业详细信息：资产和功能位置详细信息以及查看**附件**、**清单**、**工具**、**资产计数器**、**注释**、**日记帐**的其他导航链接。

![图 2](media/am-mobile-02.png)


## <a name="create-a-work-order-job"></a>创建工作订单作业

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护工作订单**。

3. 选择要为其创建新工作订单作业的工作订单。

4. 选择**添加行**按钮。

5. 选择要为其创建工作订单作业的**资产**。

6. 选择**维护作业类型**、**维护作业类型变型**和**贸易**。

7. 选择**完成**。

![图 3](media/am-mobile-03.png)


## <a name="add-attachment-to-a-work-order-job"></a>将附件添加到工作订单作业

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护工作订单**。

3. 选择要添加附件的工作订单 > 工作订单作业。
    - 或者，您也可以在主页上选择**我的工作订单作业日历**或**我的工作订单作业列表**，以导航到**工作订单作业详细信息**页面。

4. 在**工作订单作业详细信息**页面上选择**附件**。

5. 您将在工作订单作业上看到现有的附件。 选择**添加附件**。

6. 为附件输入**名称**和**注释**。

7. 选择**选择图像**以从移动库中选择照片，或选择**拍照**拍摄照片。

8. 选择**完成**。

![图 4](media/am-mobile-04.png)


## <a name="view-maintenance-checklist-on-a-work-order-job"></a>查看工作订单作业上的维护清单

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护工作订单**。

3. 选择要查看其清单的工作订单 > 工作订单作业。
    - 或者，您也可以在主页上选择**我的工作订单作业日历**或**我的工作订单作业列表**，以导航到**工作订单作业详细信息**页面。

4. 在**工作订单作业详细信息**页面上选择**清单**。

5. 您将看到与工作订单作业相关的清单行列表。 选择一个清单行以查看**说明**并添加**注释**。

6. 选择返回按钮 (**<**) 以返回到上一页。

![图 5](media/am-mobile-05.png)


## <a name="view-and-update-asset-counters-on-a-work-order-job"></a>查看和更新工作订单作业上的资产计数器

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护工作订单**。

3. 选择要查看其资产计数器的工作订单 > 工作订单作业。
    - 或者，您也可以在主页上选择**我的工作订单作业日历**或**我的工作订单作业列表**，以导航到**工作订单作业详细信息**页面。

4. 在**工作订单作业详细信息**页面上选择**资产计数器**。

5. 您将看到与工作订单作业相关的资产计数器列表。 选择资产计数器行上的铅笔图标以更新计数器值。

6. 输入一个新的计数器值，然后选择**完成**。

![图 6](media/am-mobile-06.png)


## <a name="register-consumption-on-a-work-order-job"></a>登记工作订单作业的消耗量

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护工作订单**。

3. 选择要为其添加消耗量登记的工作订单 > 工作订单作业。
    - 或者，您也可以在主页上选择**我的工作订单作业日历**或**我的工作订单作业列表**，以导航到**工作订单作业详细信息**页面。

4. 在**工作订单作业详细信息**页面上选择**日记帐**。

5. 选择**添加工时**以创建工时登记。
    1. 从查找中选择**类别**。
    2. 在**工时**字段中，输入在工作订单作业上花费的工时数。
    3. 选择合适的**行属性**。
    4. 选择**完成**。

6. 选择**添加物料**创建物料登记。
    1. 从查找中选择**物料编号**。
    2. 从查找中选择**站点**。
    3. 输入所消耗物料的**数量**。
    4. 选择**完成**。

7. 选择**添加费用**以创建费用登记。
    1. 从查找中选择**类别**。
    2. 输入费用登记的数量。
    3. 从查找中选择**销售币种**。
    4. 输入费用登记的**成本价**。
    5. 选择**完成**。

![图 7](media/am-mobile-07.png)


## <a name="update-lifecycle-state-on-a-work-order"></a>更新工作订单上的生命周期状态

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护工作订单**。

3. 选择要更新其生命周期状态的工作订单。

4. 选择屏幕底部的**更新状态**按钮。

5. 从列表中选择新的生命周期状态。

6. 选择**完成**。

![图 8](media/am-mobile-08.png)


## <a name="create-a-maintenance-request"></a>创建维护请求

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护请求**。

3. 选择屏幕底部的**操作**，然后选择**创建维护请求**。

4. 如果在**资产管理**中为维护请求启用了编号规则，则**维护请求**字段将被隐藏，因为它会自动填充。如果**维护请求**字段可见，请输入维护请求 ID。

5. 选择**维护请求类型**。

6. 为维护请求输入**描述**。

7. 选择要为其创建请求的**资产**。

8. 为维护请求选择**服务级别**。

9. 选择**完成**。

![图 9](media/am-mobile-09.png)


## <a name="add-attachment-to-a-maintenance-request"></a>将附件添加到维护请求

1. 在移动设备上，打开**资产管理**工作区。

2. 选择**所有维护请求**。

3. 选择要添加附件的维护请求。

4. 选择屏幕底部的**附件**。

5. 选择**添加附件**。

6. 为附件输入**名称**和**注释**。

7. 选择**选择图像**以从移动库中选择照片，或选择**拍照**拍摄照片。

8. 选择**完成**。

![图 10](media/am-mobile-10.png)
