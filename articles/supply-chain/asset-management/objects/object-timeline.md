# <a name="asset-event-history"></a>资产事件历史记录

[!include [banner](../../includes/banner.md)]

 

本主题介绍如何在资产管理中访问资产事件历史记录。 **资产事件历史记录** 页显示资产生命周期内创建的登记历史记录。 可从 **所有资产**、**有效资产** 和 **我的有效资产** 菜单项访问此页。 选择资产，然后选择 **事件历史记录**。

在 **资产事件历史记录** 页的 **详细信息** 快速选项卡上，可以对所有可用信息执行搜索。 例如，可以使用资产事件历史记录查找以下信息：

- 上次对资产使用作业类型的时间
- 特定工人在工作订单作业中输入标记的时间

只要打开该页，都将更新时间线。 它包含以下信息:

- 已对资产所做更改：资产位置、资产 ID、名称和供应商保修
- 资产父级
- 资产物料清单 (BOM)
- 资产生命周期状态日志
- 功能位置
- 维护请求
- 工作订单，包括已过帐物料和注释
- 故障
- 条件评估
