<span data-ttu-id="81fb9-104">工作订单用于管理维护作业，提供维护作业所需信息和登记维护作业的消耗。</span><span class="sxs-lookup"><span data-stu-id="81fb9-104">Work orders are used to manage maintenance jobs, provide required information for them, and register consumption on them.</span></span> <span data-ttu-id="81fb9-105">每个工作订单中可以包含一个或多个工作订单作业，并且可以为每个工作订单关联一个或多个资产。</span><span class="sxs-lookup"><span data-stu-id="81fb9-105">Each work order can contain one or more work order jobs, and one or more assets can be connected to each work order.</span></span> <span data-ttu-id="81fb9-106">每个工作订单作业定义为资产安排的一个维护作业。</span><span class="sxs-lookup"><span data-stu-id="81fb9-106">Each work order job defines a maintenance job that is scheduled on the asset.</span></span>

<span data-ttu-id="81fb9-107">可以通过以下方式创建工作订单：</span><span class="sxs-lookup"><span data-stu-id="81fb9-107">Work orders can be created in the following ways:</span></span>

- <span data-ttu-id="81fb9-110">对于预防性维护作业或维护请求，通过[维护安排](../preventive-and-reactive-maintenance/maintenance-schedule.md)。</span><span class="sxs-lookup"><span data-stu-id="81fb9-110">For preventive maintenance jobs or maintenance requests, from [Maintenance schedule](../preventive-and-reactive-maintenance/maintenance-schedule.md).</span></span>
