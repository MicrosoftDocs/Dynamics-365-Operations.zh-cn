# <a name="introduction-to-work-orders"></a><span data-ttu-id="20618-103">工作订单简介</span><span class="sxs-lookup"><span data-stu-id="20618-103">Introduction to work orders</span></span>

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

<span data-ttu-id="20618-104">工作订单用于管理维护作业，提供维护作业所需信息和登记维护作业的消耗。</span><span class="sxs-lookup"><span data-stu-id="20618-104">Work orders are used to manage, provide required information for, and register consumption on maintenance jobs.</span></span> <span data-ttu-id="20618-105">一个工作订单中可以包含一个或多个工作订单作业，并且可以为一个工作订单关联一个或多个资产。</span><span class="sxs-lookup"><span data-stu-id="20618-105">A work order may contain one or more work order jobs, and one or more assets can be connected to a work order.</span></span> <span data-ttu-id="20618-106">每个工作订单行定义为资产安排的一个维护作业。</span><span class="sxs-lookup"><span data-stu-id="20618-106">Each work order line defines a maintenance job scheduled on the asset.</span></span>

<span data-ttu-id="20618-107">可以自动创建工作订单，也可以手动创建：</span><span class="sxs-lookup"><span data-stu-id="20618-107">Work orders can be created automatically or manually:</span></span>

- <span data-ttu-id="20618-108">对于基于日历且设置为“自动创建”的维护计划，将自动使用**安排维护计划**窗体。</span><span class="sxs-lookup"><span data-stu-id="20618-108">Automatically using the **Schedule maintenance plans** form, for calendar-based maintenance plans set to "Auto create".</span></span>

- <span data-ttu-id="20618-110">基于维护安排（可以是预防性作业或维护请求）创建。</span><span class="sxs-lookup"><span data-stu-id="20618-110">Create from maintenance schedule, which can be preventive maintenance jobs or maintenance requests.</span></span>  

- <span data-ttu-id="20618-111">手动创建工作订单。</span><span class="sxs-lookup"><span data-stu-id="20618-111">Create a work order manually.</span></span>  

- <span data-ttu-id="20618-112">在**所有维护请求**、**有效维护请求**或**我的功能位置维护请求**中创建工作订单。</span><span class="sxs-lookup"><span data-stu-id="20618-112">Create a work order from the **All maintenance requests** or **Active maintenance requests** or **My functional location maintenance requests**.</span></span>
