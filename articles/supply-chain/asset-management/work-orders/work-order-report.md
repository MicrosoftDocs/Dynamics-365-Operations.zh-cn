<span data-ttu-id="10392-104">可生成工作订单报告，以便显示有关您选择的一个或多个工作订单的详细信息。</span><span class="sxs-lookup"><span data-stu-id="10392-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="10392-105">选择 **资产管理** > **报告** > **工作订单** > **工作订单报告**。</span><span class="sxs-lookup"><span data-stu-id="10392-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="10392-106">根据需要选择切换按钮上的“是”/“否”，以便确定报告中要包含的详细信息。</span><span class="sxs-lookup"><span data-stu-id="10392-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="10392-107">在 **打印设置** 部分中，可选择是否要包含来自关联维护作业类型的设置的附件。</span><span class="sxs-lookup"><span data-stu-id="10392-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

5. <span data-ttu-id="10392-110">选择 **确定** 以生成报告。</span><span class="sxs-lookup"><span data-stu-id="10392-110">Select **OK** to generate the report.</span></span>
