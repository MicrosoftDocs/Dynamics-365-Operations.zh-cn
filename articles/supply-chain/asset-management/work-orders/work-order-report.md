# <a name="work-order-report"></a>工作订单报告


[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]


可生成工作订单报告，以便显示有关工作订单的详细信息。 可以选择在此报告中显示一个或多个工作订单。

1. 单击**资产管理** > **报告** > **工作订单报告** > **工作订单报告**。

2. 选择切换按钮审核功能的“是”/“否”，然后根据需要填写字段，以便确定报告中要包含的详细信息。  
  a. 在**打印设置**部分中，可选择是否要在打印中包含来自关联的维护作业类型设置的附件。

3. 在**要包括的记录**快速选项卡上，可以按**工作订单**筛选报告的内容。

4. 如果需要，可以通过填写**在后台运行**快速选项卡上的字段，将工作订单报告的生成设置为批处理作业。

5. 单击**确定**以生成报表。
