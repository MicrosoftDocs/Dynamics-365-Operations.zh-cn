# <a name="work-order-report"></a>工作订单报告

[!include [banner](../../includes/banner.md)]


可生成工作订单报告，以便显示有关您选择的一个或多个工作订单的详细信息。

1. 选择 **资产管理** > **报告** > **工作订单** > **工作订单报告**。

2. 根据需要选择切换按钮上的“是”/“否”，以便确定报告中要包含的详细信息。  
    - 在 **打印设置** 部分中，可选择是否要包含来自关联维护作业类型的设置的附件。

3. 在 **要包括的记录** 快速选项卡上，可以按 **工作订单** 筛选报告的内容。

4. 在 **在后台运行** 快速选项卡上，可根据需要将工作订单报告生成设置为批处理作业。

5. 选择 **确定** 以生成报告。
