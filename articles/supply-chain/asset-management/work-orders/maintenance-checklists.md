# <a name="maintenance-checklists"></a>维护清单


[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

维护清单是对维护作业类型设置的，在处理工作订单时使用。 填写维护清单是完成工作订单时执行的部分工作。 有关如何在**维护作业类型默认**窗体中对维护作业类型设置维护清单的详细信息，请参阅[维护作业类型类别和维护工作类型、维护作业类型变型、维护作业工种和维护清单](../setup-for-work-orders/job-groups-and-job-types-variants-trades-and-checklists.md)部分。

处理工作订单的维护清单时，可以填写与维护作业类型关联的预定义维护清单。 还可以添加更多维护清单。

## <a name="fill-out-a-maintenance-checklist"></a>填写维护清单

1. 单击**资产管理** > **常用** > **工作订单** > **所有工作订单**或**有效工作订单**。

2. 选择工作订单，然后单击**维护清单**。

3. 在**工作订单维护作业清单**中，可以查看所有工作订单作业的维护清单。 如果工作订单作业有不同的维护作业类型，则每个工作订单作业的维护清单可能不同。 选择工作订单作业以处理关联的维护清单。 将在**行详细信息**快速选项卡上显示所选维护清单的详细信息。

4. 按照显示顺序填写所有维护清单行，一次一行。 类型为“标题”的维护清单行用作标题来为其下的维护清单行分组。 无需填写标题，但是对于所有维护清单行类型，可以向标题添加**注释**。

5. 如果为维护清单行关联了说明，则已选中**说明**复选框。 可在**行详细信息**快速选项卡 > **说明**部分中查看所选维护清单行的说明。

6. 填写维护清单行所需信息取决于关联的维护清单类型。 可通过填写**行详细信息**快速选项卡上的字段来填写维护清单行。 例如，在类型为“文本”的行中添加**注释**，用于说明该清单行的结果。 在类型为“度量”的行中添加读取的设备**计数器值**，如果需要，还可以添加**注释**。

- 填写维护清单行之后，选中该行的**已选中**复选框将其标记为已填写。 如果因为某个维护清单行与工作订单作业无关而希望将其放弃，请选中该行上的**不适用**复选框。 如果维护清单行标记为**必需**，则必须将其标记为“已选中”或“不适用”。  
- 只有在工作订单的生命周期状态为[有效](../setup-for-work-orders/work-order-lifecycle-states.md)时，才能更新维护清单登记。  


## <a name="add-a-maintenance-checklist-line"></a>添加维护清单行

维护清单通过定义维护作业类型默认创建，并传输到工作订单作业。 如果需要，可以向工作订单作业添加维护清单行。 手动添加的维护清单行为“手动”引用。

1. 在**工作订单维护作业清单**中，选择要为其添加维护清单的工作订单作业。

2. 如果要在所选维护清单行后面插入新行，请在**维护清单行**快速选项卡上选择一个维护清单行，然后按键盘上的向下箭头按钮。 将在**行号**字段中自动插入序列中的下一个编号。 如果要在所选维护清单行上方插入新行，也可以选择一个维护清单行，然后单击**添加行**按钮。

3. 在**名称**字段中插入该维护清单行的名称。

4. 在**类型**字段中，为维护清单行选择类型。 对于每个维护清单类型，**行详细信息**快速选项卡上将显示相关字段。  
  a. “文本”用于添加包含有关要执行的操作文本说明的维护清单行。 如果要让工人进行检查或检验，但是不需要获得具体（可度量的）结果，可使用此维护清单类型。 请在**行详细信息**快速选项卡上的**说明**部分中插入要执行的操作的说明。 b. “标题”用作标题来为标题下显示的维护清单行分组。 如果有多个可拆分为特定区域的维护清单行，这非常有用。 请在**名称**字段中插入描述性名称。  
  c. 在工作订单作业中手动添加维护清单行时，“模板”不适用。  
  d. “变量”用于定义维护清单行的一定范围可能结果。 [维护作业类型类别和维护工作类型、维护作业类型变型、维护作业贸易和维护清单](../setup-for-work-orders/job-groups-and-job-types-variants-trades-and-checklists.md)中介绍了维护清单变量的设置。 在**名称**字段中插入用于描述变量的名称。 在**变量**字段中选择变量。 请在**行详细信息**快速选项卡上的**说明**部分中插入要执行的操作的说明。  
  e. “度量”用于记录特定度量。 在**名称**字段中插入度量的名称。 在**行详细信息**快速选项卡上，选择**计数器**和**单位**。 在**说明**部分中插入要执行的操作的说明。  

5. 手动添加维护清单行之后，按照上面部分中的说明填写行。

>[!NOTE]
>不能删除**工作订单维护作业清单**中引用为“作业类型”的维护清单行。 只能删除引用为“手动”的维护清单行，此类行是您或其他维护工人手动创建的。
