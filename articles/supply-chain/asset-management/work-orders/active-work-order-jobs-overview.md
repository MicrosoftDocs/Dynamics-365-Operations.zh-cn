# <a name="active-work-order-maintenance-jobs-overview"></a>有效的工作订单维护作业概览

[!include [banner](../../includes/banner.md)]



在 **有效的工作订单维护作业** 列表页中，可根据已经为特定资产、资产类型、制造商、型号、维护作业类型等创建的工作订单的数量获取工作订单的概览。 通过选择工作订单作业，然后选择 **编辑**，可以打开关联的工作订单。 通过选择工作订单作业，然后选择“操作窗格”的 **工作订单维护作业** 选项卡上的按钮之一，您可以查看与工作订单作业相关的工作订单的数据。

要打开列表页，请选择 **资产管理** > **常用** > **工作订单** > **有效的工作订单维护作业**。 此页列出了所有有效工作订单作业，另外还显示与工作订单或工作订单作业有关的部分信息。

在 **%** 列中，一个数字以百分比形式显示工作订单的完成进度。 完成率基于两项计算：已过帐工时与预测工时比较，以及已经完成的维护清单数。

下图显示 **有效的工作订单维护作业** 列表页的示例。
