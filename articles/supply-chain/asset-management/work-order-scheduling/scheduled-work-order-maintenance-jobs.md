# <a name="scheduled-work-order-maintenance-jobs"></a>计划的工作订单维护作业

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

**计划的工作订单维护作业**页用于获取为资源分配的工作订单的概览。 列表中将显示使用的资源类型为“人力资源”、“工具”和“机器”的工作订单。 可使用此列表获取分配给特定资源的工作订单的概览。 也可以将其用于快速查找分配给今天上午打电话请病假之类维护工人的工作订单，然后快速为作业分配另一位维护工人。

## <a name="view-scheduled-work-order-maintenance-jobs"></a>查看计划的工作订单维护作业

1. 单击**资产管理** > **常用** > **工作订单** > **计划的工作订单维护作业**。 您将看到其工作订单生命周期状态设置为“已计划”或“进行中”的所有工作订单的列表。

2. 可为该列表排序，例如，按维护工人排序。 也可以使用筛选器将该列表限制为显示为特定资源或维护工人分配的工作订单。

3. 可查看有关工作订单的注释，如果需要，还可以添加新注释，方法是在列表中选择工作订单作业，然后单击**注释**。

4. 如果要为工作订单分配一位维护工人，请在列表中选择工作订单，然后单击**工作订单**。

5. 将打开**工作订单**页面。 单击**分派**将工作订单安排给特定维护工人。

>[!NOTE]
>有关为一个或多个工作订单进行安排的详细信息，请参阅[计划工作订单](../work-order-scheduling/schedule-work-orders.md)和[分派工作订单](../work-order-scheduling/dispatch-work-order.md)。
