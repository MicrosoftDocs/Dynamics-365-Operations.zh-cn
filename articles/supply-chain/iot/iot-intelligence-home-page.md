# <a name="iot-intelligence-now-replaced-by-sensor-data-intelligence"></a>IoT 智能现已替换为 Sensor Data Intelligence

[!include [banner](../../includes/banner.md)]

IoT 智能功能已更新至 2.0 版本，同时重命名为 *Sensor Data Intelligence*。 新版本目前作为 Supply Chain Management 10.0.30 或更高版本的预览功能提供。

Sensor Data Intelligence 引入了一种新的部署模型，其中所需的 Azure 组件部署在您自己的 Azure 订阅上，而不是由 Microsoft 管理。 这种新模型增强了您自定义功能并将其与第三方系统集成的能力。 新版本还引入了对资产管理的两个新物联网 (IoT) 方案的支持。

由于即将发生这些变更，Microsoft 建议暂时不要开始任何基于现有 IoT 智能功能的新项目。 如果您有兴趣将 IoT 功能添加到您的系统，我们建议您使用 Sensor Data Intelligence 预览版开始添加。 有关 Sensor Data Intelligence 及其工作原理的更多信息，请参阅 [Sensor Data Intelligence 主页](../sensor-data-intelligence/sdi-home-page.md)。
