# <a name="partial-shipment-of-a-transport-load"></a><span data-ttu-id="32e51-103">运输负荷部分装运</span><span class="sxs-lookup"><span data-stu-id="32e51-103">Partial shipment of a transport load</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="32e51-104">通过为装载设置部分装运，可处理其产能在所有行都添加到装载之前无法确定的装载。</span><span class="sxs-lookup"><span data-stu-id="32e51-104">By setting up partial shipment of loads, you can handle loads where the capacity can't be determined until all the sales lines have been added to a load.</span></span> <span data-ttu-id="32e51-105">了解到精确的货盘计数时，即可完成此过程。</span><span class="sxs-lookup"><span data-stu-id="32e51-105">The process can then be finalized when the exact pallet count is known.</span></span> <span data-ttu-id="32e51-106">因此，在货物实物从分阶段库存实际运出之前，无需确定为哪些运输分配哪些货盘。</span><span class="sxs-lookup"><span data-stu-id="32e51-106">Therefore, you don't have to decide which pallets will be assigned to which transport until the moment when a transport is being physically loaded out of the staged inventory.</span></span>

<span data-ttu-id="32e51-107">此功能提供了备用方法来代替更严格结构的实施，在后者中，必须先确定为哪些运输分配哪些货盘，才能创建领料工作或装载工作。</span><span class="sxs-lookup"><span data-stu-id="32e51-107">This feature offers an alternative to the enforcement of a more rigid structure, where you must determine which pallets will be assigned to which transport before picking work or loading work can be created.</span></span> <span data-ttu-id="32e51-108">相反，用户可为单个装载配置部分装运确认。</span><span class="sxs-lookup"><span data-stu-id="32e51-108">Instead, users can configure individual loads for a partial shipment confirmation.</span></span> <span data-ttu-id="32e51-109">然后，可能会对这些装载进行运输负荷处理。</span><span class="sxs-lookup"><span data-stu-id="32e51-109">The transport loading processes for those loads can then occur.</span></span> <span data-ttu-id="32e51-110">因此，运输规划部门在规划装载时不必考虑单次运输的产能。</span><span class="sxs-lookup"><span data-stu-id="32e51-110">Therefore, the transportation planning department can plan loads without having to consider the capacity of individual transports.</span></span>

<span data-ttu-id="32e51-111">装载时，工作人员可定义可将货盘装载到的新运输负荷。</span><span class="sxs-lookup"><span data-stu-id="32e51-111">At the time of loading, workers can define a new transport load that a pallet can be loaded to.</span></span> <span data-ttu-id="32e51-112">也可以识别现有运输负荷。</span><span class="sxs-lookup"><span data-stu-id="32e51-112">Alternatively, they can identify an existing transport load.</span></span> <span data-ttu-id="32e51-113">可通过移动设备记录此数据。</span><span class="sxs-lookup"><span data-stu-id="32e51-113">This data can be recorded via a mobile device.</span></span> <span data-ttu-id="32e51-114">因此，几位仓库工作人员可将库存从同一个负荷或不同负荷装载到同一辆卡车。</span><span class="sxs-lookup"><span data-stu-id="32e51-114">Therefore, several warehouse workers can load inventory from the same loads or different loads onto the same truck.</span></span> <span data-ttu-id="32e51-115">然后可全部或部分装运这些负荷。</span><span class="sxs-lookup"><span data-stu-id="32e51-115">The loads can then be fully or partially shipped.</span></span>

> [!NOTE] 
> <span data-ttu-id="32e51-116">要将库存从负荷装载到特定运输负荷，并部分装运负荷，必须通过使用工作模板中的装载类生成工作。</span><span class="sxs-lookup"><span data-stu-id="32e51-116">In order to load inventory from a load to a specific transport load and partially ship the load, work must be generated by using a loading class in a work template.</span></span> <span data-ttu-id="32e51-117">不能将 **领料** 类型的普通领料工作装载到运输负荷，如卡车。</span><span class="sxs-lookup"><span data-stu-id="32e51-117">Ordinary picking work of the **Picking** type can't be loaded to a transport load such as a truck.</span></span>
