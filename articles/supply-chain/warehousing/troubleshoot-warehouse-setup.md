# <a name="troubleshoot-warehouse-setup"></a>仓库设置疑难解答

[!include [banner](../includes/banner.md)]

此主题介绍如何解决在 Microsoft Dynamics 365 Supply Chain Management 中设置仓库时可能遇到的常见问题。

## <a name="i-cant-use-any-role-except-administrator-to-access-the-mobile-device-app-emulator"></a>除了管理员，我不能使用任何角色来访问移动设备应用模拟器。

### <a name="issue-description"></a>问题描述

您不能使用管理员角色以外的任何角色来访问移动设备应用模拟器。

### <a name="issue-resolution"></a>解决问题

移动设备应用模拟器设置为只能在管理员帐户中使用。 出于所有测试和实时流程目的，我们建议您使用仓库管理移动应用本身。
