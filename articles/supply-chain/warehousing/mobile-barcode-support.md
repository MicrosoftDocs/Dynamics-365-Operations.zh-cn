<span data-ttu-id="97c6e-104">由于 Android 是一个开源项目，仓库条码扫描仪的任何硬件制造商都可以构建设备来运行 Android 操作系统。</span><span class="sxs-lookup"><span data-stu-id="97c6e-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="97c6e-105">如果设备能够运行为 Android 执行环境编写的应用，其只能是 Android 兼容设备。</span><span class="sxs-lookup"><span data-stu-id="97c6e-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
