<span data-ttu-id="89349-104">由于 Android 是一个开放源代码项目，因此仓库条码扫描仪的任何硬件制造商都可以生成设备来运行 Android 操作系统。</span><span class="sxs-lookup"><span data-stu-id="89349-104">Because Android is an open-source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="89349-105">如果设备能够运行为 Android 执行环境编写的应用，其只能是 Android 兼容设备。</span><span class="sxs-lookup"><span data-stu-id="89349-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
