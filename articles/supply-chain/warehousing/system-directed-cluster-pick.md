[!include [banner](../includes/banner.md)]

<span data-ttu-id="aa604-104">群集领料是一个单件领料流程，通过将订单归并到领料群集中，您可以同时为多个订单领取物料。</span><span class="sxs-lookup"><span data-stu-id="aa604-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="aa604-105">然后，您只需访问一次领料库位。</span><span class="sxs-lookup"><span data-stu-id="aa604-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="aa604-106">通常，此功能用于小订单领料或数量少于案例数量的情况。</span><span class="sxs-lookup"><span data-stu-id="aa604-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>

<span data-ttu-id="aa604-107">设置系统导向的群集领料后，您可以基于系统生成的群集来群集选择工作标题。</span><span class="sxs-lookup"><span data-stu-id="aa604-107">When system-directed cluster picking is set up, you can cluster-pick work headers, based on a system-generated cluster.</span></span> <span data-ttu-id="aa604-108">系统最多可以群集选择订单的数量是在群集配置文件中指定的位置数。</span><span class="sxs-lookup"><span data-stu-id="aa604-108">The system cluster-picks orders up to the number of positions that is specified in the cluster profile.</span></span> <span data-ttu-id="aa604-109">因此，您可以同时选择多个订单，而不必手动创建群集。</span><span class="sxs-lookup"><span data-stu-id="aa604-109">Therefore, you can pick multiple orders at the same time without having to manually create a cluster.</span></span>
