## <a name="purchase-order-statuses"></a><span data-ttu-id="984b3-145">采购订单状态</span><span class="sxs-lookup"><span data-stu-id="984b3-145">Purchase order statuses</span></span>
<span data-ttu-id="984b3-146">采购订单包含几种状态字段，指示订单的进度。</span><span class="sxs-lookup"><span data-stu-id="984b3-146">POs include several status fields that indicate the progress of the order.</span></span> <span data-ttu-id="984b3-147">所有这些字段都显示在订单的**订单头**中，其中几个字段也在所有订单的网格概览中显示。</span><span class="sxs-lookup"><span data-stu-id="984b3-147">All these fields are visible in the **Header** view of the order, and a few of them are also visible in the grid overview of all orders.</span></span> <span data-ttu-id="984b3-148">**状态**字段显示订单上的数量的状态。</span><span class="sxs-lookup"><span data-stu-id="984b3-148">The **Status** field show the status for quantities on the order.</span></span> <span data-ttu-id="984b3-149">提供以下值：</span><span class="sxs-lookup"><span data-stu-id="984b3-149">The following values are available:</span></span>
