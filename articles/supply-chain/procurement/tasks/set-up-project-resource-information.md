# <a name="set-up-project-resource-information"></a>设置项目资源信息

[!include [banner](../../includes/banner.md)]

此过程显示如何设置项目资源。 此任务使用 USMF 数据集。

1. 转至“项目管理与核算”>“项目资源”>“资源列表”。
2. 单击“新资源”。
3. 在“工作人员”字段中，输入或选择一个值。
4. 在“日历”字段中，输入或选择一个值。
5. 在“期间代码”字段中，输入或选择一个值。
6. 单击“创建”。
7. 单击“成本价”。
8. 单击“成本价 - 工时”。
9. 单击“新建”。
10. 在“有效日期”字段中，输入日期。
11. 在“成本价”字段中，输入一个数字。
12. 单击“保存”。
13. 关闭该页面。
14. 单击“销售价”。
15. 单击“销售价 - 工时”。
16. 单击“新建”。
17. 在“有效日期”字段中，输入日期。
18. 在“定价”字段中输入数字。
19. 在“有效”字段中，选择一个选项。
20. 单击“保存”。
21. 关闭该页面。
