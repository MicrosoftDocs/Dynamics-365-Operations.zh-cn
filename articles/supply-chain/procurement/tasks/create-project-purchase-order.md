# <a name="create-project-purchase-order"></a>创建项目采购订单

[!include [banner](../../includes/banner.md)]

此过程说明如何创建项目采购订单。 此任务使用 USSI 数据集。

1. 转到项目管理和记账>项目>全部项目。
2. 在列表中，单击所选行中的链接。
3. 在“操作窗格”上单击“管理”。
4. 单击“单项任务”。
5. 单击“采购订单”。
6. 在“供应商帐户”字段中，输入或选择一个值。
7. 在“站点”字段中，输入或选择一个值。
    * 这些步骤并非必须执行的步骤，但是可以通过为采购订单行设置默认站点和仓库来简化采购订单。  
8. 在“仓库”字段中，输入或选择一个值。
9. 单击“确定”。
10. 在列表中，标记所选的行。
11. 在“物料编号”字段中，输入或选择一个值。
    * 这可以是物料编号或采购类别。  
12. 展开“行明细”部分。
13. 单击“项目”选项卡。
    * 验证销售和成本价可用。 如果不可用但需要，请输入信息。  
14. 单击“保存”。
