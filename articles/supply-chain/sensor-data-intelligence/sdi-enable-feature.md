# <a name="turn-on-sensor-data-intelligence-for-your-system"></a>为您的系统打开 Sensor Data Intelligence

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]

必须在系统中开启 Sensor Data Intelligence，然后才能使用此功能。

1. 转到 **系统管理 \> 工作区 \> 功能管理**。 （有关更多信息，请参见[功能管理概述](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md)。）
1. 在 **全部** 选项卡上，使用 **筛选** 字段搜索名为 *Sensor Data Intelligence* 的功能。
1. 如果您的系统上启用了 *Sensor Data Intelligence* 功能，在列表中选择它，然后选择 **禁用** 将其禁用。 您不能将此功能的旧版本与您将在下一步启用的新预览版本一起使用。
1. 使用 **筛选** 字段搜索名为 *（预览）Sensor Data Intelligence* 的功能。
1. 在列表中选择此功能，然后选择 **立即启用** 启用它。
