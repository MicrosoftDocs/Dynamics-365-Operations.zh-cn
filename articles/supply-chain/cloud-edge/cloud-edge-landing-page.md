# <a name="scale-unit-capability-for-supply-chain-management-is-currently-paused-for-new-customers"></a>当前已为新客户暂停 Supply Chain Management 的缩放单元功能

[!include [banner](../includes/banner.md)]

> [!IMPORTANT]
> 已为新客户暂停了 Microsoft Dynamics 365 Supply Chain Management 的缩放单元功能，因为正在增强体系结构。 我们目前没有针对该解决方案招收任何新客户，但正在积极为我们的供应链业务线开发一个可组合、灵活且易于使用的体系结构，这将进一步增强我们为客户提供世界一流缩放单元的能力。
>
> 我们正在努力增强 Dynamics 365 Supply Chain Management，以便在仓库管理等特定模块中进一步提高可组合性。 我们将继续计划和开发新的云和边缘缩放单元产品/服务，以满足客户的需求。 有关详细信息，请联系 [d365scmcneext@microsoft.com](mailto:d365scmcneext@microsoft.com)。
>
> Microsoft Dynamics 365 Commerce 的 Commerce Scale Unit (CSU) Core 平台与 Dynamics 365 Supply Chain Management 的缩放单元功能没有任何关系。 CSU Core 不受此发布的影响，仍完全支持 Dynamics 365 Commerce 应用程序。 有关该功能的更多信息，请参阅 [Commerce Scale Unit (CSU) Core 简介](../../commerce/dev-itpro/CSU-core.md)。
