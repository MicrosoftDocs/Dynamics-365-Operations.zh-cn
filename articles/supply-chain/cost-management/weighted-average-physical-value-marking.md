# <a name="weighted-average-with-physical-value-and-marking"></a><span data-ttu-id="3e4a2-103">具有实际成本和标记的加权平均</span><span class="sxs-lookup"><span data-stu-id="3e4a2-103">Weighted average with physical value and marking</span></span>

[!include[banner](../includes/banner.md)]

[!include[retail name](../includes/retail-name.md)]

<span data-ttu-id="3e4a2-104">加权平均是基于加权平均原则的一种库存模型，其中，按照在库存结转期间入库物料的平均值加上前一期间的所有现有库存量，计算库存发货量。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-104">Weighted average is an inventory model based on the weighted average principle, where issues from inventory are valued at the average value of the items that are received into inventory during the inventory closing period, plus any on-hand inventory from the previous period.</span></span>

<span data-ttu-id="3e4a2-105">在您执行库存结转时，将根据虚拟发货结算所有收货，这将保留收货总数量和值。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="3e4a2-106">此虚拟发货具有将从其结算发货的相应虚拟收货。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="3e4a2-107">通过这种方法，所有发货都具有相同的平均成本。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="3e4a2-108">虚拟发货和收货可以视为虚拟转移，称为“加权平均库存结转转移”。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>

<span data-ttu-id="3e4a2-109">如果只存在一个收货，则可以根据该收货结算所有发货，并且将不创建虚拟转移。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-109">If there is only one receipt, all issues can be settled from it and the virtual transfer will not be created.</span></span> 

<span data-ttu-id="3e4a2-110">在使用加权平均时，您可以标记库存交易记录，以便根据特定发货（而非使用加权平均规则）结算特定库存收货。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-110">When using weighted average, you can mark inventory transactions so that a specific item receipt is settled against a specific issue, instead of using the weighted average rule.</span></span> 

<span data-ttu-id="3e4a2-111">我们建议在您使用加权平均库存模型时使用每月库存结转。</span><span class="sxs-lookup"><span data-stu-id="3e4a2-111">We recommend a monthly inventory closing when you use the weighted average inventory model.</span></span> 

<span data-ttu-id="3e4a2-112">按以下公式计算加权平均库存结转法：</span><span class="sxs-lookup"><span data-stu-id="3e4a2-112">The weighted average inventory costing method is calculated by the following formula:</span></span>
