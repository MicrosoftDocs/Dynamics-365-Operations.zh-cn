# <a name="view-cost-object-balance"></a>查看成本对象余额

[!include [banner](../../includes/banner.md)]

创建此程序的演示数据公司是 USMF。 此程序是专为成本控制设计的。

1. 选择 **成本管理**。
2. 选择 **已发布产品**。
3. 使用值 *m0004* 通过快速筛选器在 **物料编号** 字段中进行筛选。
4. 在“操作”窗格上，选择 **管理成本**。
5. 选择 **成本对象**。
6. 选择 **实际数量**。
7. 选择 **维度显示方式**。
8. 清除 **位置** 复选框。
9. 选择 **确定**。
