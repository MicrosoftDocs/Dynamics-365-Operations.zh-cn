# <a name="view-cost-object-balance"></a>查看成本对象余额

[!include [banner](../../includes/banner.md)]

创建此程序的演示数据公司是 USMF。 此程序是专为成本控制设计的。

1. 单击“成本管理”。
2. 单击“已发布产品”。
3. 使用值“m0004”通过快速筛选器在“物料编号”字段中进行筛选。
4. 在“操作窗格”中，单击“管理成本”。
5. 单击“成本对象”。
6. 单击“实际数量”。
7. 单击“显示维度”。
8. 清除“位置”复选框。
9. 单击“确定”。
