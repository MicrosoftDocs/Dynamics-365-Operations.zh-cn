# <a name="view-inventory-statement-and-kpi-by-cost-object"></a>按成本对象查看库存报表和 KPI

[!include [banner](../../includes/banner.md)]

创建此程序的演示数据公司是 USMF。 此程序是专为成本控制设计的。

1. 单击“成本管理”。
2. 单击“已发布产品”。
3. 使用“快速筛选器”以查找记录。 例如，使用值“d0005”在“物料编号”字段中进行筛选。
4. 在“操作窗格”中，单击“管理成本”。
5. 单击“成本对象”。
6. 单击“库存报表和 KPI”。
7. 在“开始日期”字段中输入日期。
8. 在“结束日期”字段中输入日期。
