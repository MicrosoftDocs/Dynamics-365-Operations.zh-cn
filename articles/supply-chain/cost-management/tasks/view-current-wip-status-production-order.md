# <a name="view-current-wip-status-on-a-production-order"></a>查看生产订单当前的 WIP 状态

[!include [banner](../../includes/banner.md)]

该过程显示如何查看生产订单的“WIP 报表”。 创建此程序的演示数据公司是 USMF。 此程序是专为成本控制设计的。

1. 单击“成本管理”。
2. 单击“生产订单”。
3. 使用“快速筛选”筛选带有“p000153”值的“生产”字段。
4. 在“操作窗格”中，单击“管理成本”。
5. 单击“生产 WIP 报表”。
6. 在“开始日期”字段中，设置日期为“2012-12-01”。
7. 在“结束日期”字段中，设置日期为“2012-12-31”。
