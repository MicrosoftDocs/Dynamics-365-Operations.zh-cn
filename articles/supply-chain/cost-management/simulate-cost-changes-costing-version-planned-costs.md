# <a name="simulate-cost-changes-by-using-a-costing-version-for-planned-costs"></a>通过将成本计算版本用于计划成本，模拟成本变化

[!include [banner](../includes/banner.md)]

本文说明您如何使用针对计划成本的某一单独成本计算版本模拟成本变化对制造物料的计算成本的影响。

您可以使用针对计划成本的某一单独成本计算版本模拟成本变化对制造物料的计算成本的影响。 使用此单独的成本计算版本来输入反映增加的成本变化的未决成本记录，以及计算对制造物料的成本影响。 由于有效成本回退原则将用于物料清单 (BOM) 计算，只有增加的成本变化必须输入。

## <a name="guidelines-for-defining-the-simulation-costing-version"></a>用于定义模拟成本计算版本的准则
在定义模拟成本计算版本时遵循以下准则：

-   分配**计划成本**的成本计算类型。
-   为成本计算版本分配有意义的标识符，如**模拟**。
-   请确保内容包括成本记录。
-   允许输入成本记录。 不要阻止未决成本的输入。
-   允许输入所有站点的成本记录。 如果指定某个站点，将限制成本记录的输入为该站。
-   阻止激活未决成本。 在模拟成本计算版本中，仅必须为成本记录输入未决成本。
-   不要输入“开始”日期。 将为使用模拟成本计算版本的每个物料清单计算输入计算日期。
-   指定**当前有效**的回退原则。 该回退原则支持出于模拟目的输入增加的成本变化，但使用当前有效的成本作为所有其他成本记录的来源。 我们假定对于所有其他成本记录都存在所有当前有效的成本。
-   指定**版本成本价**的成本价模型，但不限制计算。 例如，该模拟还可以使用**物料清单计算组**成本价模型，以便指示采购物料的成本份额数据来源。
-   指定**多级**分解模式，但不限制计算。 这些模拟可以使用其他分解模式。

## <a name="costing-versions"></a>成本计算版本
以下场景说明如何使用模拟成本计算版本来模拟成本变化的影响。 在输入模拟的成本变化前，请删除模板成本计算版本内的未决成本记录，以便您具有明确的起始点。 使用**物料价格**页查看和删除与物料价格和成本类别价格相关的未决成本记录。

-   模拟某一采购物料的成本变化。 例如，成本变化可能反映关键采购材料成本的预期增减。 若要为某一采购物料定义不同成本，请使用**物料价格**页输入模拟成本计算版本内的未决成本记录。
-   模拟某一成本类别的成本变化。 例如，成本变化可以反映人工费率的预期增减，或者工序资源的小时工资率的增减。 若要为某一成本类别定义不同成本，请使用**成本类别价格**页输入模拟成本计算版本内的未决成本记录。
-   模拟间接成本计算公式中的成本变化。 例如，成本变化可以反映制造开销中的预期增减。 若要定义间接成本计算公式中的变化，请使用**成本计算单设置**页输入模拟成本计算版本中的未决成本记录，以及验证并保存该变化。

在输入模拟成本变化后，计算受成本变化影响的制造物料的成本。 使用**计算**页来用于模拟成本计算版本，并且标识将受成本变化影响的所选制造物料。 如果您没有选择特定物料，物料清单计算将适用于所有制造物料。 或者，您可以使用物料清单计算选项来用于用途更新。 查看模拟成本计算版本中的物料成本记录，以便分析模拟成本变化是如何影响所选制造物料的成本的。 使用**物料价格**页和**计算物料成本**页可以查看和分析成本。
