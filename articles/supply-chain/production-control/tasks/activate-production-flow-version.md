# <a name="activate-a-production-flow-version"></a>启用生产流版本

[!include [banner](../../includes/banner.md)]

应先启用生产流版本，才能为该版本创建看板作业和处理活动。 最佳实践是，在版本的生效开始日期和时间到期前启用此版本。 要完成这项任务，您必须有状态为“草稿”的生产流版本以启用该版本。 


## <a name="find-and-activate-a-production-flow-version"></a>找到并启用生产流版本
1. 转到“生产流”。
2. 在列表中，找到并选择所需记录。
    * 选择“计划状态”字段中为“草稿”的生产流版本。  
3. 在列表中，单击所选行中的链接。
4. 在列表中，找到并选择所需记录。
    * 选择“计划状态”字段中为“草稿”的生产流版本。  
5. 单击“启用”。
6. 单击“确定”。
