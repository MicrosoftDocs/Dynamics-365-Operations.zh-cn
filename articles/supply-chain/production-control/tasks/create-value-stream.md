# <a name="create-a-value-stream"></a>创建价值流

[!include [task guide banner](../../includes/task-guide-banner.md)]

价值流是归于一个或多个生产流的运营单位。 若要定义生产流，您首先必须定义价值流。


## <a name="open-the-form"></a>打开窗体
1. 转到“生产控制”>“设置”>“精艺生产流程”>“价值流”。

## <a name="create-a-new-value-stream"></a>创建新价值流
1. 单击“新建”，以打开对话框。
2. 在“名称”字段中，键入一个值。

## <a name="select-a-value-stream-manager"></a>选择价值流经理
1. 在“经理”字段中，单击下拉按钮以打开查找。
    * 定义价值流经理，以记录组织。  
2. 在列表中，找到并选择所需记录。
3. 在列表中，单击所选行中的链接。

## <a name="save-and-close"></a>保存并关闭
1. 单击“保存”。
2. 关闭该页面。
3. 刷新该页面。
