# <a name="allocate-time-to-jobs-in-a-job-bundle"></a><span data-ttu-id="08a9d-104">分配时间到作业捆绑中的作业</span><span class="sxs-lookup"><span data-stu-id="08a9d-104">Allocate time to jobs in a job bundle</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="08a9d-105">在“制造执行”中，您可以捆绑作业。</span><span class="sxs-lookup"><span data-stu-id="08a9d-105">In Manufacturing execution, you can bundle jobs.</span></span> <span data-ttu-id="08a9d-106">然后您可以在工作列表页同时开始多个作业。</span><span class="sxs-lookup"><span data-stu-id="08a9d-106">You can then start multiple jobs at the same time on the Job list page.</span></span>

<span data-ttu-id="08a9d-107">如果您捆绑作业，则必须定义应如何将所有作业的总登记时间分配给每个作业。</span><span class="sxs-lookup"><span data-stu-id="08a9d-107">If you bundle jobs, you must define how the total registered time for all the jobs should be allocated to each job.</span></span> <span data-ttu-id="08a9d-108">您通过在 **分配参数** 页的 **捆绑类型** 字段中选择以下选项之一定义分配：</span><span class="sxs-lookup"><span data-stu-id="08a9d-108">You define the allocation by selecting one of the following options in the **Bundle type** field on the **Allocation keys** page:</span></span>

-   <span data-ttu-id="08a9d-112">**实时** – 分配实际作业时间。</span><span class="sxs-lookup"><span data-stu-id="08a9d-112">**Real time** – Actual job time is allocated.</span></span> <span data-ttu-id="08a9d-113">可以基于实际工资成本计算成本。</span><span class="sxs-lookup"><span data-stu-id="08a9d-113">The cost can be calculated based on the actual payroll cost.</span></span> <span data-ttu-id="08a9d-114">**注意：** 只有当您的公司在“考勤管理”中使用工资单功能时 **实时** 分配参数才可用。</span><span class="sxs-lookup"><span data-stu-id="08a9d-114">**Note:** The **Real time** allocation key is available only if your company uses the payroll functionality in Time and attendance.</span></span>
