# <a name="production-feedback"></a>生产反馈

[!include [banner](../includes/banner.md)]

本文提供有关向工作人员提供生产作业反馈的生产反馈的信息。 本文包括有关可以更新生产反馈的各个方法的信息。

生产反馈为工作人员提供有关生产作业的反馈。 它记录生产订单的时间和物料消耗量、工序数量和状态以及导致作业或工序失败的错误。 生产反馈可以在与生产订单相关的日记帐中更新。 **生产作业卡**和**生产工艺路线卡**日记帐用于报告每个作业或工序的时间和数量。 对于有关最后作业或工序的报告，成品数量可以报告为已完工入库。 生产反馈还可以在**作业卡终端**和**作业卡设备**页更新。 这些页面使生产反馈在车间更新，并为**生产控制**模块中制造执行功能的一部分。 **作业卡终端**页具有可配置用户界面，以优先级顺序显示所选工作区已发布作业的列表。 它还提供高级选项，如捆绑作业和团队工作。 **作业卡设备**页具有触摸优化的用户界面。 在这两页的生产反馈从生产日记帐更新。
