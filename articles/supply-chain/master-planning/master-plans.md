# <a name="master-plans"></a><span data-ttu-id="26ef1-103">主计划</span><span class="sxs-lookup"><span data-stu-id="26ef1-103">Master plans</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="26ef1-104">使用各种主计划来支持公司的日常运营、模拟要监控的不同计划策略、实现公司策略，例如有关内部绩效或客户满意度的政策。</span><span class="sxs-lookup"><span data-stu-id="26ef1-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="26ef1-105">您可以配置**主计划**页上的主计划。</span><span class="sxs-lookup"><span data-stu-id="26ef1-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="26ef1-106">有以下两种计划类型：</span><span class="sxs-lookup"><span data-stu-id="26ef1-106">There are two types of plans:</span></span>
-   <span data-ttu-id="26ef1-107">**静态计划** – 主计划计算使用当前数据来生成净需求计划。</span><span class="sxs-lookup"><span data-stu-id="26ef1-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="26ef1-108">在下次运行主计划之前，该计划一直保持不变。</span><span class="sxs-lookup"><span data-stu-id="26ef1-108">This plan remains unchanged until the next time that you run master planning.</span></span> <span data-ttu-id="26ef1-109">它是一个运营计划，公司的各类人员（例如采购人员或生产规划人员）可将其用作决策基础，也可使用它来执行日常任务和活动。</span><span class="sxs-lookup"><span data-stu-id="26ef1-109">It is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily tasks and activities.</span></span>
-   <span data-ttu-id="26ef1-110">**动态计划** – 此计划开始时具有主计划所生成的净需求计划。</span><span class="sxs-lookup"><span data-stu-id="26ef1-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="26ef1-111">但是，每当主数据更改时都可以更新该动态计划。</span><span class="sxs-lookup"><span data-stu-id="26ef1-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="26ef1-112">例如在创建新销售订单时。</span><span class="sxs-lookup"><span data-stu-id="26ef1-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="26ef1-113">这样您便可以监控更改的订单网络以及物料可用性，而无需中断其他人正在工作过程中使用的静态计划。</span><span class="sxs-lookup"><span data-stu-id="26ef1-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="26ef1-114">公司可以选择只使用静态计划，或可以同时使用动态计划和动态计划。</span><span class="sxs-lookup"><span data-stu-id="26ef1-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="26ef1-115">此外，还可以将任何主计划配置为反映特定策略或解决某一问题。</span><span class="sxs-lookup"><span data-stu-id="26ef1-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="26ef1-116">示例如下：</span><span class="sxs-lookup"><span data-stu-id="26ef1-116">Examples are as follows:</span></span>
-   <span data-ttu-id="26ef1-117">设置较高的库存水平，以防止缺货。</span><span class="sxs-lookup"><span data-stu-id="26ef1-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="26ef1-118">设置较长的安全宽限期，以防范不可靠的供应商。</span><span class="sxs-lookup"><span data-stu-id="26ef1-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="26ef1-119">还可以设置起始动态计划，以便每次运行主计划时都以新需求计划进行更新。</span><span class="sxs-lookup"><span data-stu-id="26ef1-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="26ef1-120">您可以在**主计划参数**页上指定这些设置。</span><span class="sxs-lookup"><span data-stu-id="26ef1-120">You can specify these settings on the **Master planning parameters** page.</span></span>
