# <a name="cant-cancel-an-intercompany-purchase-order-thats-linked-to-a-sales-order"></a>无法取消链接到销售订单的内部公司采购订单

## <a name="symptoms"></a>故障特征

如果尝试取消链接到销售订单的内部公司采购订单，您可能会收到以下错误消息：

> 无法减少数量，因为剩余更新数量更改了符号。

## <a name="resolution"></a>解决方法

此问题已在 Microsoft Dynamics 365 Supply Chain Management 版本 10.0.13 中修复。 在该版本及更高版本中，您现在可以取消链接到销售订单的内部公司采购订单。
