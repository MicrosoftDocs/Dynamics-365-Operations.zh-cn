# <a name="purchase-orders-dont-show-translated-product-text"></a>采购订单不显示翻译后的产品文本

知识库编号：4612098

## <a name="symptoms"></a>故障特征

您已经为产品名称定义了文本翻译。 但是，无论应用中的语言设置如何，产品名称始终以原始语言显示。

## <a name="resolution"></a>解决方法

产品翻译仅显示在面向外部的文档上。 不显示给内部用户。 有关详细信息，请参阅[与产品相关的翻译常见问题](../../pim/translations-product-related-information.md#where-can-i-view-the-translated-information)。
