# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>只能使用管理员角色访问移动设备应用模拟器

## <a name="symptoms"></a>故障特征

在 Supply Chain Management 中设置仓库时，您不能使用除管理员角色之外的其他任何角色访问移动设备应用模拟器。

## <a name="resolution"></a>解决方法

移动设备应用模拟器设置为只能在管理员帐户中使用。 出于所有测试和实时流程目的，我们建议您使用仓库管理移动应用本身。
