# <a name="invalid-license-plate-or-location-error-when-scanning-in-the-mobile-app"></a>在移动应用中扫描时的牌照或库位无效错误

## <a name="symptoms"></a>故障特征

当您扫描牌照 ID 或库位时，可能会收到以下错误消息：

> 牌照或位置无效。

## <a name="resolution"></a>解决方法

请确保牌照 ID 未在别处保留。 当用户在仓库管理移动应用中扫描的值既是有效库位又是有效牌照 ID 时，过去常会发生此问题。 但是，此问题已在版本 10.0.11 中解决。
