# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>已确认负荷的装运，但未过帐任何行

## <a name="symptoms"></a>故障特征

处理处站仓库工序时，您可能会收到以下消息：

> 已确认负荷 1% 的装运。

但是，未进行进一步过帐。

## <a name="resolution"></a>解决方法

这是有意行为。 装运确认不会影响过帐。 它只是更新装运和负荷状态。 过帐必须在单独的流程中进行。
