# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>在移动应用中扫描牌照 ID 时发生牌照无效错误

## <a name="symptoms"></a>故障特征

在 Warehouse Management 移动应用中扫描牌照 ID 时，您可能会收到以下错误消息：

> 牌照无效。

## <a name="resolution"></a>解决方法

请确保牌照表中存在该牌照 ID，并且牌照上的物料和数量均可用（换言之，它们没有被锁定）。
