# <a name="packing-station-doesnt-show-product-notes"></a>装箱工作站不显示产品注释

知识库编号：4614615

## <a name="symptoms"></a>故障特征

当装箱说明作为基础产品或产品变型的附件添加时，装箱窗体中不显示装箱注释。

## <a name="resolution"></a>解决方法

此系统行为是有意设计的。

在装箱窗体中显示装箱注释的当前逻辑需要注释与装运关联。
