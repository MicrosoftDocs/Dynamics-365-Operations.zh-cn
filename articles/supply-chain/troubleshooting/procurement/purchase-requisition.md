# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>请求更改后您无法在采购申请中添加行

知识库编号：4611211

## <a name="symptoms"></a>故障特征

系统不允许您在请求更改后在采购申请中添加行。

## <a name="resolution"></a>解决方法

您必须撤回工作流。 采购申请处于 *草稿* 状态后，您可以继续对其进行编辑或向其添加行。
