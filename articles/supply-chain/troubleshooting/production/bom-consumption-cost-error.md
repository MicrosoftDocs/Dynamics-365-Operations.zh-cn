# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>由于物料清单 consumptionCost 值错误无法结束生产订单

## <a name="symptoms"></a>故障特征

当您尝试结束生产订单时，将收到以下错误消息：

> 从库存签发时计算物料清单 consumptionCost 值必须为负。

## <a name="resolution"></a>解决方法

此问题在版本 10.0.15 中已修复。
