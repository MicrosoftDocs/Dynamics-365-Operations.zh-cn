# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>在全球通讯簿中更改工作人员姓名后不会更新资源名称

## <a name="symptoms"></a>故障特征

如果您在全球通讯簿中更改工作人员姓名，匹配的资源姓名不会在主资源组中更新。

## <a name="resolution"></a>解决方法

现在不会自动更新匹配的资源名称。 若要解决此问题，您必须手动更新资源姓名。
