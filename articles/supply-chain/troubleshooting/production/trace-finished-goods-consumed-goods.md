# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>不能根据消耗品的序列号跟踪成品的序列号

## <a name="symptoms"></a>故障特征

您不能根据生产订单生产这些成品所消耗的物料的序列号来跟踪成品的序列号。

## <a name="workaround"></a>解决方法

解决方法是创建数量为 1 的生产订单。
