# <a name="the-delay-value-isnt-updated-when-you-reschedule-a-planned-order"></a>重新计划计划订单时，延迟值不更新

## <a name="symptoms"></a>故障特征

重新计划计划订单时，延迟值不更新。

## <a name="resolution"></a>解决方法

要更新计划订单的延迟，请打开计划订单的 **重新计划** 对话框。 在 **分解** 快速选项卡上，确保将 **重新计划后执行分解** 选项设置为 *是*。
