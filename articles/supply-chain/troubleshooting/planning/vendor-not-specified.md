# <a name="vendor-isnt-specified-when-planned-orders-are-firmed"></a>确认计划订单时未指定供应商

错误代码：SYS23532

## <a name="symptoms"></a>故障特征

当您尝试确认计划订单时，将收到以下错误消息：

> 未指定供应商

## <a name="resolution"></a>解决方法

若要指定供应商，请按照以下步骤操作。

1. 打开缺少供应商的计划订单。
1. 在 **计划供应** 快速选项卡上的 **供应商** 字段中，选择供应商。
