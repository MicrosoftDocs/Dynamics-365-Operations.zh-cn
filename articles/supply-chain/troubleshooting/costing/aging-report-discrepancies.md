# <a name="inventory-aging-report-discrepancies"></a>库龄报表差异

## <a name="symptoms"></a>故障特征

**库龄报表** 在不同的存储维度（如站点或仓库）查看时，显示不同的值。

## <a name="resolution"></a>解决方法

有关报告逻辑的详细信息，请参阅[库龄报表示例和逻辑](../../cost-management/inventory-aging-report.md)。
