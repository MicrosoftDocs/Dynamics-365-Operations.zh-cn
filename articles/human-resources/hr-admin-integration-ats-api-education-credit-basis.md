# <a name="education-credit-basis"></a>教育信用基础

本主题介绍 Dynamics 365 Human Resources 的教育信用基础选项集。

物理名称：mshr_hcmeducationcreditbasis

此枚举为应聘者教育记录的教育信用基础提供值选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 空白 | 未选择值。 |
| 200000001 | 半年  | 半年。 |
| 200000002 | 季度 | 季度。 |
| 200000003 | 四个月 | 四个月。 |
| 200000004 | 术语 | 学期。 |
| 200000005 | 其他 | 其他。 |
