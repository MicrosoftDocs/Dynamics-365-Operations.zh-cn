# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Dynamics 365 Human resources 中已删除或弃用的功能

本文介绍 Microsoft Dynamics 365 Human Resources 中已经删除或计划删除的功能。

- *已移除* 的功能在产品中不再可用。
- *已弃用* 的功能在活跃的开发中不存在，而且在将来的更新中可能被移除。

此列表旨在帮助您在您自己的计划中考虑这些功能的移除和弃用。

## <a name="features-removed-or-deprecated-in-human-resources"></a>Human Resources 中已删除或弃用的功能 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>在 Human Resources（独立版）中创建新 LCS 项目 

[预配 Human Resources](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **弃用/移除的原因** | 已替换为在财务和运营基础结构上创建项目。 要详细了解基础结构合并，请参阅 [Dynamics 365 Human Resource 基础结构合并常见问题解答](/hr-infrastructure-merge-faq)。 |
| **被另一个功能取代？**   | 是 |
| **影响的产品区域**         | 应用程序 |
| **部署选项**              | 所有 |
| **Status**                         | 已弃用：从 2022 年 7 月，我们不再支持在 Human Resources（独立版）中创建新项目。 |
