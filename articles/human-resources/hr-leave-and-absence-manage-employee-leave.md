# <a name="manage-employee-leave"></a>管理员工休假

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

您可以按休假类型管理员工的休假。 这包括终止休假登记和调整休假类型余额。 

## <a name="adjust-leave-balances"></a>调整休假余额

1. 在员工的记录上，选择 **休假**。

2. 选择 **休假和缺勤设置**。

3. 选择 **调整余额**。

4. 选择 **休假类型**。

5. 输入 **调整金额**。 

6. （可选）您可以选择 **日期**。 

调整员工的休假余额时，您可以包括原因代码和注释。 

现在，当您悬停在任何休假余额上时，将显示以下信息：

- **可用** – **本年度总计** 值减去 **本年度所用** 值。
- **本年度总计** – 该年度的所有应计、调整和结转。
- **本年度所用** – 所有已批准的休假。
