## <a name="option-set-process-type"></a>选项集：流程类型

物理名称：mshr_hrmCompProcessType

此枚举为 Human Resources 虚拟实体中的流程类型提供选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 否 | 无。 |

| 200000002 | 已停征 | 已停征。 |
