# <a name="set-up-employment-types"></a>设置雇用类型

[!include [banner](includes/preview-feature.md)]

在 Dynamics 365 Human Resources 中设置雇用类型以匹配您的工作人员。

1. 在**福利管理**工作区中，在**设置**下，选择**雇用类型**。

2. 选择**新建**。

3. 为以下字段指定值：

   | 字段 | 说明 |
   | --- | --- |
   | 雇用类型 | 雇用类型的唯一标识。 |
   | 说明 | 类型类型的描述。 |

4. 选择**保存**。
