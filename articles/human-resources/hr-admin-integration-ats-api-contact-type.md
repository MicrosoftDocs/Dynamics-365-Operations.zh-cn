# <a name="contact-type"></a>联系人类型


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文介绍 Dynamics 365 Human Resources 的联系人类型选项集。

物理名称：mshr_logisticselectronicaddressmethodtype

此枚举为应聘者联系人类型提供值选项集。 

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | None | 未选择类型。 |
| 200000001 | 电话 | 电话号码: |
| 200000002 | 电子邮件地址 | 电子邮件地址。 |
| 200000003 | URL | 网站 URL。 |
| 200000004 | 电报 | 电报号码。 |
| 200000005 | 传真 | 传真号。 |
| 200000006 | Facebook | Facebook 帐户。 使用用户 ID 标识。 |
| 200000007 | Twitter | Twitter 帐户。 使用“@用户名”标识。 |
| 200000008 | LinkedIn | LinkedIn 帐户。 使用用户名标识。 |
