# <a name="payroll---employment-type-option-set"></a>工资单 -“雇用类型”选项集

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文介绍 Dynamics 365 Human Resources 的“雇用类型选项集。

## <a name="option-set-employment-type"></a>选项集：雇用类型

此枚举为员工提供雇用类型选项集。 在 mshr_hcmemploymenttype 选项集中提供。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 盘点责任人 | 盘点责任人 |
| 200000001 | 合同工 | 合同工 |
