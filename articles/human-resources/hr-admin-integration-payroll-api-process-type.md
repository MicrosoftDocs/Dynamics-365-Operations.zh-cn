# <a name="process-type"></a>流程类型


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文介绍 Dynamics 365 Human Resources 的“否/是”选项集。

## <a name="option-set-process-type"></a>选项集：流程类型

物理名称：mshr_hrmCompProcessType

此枚举为 Human Resources 虚拟实体中的流程类型提供选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 否 | 无。 |
| 200000001 | 每年 | 每年。 |
| 200000002 | 周年 | 周年纪念日。 |
| 200000003 | 非周期 | 非周期。 |
| 200000004 | ½ 年 | 半年。 |
| 200000005 | 季度 | 季度。 |
| 200000006 | 月度 | 月度。 |
