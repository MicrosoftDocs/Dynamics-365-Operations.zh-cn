# <a name="set-up-tier-codes"></a>设置层代码

[!include [banner](includes/preview-feature.md)]

Dynamics 365 Human Resources 将层代码用于员工和雇主支付的福利比率。 您可以定义无限数量的层代码并设置比率计算以满足公司的需求。

1. 在**福利管理**工作区中，在**设置**下，选择**层代码**。

2. 选择**新建**。

3. 为以下字段指定值：

   | 字段 | 说明 |
   | --- | --- |
   | 层代码 | 用于识别层代码的唯一福利层代码或数字。 |
   | 说明 | 福利层代码的描述。 |

4. 选择**保存**。
