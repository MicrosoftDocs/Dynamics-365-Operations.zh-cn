# <a name="job-detail-api"></a>作业详细信息 API

本主题提供 Dynamics 365 Human Resources 中作业详细信息实体的详细信息和示例查询。

## <a name="properties"></a>属性

| 属性<br>**物理名称**<br>**_类型_** | 使用 | 说明 |
| --- | --- | --- |
| **工作 ID**<br>mshr_jobid<br>*字符串* | 只读<br>必填 | 作业的唯一 ID。 |
| **市场价格低阈值**<br>mshr_marketpricelowthreshold<br>*十进制* | | 系统生成的用于唯一标识职位的 GUID 值。  |
