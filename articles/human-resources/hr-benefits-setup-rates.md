# <a name="configure-rates"></a>配置费率


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

比率定义雇主和员工向福利缴费的数量。 此值可以是弹性信贷金额或数量，具体取决于您的配置。

使用费率根据几个因素来确定员工和雇主为每种福利支付多少费用。 覆盖范围比率是有有效日期的，因此您可以保留比率的历史记录。 

## <a name="set-up-rates"></a>设置比率

1. 在 **福利管理** 工作区中，在 **设置** 下，选择 **比率**。

2. 选择 **新建**。

3. 为以下字段指定值：

   | 字段 | 说明 |
   | --- | --- |
   | **费率** | 标识福利比率的唯一名称。 |
   | **说明** | 福利比率的描述。 |
   | **生效日期** | 比率生效的日期。 当前系统日期是默认值。 此日期应在您的福利期间内或之前。 一个好的做法是将此日期设置为福利计划的日期。 |
   | **到期日期** | 比率的结束日期。 默认值是 12/31/2154（表示永不）。 |
   | **使用层** |  如果您有必须用于确定比率的逻辑，请使用此字段。 例如，如果比率必须根据年龄增加，则在此处选择一个值。 为一层福利比率选择 **单层**，或为两层福利比率选择 **双层**。 双层的一个示例是基于性别和年龄的层。 选择值后，选择 **操作**，然后选择 **层比率**。 如果您有不改变的固定比率，请将此字段留空。 |
   | **付款频率** | 指定应向福利提供商支付福利金比率的频率。 您在本主题后面介绍的页面上输入的比率将基于您在此处指定的付款频率。 例如，如果您在此字段中输入 **每月**，员工比率输入 **100 美元**，则会假定该福利每月将扣掉员工 100 美元。 但是，根据员工记录中设置的福利付款频率，员工可能每月支付两次。 在这种情况下，当员工登录 **员工自助服务** 时，他们支付的金额将为 50 美元，因为 **员工自助服务** 显示的比率基于员工的付款频率。 |
   | **付薪频率舍入** | 舍入比率的方法有：标准、截断、正常、舍尾和进位。 </br></br><ul><li>**标准** – 始终向上舍入。 例如，10.611 将舍入到 10.62。 -10.231 将舍入到 -10.23。 </li><li>**截断** – 始终向下舍入。 例如，10.619 将舍入到 10.61。 -10.231 将舍入到 -10.24。 </li><li>**正常** – 以 5 结尾或大于 5 的十进制值将从零舍入。 以 4 结尾或小于 4 的十进制值将向零舍入。 例如，10.615 将舍入到 10.62。 -10.235 将舍入到 -10.24。 10.614 将舍入到 10.61。 -10.234 将舍入到 -10.23。 </li><li>**舍尾** – 向零舍入。 例如，10.619 将舍入到 10.61。 -10.231 将舍入到 -10.23。 </li><li>**进位** – 从零舍入。 例如，10.619 将舍入到 10.62。 -10.231 将舍入到 -10.24。 |
   | **非吸烟者员工金额** | 福利提供商向不吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **非吸烟者雇主金额** | 福利提供商向不吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **吸烟者员工金额** | 福利提供商向吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **吸烟者雇主金额** | 福利提供商向吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **管理金额** | 第三方管理人收取的管理金额。 这是雇主支付给第三方管理人的金额，应基于比率设置的付款频率。 |
   | **弹性信贷比率** | 福利使用的弹性信贷的数量。 这仅适用于与弹性信贷项目关联的福利计划的比率。 如果您使用层比率，在“操作”>“层比率”中定义弹性信贷比率。 |
   | **更改生效日期** | 福利比率更改的生效日期。 如果您运行比率更改更新处理，系统将自动更改福利比率并更新与此比率相关联的所有福利计划。 除非您希望系统根据此比率自动更新员工福利计划，否则请勿设置此日期。 此日期通常为将来的自动比率更改处理保留的。 **更改生效日期** 必须在福利比率生效日期和到期日期之间。 |
   | **已完成比率更改** | 在比率更改更新处理完成福利比率更改后，将自动选中 **已完成比率更改** 复选框。 |

4. 要跟踪并维护对福利比率设置的更改，请选择 **操作**，然后选择 **维护版本**。

5. 选择 **保存**。 

## <a name="set-up-tier-rates"></a>设置层比率

如果比率取决于多种因素，您可以在比率设置中使用层比率。 例如，您可以将层比率配置为指示如果您的年龄不超过 34.99 岁，非吸烟者金额为 2。 如果您的年龄不超过 39.99 岁，非吸烟者金额为 3。

您也可以使用双层。 如果您在 **比率设置** 窗体上为 **使用层** 值选择了 **双层**，您可以基于两个维度定义比率。 例如，您可以配置一个双层制度，指示如果您是男性并且年龄不超过 34.99 岁，非吸烟者金额为 2。 如果您是男性并且年龄不超过 39.99 岁，非吸烟者金额为 3。 如果您是女性并且年龄不超过 34.99 岁，非吸烟者金额为 1.8。 如果您是女性并且年龄不超过 39.99 岁，非吸烟者金额为 2.8。

> [!IMPORTANT]
> 工作人员记录上 **个人信息** 下有一个选项用于指示该员工是否为吸烟者。 如果将员工记录为吸烟者，将使用吸烟者比率。 （吸烟者指示永远不会向员工显示。）
   
1. 在 **福利管理** 工作区中，在 **设置** 下，选择 **比率**。

2. 从列表中选择一个或多个比率，选择 **操作**，然后选择 **层比率**。

3. 选择 **新建**。

3. 为以下字段指定值：

   | 字段 | 说明 |
   | --- | --- | 
   | **说明** | **说明** 字段值将应用来自比率设置记录的说明内容。 这可以帮助您确定层比率链接到哪个比率设置。 |
   | **层代码** | 选择层代码。 层代码在 **层代码** 页面中定义。 系统将自动在左侧的网格中显示层代码的描述。 |
   | **层类型** | 指定应将哪个字段用作层比率计算流程的选择条件。 例如:</br></br><ul><li>如果使用 **年龄**，系统将在福利比率计算流程中使用员工的出生日期。</li><li>如果使用 **薪金**，系统将在福利比率计算流程中使用员工的年度福利薪金。</li><li>如果使用 **工作类型**，将使用员工当前的有效职位记录，通过链接到该职位的工作记录来确定工作类型。</li></ul></br></br>层类型有 **年龄**、**薪金**、**身体**、**性别**、**全职等价**、**工作类型**、**薪酬区域** 和 **层级**。 | 
   | **层级** | 在福利比率计算流程中与层类型一起使用的值。 例如:</br></br><ul><li>如果层类型为 **年龄**，这将是年龄值。</li><li>如果层类型为 **薪金**，这将是薪金金额。</li><li> 如果层类型为 **工作类型**，这将是工作类型。</li></ul></br></br>如果层类型为 **年龄** 或 **薪金**，**层级** 字段中的值将表示层的上限。 如果层类型为 **工作类型**，在选择层比率时将使用精确匹配方法。 |
   | **计算类型** | 指定如何使用计算金额字段中的金额以及必要时要执行的数学计算。 如果计算类型是固定金额，将按原样使用金额字段。 如果计算类型是按薪金或覆盖范围的美元金额，将在数学计算中使用计算金额和计算方向。</br></br>如果计算类型为按薪金的美元金额，将使用以下数学方程式：</br></br>年度福利薪金除以计算金额（对所得值进行向上或向下舍入），然后乘以员工或雇主的吸烟者或非吸烟者金额。</br></br>如果计算类型为按覆盖范围的美元金额，将使用以下数学方程式：</br></br>覆盖范围金额除以计算金额（对所得值进行向上或向下舍入），然后乘以员工或雇主的吸烟者或非吸烟者金额。</br></br>在这两种计算方法中，均使用计算方向来确定是对年度福利薪金或覆盖范围金额除以计算金额所得的值进行向上还是向下舍入。 |
   | **计算金额** | 在福利比率计算流程中要使用的金额。 此金额是进行层比率数学计算时所用的除数。 |
   | **计算方向** | 对计算所得金额应采用的舍入方向。 系统支持三种计算方向：空（精确方法）、**增加** 和 **减小**。</br></br><ul><li>如果方向为空，系统将使用薪金/覆盖范围金额除以计算金额所得的精确计算值。 如果此值带有小数，则其计算中将使用此小数。</li><li>如果为 **增加**，将增加薪金/覆盖范围金额除以计算金额所得的数学计算值，将其进位到下一个整数，这意味着 12.25 将增加为 13。</li><li>如果为 **减小**，将减小薪金/覆盖范围金额除以计算金额所得的数学计算值，仅保留当前整数，这意味着 12.25 将减小为 12。</li></ul> |
   | **非吸烟者员工金额** | 福利提供商向不吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **非吸烟者雇主金额** | 福利提供商向不吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **吸烟者员工金额** | 福利提供商向不吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **吸烟者雇主金额** | 福利提供商向不吸烟员工收取的金额。 这是雇主支付给福利提供商的金额，应基于比率设置的付款频率。 |
   | **管理金额** | 第三方管理人收取的管理金额。 这是雇主支付给第三方管理人的金额，应基于比率设置的付款频率。 |
   | **弹性信贷非吸烟者比率** | 根据为非吸烟者的层级定义的计算，福利使用的弹性信贷的数量。 例如，如果计算类型为 **按覆盖范围的美元金额**，计算金额为 10,000，弹性信贷非吸烟者比率为 6，这意味着如果不吸烟的员工选择 10,000 美元覆盖范围，将使用 6 个弹性信贷。 如果他们选择 20,000 美元覆盖范围，将使用 12 个弹性信贷，依此类推。 |
   | **弹性信贷吸烟者比率** | 根据为吸烟者的层级定义的计算，福利使用的弹性信贷的数量。 |

5. 选择 **保存**。
