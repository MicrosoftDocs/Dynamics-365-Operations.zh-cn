# <a name="screening-frequency"></a>筛选频率


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文介绍 Dynamics 365 Human Resources 的筛选频率选项集。

物理名称：mshr_hcmfrequencyunit

此枚举为筛选频率提供值选项集。 

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 一次性 | 筛选只需要执行一次。 |
| 200000001 每天 | 筛选频率按天计算。 |
| 200000002 每周 | 筛选频率按周计算。 |
| 200000003 每月 | 筛选频率按月计算。 |
| 200000004 每年 | 筛选频率按年计算。 |
