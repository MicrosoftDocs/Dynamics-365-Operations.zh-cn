# <a name="regulatory-job-category"></a>监管工作类别

本主题介绍 Dynamics 365 Human Resources 的监管工作类别选项集。

物理名称：mshr_hcmregulatoryjobcategory

此枚举指定监管工作类别选项集值的选项集。 在现有 mshr_hcmregulatoryjobcategory 选项集中提供。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | None | 无。 |
| 200000001 | 主管人员 | 主管/高级行政人员和经理。 |
| 200000002 | 管理员 | 一级/中级行政人员和经理。 |
| 200000003 | 专业 | 专业人员。 |
| 200000004 | 技术人员 | 技术人员。 |
| 200000005 | SalesWorker | 销售人员。 |
| 200000006 | 管理 | 行政支持人员。 |
| 200000007 | CraftWorker | 工艺工人。 |
| 200000008 | 作业 | 作业人员。 |
| 200000009 | 工人 | 工人/助理工人。 |
| 200000010 | ServiceWorker | 服务人员。 |
