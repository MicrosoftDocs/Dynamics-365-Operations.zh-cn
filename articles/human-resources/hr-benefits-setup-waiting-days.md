# <a name="configure-waiting-days"></a>配置等待天数


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

在 Microsoft Dynamics 365 Human Resources 中，等待天数建立用于福利计划的每月里程碑。 例如，等待天数 15 表示每个月的第 15 天。 

1. 在 **福利管理** 工作区中，在 **设置** 下，选择 **等待天数**。

2. 选择 **新建**。

3. 为以下字段指定值：

   | 字段 | 说明 |
   | --- | --- |
   | **等待天数** | 等待天数的唯一标识符。 |
   | **说明** | 等待天数的描述。 |
   | **天** | 系统应设置为等待期间日期的那个月的那一天。 |
   
4. 选择 **保存**。
