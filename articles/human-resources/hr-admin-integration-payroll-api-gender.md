# <a name="payroll---gender-option-set"></a>工资单 - 性别选项集

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本主题介绍 Dynamics 365 Human Resources 的性别选项集。

## <a name="option-set-gender-option"></a>选项集：性别选项

此枚举为员工提供性别选项集。 在 mshr_hcmpersongender 选项集中提供。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | None | 未指定性别。 |
| 200000001 | 男 | 男。 |
| 200000002 | 女 | 女。 |
| 200000003 | 非特定 | 选择非特定性别。 |
