# <a name="configure-future-life-events"></a>配置未来的生命事件

[!include [banner](includes/preview-feature.md)]

您可以在 Dynamics 365 Human Resources 中计划未来的生命事件。

1. 在**福利管理**工作区中，在**设置**下，选择**未来生命事件**。

2. 选择**新建**。

3. 为以下字段指定值：

   | 字段 | 说明 |
   | --- | --- |
   | 生命事件日期 | 系统将处理在登记期间直到该日期为止的所有事件。 |
   | 已记录生命事件 | 记录生命事件的日期和时间。 |
   | 日志类型 | 显示操作是否为以下操作之一：</br></br>- **更新** – 对跟踪生命事件的现有记录进行更改</br></br>- **插入** – 创建新的生命事件记录 |
   | 生命事件类型 ID | 生命事件类型的唯一标识符。 |
   | 生命事件类型 | 促进更新员工福利登记的催化剂。 有关更多详细信息，请参阅“生命事件触发器”一节。 |
   | 状态 | 生命事件是否已处理。 |
   | 扁平 | 未来生命事件的行号。 |

4. 选择**保存**。
