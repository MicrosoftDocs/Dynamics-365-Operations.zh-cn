# <a name="request-a-leave-of-absence"></a>申请休假

>[!Important]
>本文中提到的功能目前对独立 Dynamics 365 Human Resources 上的客户可用。 在 Finance 版本 10.0.26 之后，部分或全部功能将作为未来版本的一部分在 Finance 基础结构上提供。


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

您可以提交休假请求，并在 Dynamics 365 Human Resources 中查看休假请求的状态。

## <a name="request-a-leave-of-absence"></a>申请休假

1. 在 **员工自助服务** 工作区中，在 **休息时间余额** 磁贴中选择 **更多** (...)。

2. 要提交休假请求，请选择 **申请休假**。

3. 输入 **休假类型**、**开始日期** 和 **结束日期** 信息。

4. 如果您需要提交支持文档，请在 **附件** 下选择 **上载**。

5. 如果需要，在 **注释** 中输入信息。

6. 准备好提交请求时选择 **提交**。 否则，选择 **保存草稿**。


## <a name="view-leave-of-absence-request-status"></a>查看休假请求状态

1. 在 **员工自助服务** 工作区中，在 **休息时间余额** 磁贴中选择 **更多** (...)。

2. 要查看您的休假请求，请选择 **查看休假**。

## <a name="update-a-leave-of-absence-request"></a>更新休假请求

1. 在 **员工自助服务** 工作区中，在 **休假** 磁贴中选择 **更多 (...)**。
2. 选择要更新的休假申请，然后选择 **更新休假**。
3. 在 **结束日期** 字段中，适当更新值来延长或缩短休假时间。
4. 结束日期确认后，将 **确认结束日期** 选项设置为 **是**。
5. 在 **确认结束日期** 选项设置为 **是** 后，您可以上载重返工作岗位通知。 然后选中复选框确认已上载重返工作岗位通知。
6. 选择 **提交** 更新休假申请。

## <a name="cancel-a-leave-of-absence-request"></a>取消休假申请

1. 在 **员工自助服务** 工作区中，在 **休假** 磁贴中选择 **更多 (...)**。
2. 选择要取消的休假申请，然后选择 **更新休假**。
3. 将 **取消休假** 选项设置为 **是**。
4. 选择 **提交** 取消休假申请。

## <a name="importing-leave-requests-from-other-systems-or-older-systems"></a>从其他系统或旧系统导入休假请求

要从另一个系统导入休假请求，您必须通过常规工作流创建适当的休假交易。 或者，您可以导入处于已完成状态的休假银行交易和休假请求。 请注意，如果您仅导入休假请求，不会自动创建休假银行交易。
