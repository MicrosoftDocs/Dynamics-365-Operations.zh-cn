# <a name="award-type"></a>奖励类型


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文介绍 Dynamics 365 Human Resources 的奖励类型选项集。

## <a name="option-set-award-type"></a>选项集：奖励类型

物理名称：mshr_HrmCompVarAwardEmplType

此枚举为 Human Resources 虚拟实体中的可变薪酬奖励类型提供选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 一般奖励 | 一般奖励。 |
| 200000001 | 总计固定付薪 | 总计固定付薪。 |
