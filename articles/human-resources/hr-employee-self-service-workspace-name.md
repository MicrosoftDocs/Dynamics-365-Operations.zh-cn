# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="aec13-103">更改员工自助服务工作区名称</span><span class="sxs-lookup"><span data-stu-id="aec13-103">Change Employee self service workspace name</span></span>

<span data-ttu-id="aec13-104">如果您有志愿者或其他非员工人员，则可能需要更改**员工自助服务**工作区。</span><span class="sxs-lookup"><span data-stu-id="aec13-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="aec13-105">您可以将此工作区更改为**自助服务**。</span><span class="sxs-lookup"><span data-stu-id="aec13-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="aec13-106">更改**员工自助服务**工作区的名称也会更改 Dynamics 365 Human Resources 内部使用的菜单项。</span><span class="sxs-lookup"><span data-stu-id="aec13-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="aec13-107">如果您以前对 **HcmEmployeeSelfServiceWorkspace** 菜单项应用了安全性自定义，我们建议对 **HcmSelfServiceWorkspace** 应用相同的更改以保持相同。</span><span class="sxs-lookup"><span data-stu-id="aec13-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="aec13-108">在 Human Resources 中，选择**人事管理**，选择**链接**，然后选择**人力资源参数**。</span><span class="sxs-lookup"><span data-stu-id="aec13-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

3. <span data-ttu-id="aec13-110">在**显示名称**下，选择**自助服务**。</span><span class="sxs-lookup"><span data-stu-id="aec13-110">Under **Display name**, select **Self service**.</span></span>

   ![将员工自助服务工作区名称更改为“自助服务”](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="aec13-112">选择**保存**。</span><span class="sxs-lookup"><span data-stu-id="aec13-112">Select **Save**.</span></span>
