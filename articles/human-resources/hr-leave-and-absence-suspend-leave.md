# <a name="suspend-leave"></a><span data-ttu-id="fbdac-103">暂停休假</span><span class="sxs-lookup"><span data-stu-id="fbdac-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="fbdac-104">可以暂停员工的休假，以便停止处理所选休假类型的休假应计项目。</span><span class="sxs-lookup"><span data-stu-id="fbdac-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="fbdac-105">暂停员工的休假和缺勤</span><span class="sxs-lookup"><span data-stu-id="fbdac-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="fbdac-106">在员工的记录上，选择 **休假**。</span><span class="sxs-lookup"><span data-stu-id="fbdac-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="fbdac-107">选择 **暂停休假**。</span><span class="sxs-lookup"><span data-stu-id="fbdac-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="fbdac-108">选择 **新建**。</span><span class="sxs-lookup"><span data-stu-id="fbdac-108">Select **New**.</span></span>

4. <span data-ttu-id="fbdac-109">在 **暂停休假假期额度记录** 对话框中，选择暂停的 **休假类型** 和 **开始日期** 与 **结束日期**。</span><span class="sxs-lookup"><span data-stu-id="fbdac-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

5. <span data-ttu-id="fbdac-110">（可选）也可以为暂停添加 **注释**。</span><span class="sxs-lookup"><span data-stu-id="fbdac-110">Optionally, you can add a **Comment** for the suspension.</span></span> 

<span data-ttu-id="fbdac-111">如果在已暂停员工的休假时处理应计项目，则不会为暂停该团队休假类型执行累积。</span><span class="sxs-lookup"><span data-stu-id="fbdac-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
