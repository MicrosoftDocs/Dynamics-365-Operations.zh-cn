# <a name="employee-and-manager-self-service-overview"></a>员工和经理自助服务概览


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文提供了“员工自助服务”和“经理自助服务”工作区的概述。

## <a name="edit-personal-details"></a>编辑个人详细信息

如果您需要添加或更改任何个人信息，请参阅[编辑个人信息](hr-employee-manager-self-service-edit-personal-information.md)。

## <a name="user-not-assigned-to-a-worker-record"></a>用户未分配给工作人员记录

如果您尚未将用户链接到 **用户** 页面中的 **工作人员** 记录，将显示以下消息：

**您的用户 ID 未与系统中的员工记录关联。在二者关联之前，您将无法查看或更新您的信息。请与您的经理或支持团队联系以获取帮助。**

要将用户与 **工作人员** 记录关联，请导航到 **用户** 并选择用户。 选择 **编辑**，在页面的 **人员** 字段中添加相应的工作人员，然后选择 **保存**。 您现在应该可以访问 **员工自助服务**。

## <a name="security-requirements-for-employee-and-manager-self-service"></a>员工和经理自助服务的安全性要求

员工和经理自助服务需要两个安全角色：

- 员工需要员工角色。
- 经理同时需要员工和经理角色。

>[!NOTE]
>您还可以使用自定义角色来访问员工和经理自助服务，只要它们已被授予对员工和经理工作区的访问权限。<br>
>经理对员工信息的访问基于 Human Resources 中定义的当前职位行层次结构。

## <a name="employee-self-service"></a>员工自助服务

**我的信息** 选项卡显示 **员工自助服务** 的以下信息。  

### <a name="summary"></a>摘要

**分配给我的工作项** 显示分配给员工的所有审核和工作流项目。 您可以配置工作流项目以向用户发送电子邮件。

**分配给我的调查表** 显示所有直接分配给员工或组的计划的调查表。

**公司目录** 让员工可以查找与组织中的个人相关的信息。 所有员工都可以使用公共联系信息。 公司目录仅限于员工已登录的公司。

**团队日历** 显示您团队的日历信息。 有关详细信息，请参阅[查看团队和公司日历](hr-employee-self-service-calendar.md)。

### <a name="my-career-information"></a>我的职业信息

**员工自助服务** 的 **我的职业信息** 部分显示与休假和缺勤、绩效管理、能力、福利、任务和附件相关的卡片。

**休息时间余额** 磁贴显示所有已登记计划的余额。 此磁贴根据您的应计方式预测您的余额。 您可以输入和提交休息请求，之后请求将经过审核工作流程。 有关休假和缺勤的详细信息，请参阅[休假和缺勤概述](hr-leave-and-absence-overview.md)。

**任务** 磁贴显示分配给您的任务，允许您查看和管理它们。

**下一个注册的课程** 显示您注册的下一个课程。 您可以查看和注册任何公开课程。 所有公开的可供注册的课程的状态为 **已开始**，允许员工自助注册以显示。 根据您组织的设置，您的课程注册可能会经过审核流程。

**证书** 磁贴显示证书和最接近当前日期的证书的到期日期。 您可以更新、添加或删除证书。 根据您组织的设置，证书更新可能会经过审核流程。

**已计划的下一次审核** 显示您的下一次绩效审核。 您可以从此磁贴开始新的审核。 您的经理或人力资源代表也可以发起审核。 根据组织的设置，您可能还可以查看、更新、提交、退出审核。

您可以使用 **绩效目标** 管理您的目标。 此磁贴显示您所具有的每种状态（**未开始**、**正在跟踪** 和 **需要改进**）的目标数。 您可以根据基于被分配角色的安全性来创建、更新和删除目标。 如果需要，可以从组或模板添加新目标。 经理和人力资源也可以代表员工创建目标，并确定每个目标的详细程度。 经理和员工可以就目标进行协作，并更新活动、度量和状态。 您还可以包含附件。

您可以在 **全部技能** 磁贴上查看您的现有技能。 您可以更新技能、添加新技能或删除不再相关的技能。 根据您组织的设置，对技能的更改可能会经过审核流程。

您可以在 **薪酬** 上查看您的当前薪酬。 选择 **显示** 查看您的年度付薪和上一次的增加金额。 如果您受雇于多家公司，每个年度金额都会显示。 要查看详细的薪酬历史记录，请选择 **年度薪水** 金额打开 **固定和可变薪酬历史记录** 页。 将来的薪酬不会在此页中显示。 如果您有一份以上的工作，可以在此页中在公司之间切换来查看您的薪酬历史记录，无需登录每个公司。

使用 **附件** 磁贴查看和管理文档。 您可以管理所有 **外部** 附件。 人力资源和员工均可通过 **员工自助服务** 或 **工作人员** 页添加附件。 附件默认设置为 **外部**。

### <a name="additional-information"></a>其他信息

此部分提供其他 **员工自助服务** 区域的链接，类似于 **我的职业信息** 部分。

通过 **福利** 链接注册福利。 有关福利管理的详细信息，请参阅[福利概述](hr-benefits-management-overview.md)。

在 **绩效** 下，您可以选择 **绩效日记帐** 来创建同时用于绩效目标和审核的绩效日记帐条目。 您可以选择 **发送反馈** 为组织内的其他员工提供反馈。 根据组织的设置，电子邮件可能会发送给收件人、发件人和经理。 您可以将反馈发送给组织内的所有员工。 发送反馈不受公司的限制。

在 **能力** 下，您可以对 **课程**、**教育**、**诚信职位** 和 **专业经验** 进行更改。 根据您组织的设置，对这些能力的更新可能会经过审核流程。

您可以在 **组织** 下查看工作详细信息。 工作详细信息包括技能、证书以及您的主要职位的职责范围。 您还可以查看登记到您名下的任何借出设备。 根据您组织的设置，对借出设备的更改可能会经过审核流程。

在 **调查表** 下，您可以看到完整的调查表。 您还可以查看尚未完成的公司范围的调查表。 您可以选择在任何时间完成调查表。 调查表的作者可以确定时间范围以及该调查表适用的对象。

您可以在 **人力资源参数** 中配置用户定义的链接。 例如，您可以定义指向工资单、年终文档或外部解决方案的链接。 这些链接显示在此部分的底部，但您可以使用个性化设置来移动它们。

您还可以通过在 **员工自助服务** 工作区嵌入 Power Apps 来创建其他选项卡。 使用 **设置** 菜单个性化任何 Power Apps 的页面。 在 **设置** 菜单中，您可以选择添加 Power App、完成详细信息、插入应用。 默认情况下，Power Apps 显示为序列中的第一个选项卡。 您可以使用标准个性化设置更改此顺序。

## <a name="my-team"></a>我的团队

**我的团队** 选项卡显示 **经理自助服务** 的以下信息。 只有经理可以访问 **我的团队** 选项卡。

### <a name="personnel-actions"></a>人事行动

人事行动根据 **人力资源共享参数** 和 **人力资源参数** 中的配置选项显示。 为 **工作人员** 启用后，人事行动将启用新的菜单选项，包括：

- **请求新员工**
- **请求新的合同工**
- **请求工作人员重新分配**
- **请求终止**
- **请求更改薪酬**

您可以将这些选项配置为经过可选的审核与审批工作流。

启用 **职位行动** 将启用以下选项：

- **请求新职位**
- **请求更改职位详细信息**

您还可以将这些选项配置为经过可选的审核与审批工作流。

### <a name="summary"></a>摘要

**摘要** 部分中的信息取决于人力资源在 **人力资源参数** 中选择的选项。 在 **人力资源参数** 页面的 **经理自助服务** 选项卡上，您可以配置显示即将过期的记录和空缺职位的选项。 启用这些选项将确定经理可以在 **摘要** 部分看到的信息。

您可以为经理配置以下磁贴：

- **我的团队的过期证书**
- **我的团队的即将过期的标识号**
- **我的团队的即将过期的检查**
- **我的团队的即将过期的筛选**
- **我的团队的即将过期的测试**
- **直接和/或间接下属空缺职位**
- **我的团队的待处理休息请求**
- **团队技能评估**
- **技能差距分析**
- **团队绩效日记帐**
- **团队绩效目标**
- **团队绩效审核**
- **正离职的工作人员**

您可以配置以下选项，让经理可以代表他们的直接下属进行更改或添加休假请求：

- 证书
- 课程
- 借出物品
- 技能
- 休假和缺勤请求

### <a name="my-team-information"></a>我的团队信息

**我的团队** 让经理可以查看和更新直接和间接下属。 要访问间接下属，请选择具有直接下属的员工，然后在磁贴上选择 **查看团队**。 所有相同的选项都会作为直接下属应用于间接下属。 

#### <a name="summary-tab"></a>“摘要”选项卡

**摘要** 选项卡可让您快速查看您的直接下属。 如果直接下属也有向他们报告工作的工作人员，卡会在上面部分显示直接下属的数量，以及 **查看团队** 按钮。 每个磁贴上方的选项将应用于所选员工。 例如，如果您要代表员工输入休假请求，请选择该员工，然后选择 **请求休息**。 

如果您在选择员工后选择 **详细信息** 按钮，将显示以下选项：

- **证书**
- **薪酬**
- **课程**
- **审查**
- **休假**
- **借出物品**
- **绩效目标**
- **已登记的课程**
- **技能**
- **发送反馈**

根据组织的设置，您可以进行更改或只能查看。

#### <a name="position-tab"></a>“职位”选项卡

**职位** 选项卡在员工的主要职位中提供员工的摘要视图。 姓名、职位名称和部门显示在每个磁贴的标题区域内。 此磁贴中有：

- **受聘日期** - 从 **工作人员** 页的工作人员摘要部分显示。
- **工作年限** - 根据员工的雇用开始日期计算。
- **先前职位数** - 根据职位历史记录，选择此数字将打开所有先前曾担任职位的详细视图。
- **出生日期** - 员工出生日期的月和日。

您可以查看直接和间接下属的职位数据。

#### <a name="compensation-tab"></a>“薪酬”选项卡

**薪酬** 选项卡显示员工的年度薪水。 公司标识符在薪水金额下显示。 如果某个员工有多个工作并且从多个法人那里领工资，该员工将有多个薪酬计划。 若要在不切换公司的情况下跨法人查看所有薪酬计划，您必须在 **Human Resources > 共享参数 > 高级访问 > 启用跨公司薪酬** 下启用跨薪酬。

要查看薪酬历史记录，请选择 **薪水金额** 打开 **详细信息** 页。 仅当前和历史固定和可变薪酬记录显示在 **薪酬** 页中。 如果某个员工有多个工作，您可以在公司之间切换以查看每个公司的薪酬历史记录，或在 **Human Resources 共享参数** 中启用跨公司薪酬以查看所有薪酬计划。

您可以查看直接和间接下属的薪酬。

#### <a name="leave-and-absence-tab"></a>“休假和缺勤”选项卡

**休假和缺勤** 选项卡显示有活动的员工的最高余额。 要执行操作或查看活动的完整列表，请选择 **详细信息**，然后选择 **休息时间**。 在 **休息时间** 页中，您可以查看余额、请求、批准的休息时间和预测余额，以帮助员工更好地管理时间。 根据组织的设置，您还可以为直接和间接下属请求休息时间。

#### <a name="performance-goals-tab"></a>“绩效目标”选项卡

**绩效目标** 选项卡按状态汇总绩效目标。 选择一个状态编号或从 **详细信息** 选择绩效目标可以查看员工的所有目标。 经理和员工可以在目标的持续时间内根据需要更新目标。

经理可以通过 **我的团队** 的 **摘要** 部分中的 **团队绩效目标** 磁贴查看其团队的所有目标。

#### <a name="reviews-tab"></a>“审核”选项卡

**审核** 选项卡汇总员工所有的每个状态的审核：**进行中**、**审核就绪** 和 **最终审核**。 要访问员工的审核，请选择 **详细信息** 按钮，然后选择要协作进行的审核。 根据审核在工作流程中的位置，您可以查看审核是否可以更新。 

您可以通过 **我的团队** 的 **摘要** 部分中的 **团队绩效审核** 磁贴查看您团队的所有审核。
