[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文介绍 Dynamics 365 Human Resources 的技能级别类型选项集。

物理名称：mshr_hrmskillleveltype

此枚举提供对分配给应聘者技能的级别进行分类的值选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 实际值 | 实际技能级别。 |
| 200000001 | 目标 | 技能级别的目标值。 |
