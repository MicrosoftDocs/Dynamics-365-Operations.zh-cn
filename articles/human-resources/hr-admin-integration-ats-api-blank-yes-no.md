# <a name="blank-yes-no"></a>空白/是/否

本主题介绍 Dynamics 365 Human Resources 的“空白/是/否”选项集。

物理名称：mshr_hrmblankyesno

此枚举为同时还允许空白的“是/否”属性提供选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 空白 | 未选择值。 |
| 200000001 | 是 | 是。 |
| 200000002 | 不 | 编号 |
