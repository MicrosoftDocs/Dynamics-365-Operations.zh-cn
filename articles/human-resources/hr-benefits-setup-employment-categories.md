# <a name="create-employment-categories"></a>创建雇用类别

在 Dynamics 365 Human Resources 中设置雇用类别以匹配您的工作人员。

1. 在 **福利管理** 工作区中，在 **设置** 下，选择 **雇用类别**。

2. 选择 **新建**。

3. 为 **工作人员类型**、**雇用类别** 和 **描述** 指定值。

4. 选择 **保存**。
