# <a name="completion-status"></a>完成状态

本主题介绍 Dynamics 365 Human Resources 的完成状态选项集。

物理名称：mshr_hcmcompletionstatus

此枚举为应聘者筛选提供状态值选项集。 

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | 未完成 | 应聘者尚未完成筛选。 |
| 200000001 | 通过 | 应聘者已通过筛选。 |
| 200000002 | 失败 | 应聘者未通过筛选。 |
