# <a name="marital-status"></a>婚姻状况


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本主题介绍 Dynamics 365 Human Resources 的婚姻状况选项集。

物理名称：mshr_dirpersonmaritalstatus

此枚举为人员记录的婚姻状况提供选项集。

| 值 | 标签 | 说明 |
| --- | --- | --- |
| 200000000 | None | 未选择值。
| 200000001 | 未婚 | 未婚。 |
| 200000002 | 已婚 | 已婚。 |
| 200000003 | 离异 | 离异。 |
| 200000004 | 丧偶 | 丧偶。 |
