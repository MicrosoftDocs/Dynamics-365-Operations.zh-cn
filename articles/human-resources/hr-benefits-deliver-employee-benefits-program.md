# <a name="deliver-employee-benefits-program"></a>交付员工福利计划

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

本文将向您展示如何创建在创建新福利时需要使用的福利元素。 创建此任务的演示数据公司是 USMF。 此任务专门面向薪资和福利经理。


## <a name="create-benefit-elements"></a>创建福利元素
1. 从福利列表页开始此任务。 通过打开该页或搜索福利列表页，开始此任务。
2. 单击“新建”。
3. 在“类型”字段中，输入您创建的福利类型的名称。
4. 在“描述”字段中，键入一个值。
5. 在“同时注册”字段中，选择一个选项。
    * 为限制员工的登记多个医疗计划的可能，每一种类型选择一个登记。  
6. 在“工资类别”字段中，选择一个选项。
7. 单击“计划”选项卡。
8. 单击“新建”。
9. 在“计划”字段中，输入一个值。
10. 在“描述”字段中，键入一个值。
11. 在“类型”字段中，单击下拉按钮以打开查找。
12. 在列表中，找到并选择所需记录。
13. 在列表中，单击所选行中的链接。
14. 在“工资影响”字段中，选择一个选项。
15. 单击“选项”选项卡。
16. 单击“新建”。
17. 在“选项”字段中，输入一个值。
18. 在“描述”字段中，键入一个值。

## <a name="create-a-benefit"></a>创建福利
1. 关闭该页面。
2. 转到“人力资源”>“福利”>“福利”。
3. 单击“新建”，以打开对话框。
4. 在“计划”字段中，单击下拉按钮以打开查找。
5. 在列表中，找到并选择所需记录。
6. 在列表中，单击所选行中的链接。
7. 在“选项”字段中，单击下拉按钮以打开查找。
8. 在列表中，找到并选择所需记录。
9. 在列表中，单击所选行中的链接。
10. 在"有效日期"字段中，输入日期和时间。
11. 单击“创建福利”。
12. 切换“支付详细信息”部分的扩展。
13. 在“频率”字段中，单击下拉按钮以打开查找。
14. 在列表中，找到并选择所需记录。
15. 在列表中，单击所选行中的链接。
16. 在“基础”字段中，选择一个选项。
17. 在“金额或费率”字段中，输入一个数值。
