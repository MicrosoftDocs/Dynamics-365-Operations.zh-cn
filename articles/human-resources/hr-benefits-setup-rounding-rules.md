# <a name="set-up-rounding-rules"></a>设置舍入规则

您可以在 Dynamics 365 Human Resources 中设置舍入规则来满足组织的需求。

1. 在 **福利管理** 工作区中，在 **设置** 下，选择 **舍入规则**。

2. 选择 **新建**。

3. 为 **舍入到最接近的值**、**场景** 和 **舍入形式** 指定值。

4. 选择 **保存**。
