# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Dynamics 365 Commerce 站点的浏览器要求

[!include [banner](includes/banner.md)]

本文介绍 Microsoft Dynamics 365 Commerce 站点的浏览器要求。

以下 Web 浏览器对 Dynamics 365 Commerce 站点的支持度最高。 要查找每个 Web 浏览器的最新版本，请转至软件制造商的网站。

- [Microsoft Edge](https://www.microsoft.com/edge)（直到当前版本减 1）
- [Mozilla Firefox](https://www.mozilla.org/firefox/new)（直到当前版本减 1）
- [Google Chrome](https://www.google.com/chrome)（直到当前版本减 1）
- [Apple Safari](https://support.apple.com/downloads/safari)（直到当前版本减 1）

> [!NOTE]
> Internet Explorer 不受支持。 如果您使用 Internet Explorer，有些功能可能无法正常工作，用户体验可能不合标准。
