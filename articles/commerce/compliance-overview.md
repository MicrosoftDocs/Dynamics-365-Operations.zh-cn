# <a name="compliance-overview"></a>合规概述


[!include [banner](includes/banner.md)]

本主题概述了随 Microsoft Dynamics 365 Commerce 提供的帮助您确保公司站点合规性的合规指南。 合规性是所有企业的重要因素，因为它可以帮助其用户群与站点内容建立联系。 通过保持合规性，您还可以帮助公司避免昂贵的法律诉讼或高额罚款。

此合规性文档包括对使用 Dynamics 365 Commerce 时应考虑的区域的审查，以帮助您满足公司的合规要求。

您负责审查公司的合规要求，并负责创作和建立符合这些要求的标准的站点。 以下主题提供了一些示例，说明在使用 Commerce 创作工具时如何考虑合规性。

要了解有关 Microsoft 使用的基本原则的详细信息，请访问 [Microsoft 信任中心](https://www.microsoft.com/trust-center)。 在该站点上，您还可以获得有关合规区域的更多信息。
