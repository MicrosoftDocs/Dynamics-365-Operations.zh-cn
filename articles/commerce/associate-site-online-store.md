# <a name="associate-a-dynamics-365-commerce-site-with-an-online-channel"></a>将 Dynamics 365 Commerce 站点与在线渠道相关联

[!include [banner](includes/banner.md)]

此主题介绍如何将 Microsoft Dynamics 365 Commerce 站点绑定到一个或多个在线商店。 

在使用 Microsoft Dynamics Lifecycle Services (LCS) 门户预配 Dynamics 365 Commerce 电子商务环境之后，即可建立您的第一个电子商务网站。 创建初始站点时，将站点与之前创建的在线商店关联。 此步骤将站点绑定到在线渠道，让站点显示导航层次结构、产品、类别、价格、装运选项和您在在线商店中定义的其他所有项。

若要建立新站点和将在线商店与其关联，请在 LCS 中选择站点制作环境的链接。 然后，在站点制作环境的页面中，选择 **新建站点**。 在 **新站点** 对话框中，必须提供有关站点的一些基本信息。 有关必须提供的信息的完整说明，请参阅[创建新电子商务站点](create-ecommerce-site.md)。

创建站点之后，可通过选择 **产品** 选项卡将其与在线商店关联。应该可以查看已经分配给在线商店的产品的分类。 还可以使用页面左上角的下拉字段按类别访问产品。
