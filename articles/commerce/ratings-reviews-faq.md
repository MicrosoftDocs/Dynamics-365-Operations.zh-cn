# <a name="ratings-and-reviews-faq"></a>评分和评价常见问题解答

[!include [banner](includes/banner.md)]

本文提供对有关 Microsoft Dynamics 365 Commerce 评分和评论解决方案的常见问题的解答。

### <a name="how-do-i-hide-the-ratings-refiner-that-appears-on-search-results-and-category-pages-if-my-e-commerce-site-doesnt-use-the-ratings-and-reviews-solution"></a>如果我的电子商务站点不使用评分和评价解决方案，如何隐藏搜索结果和类别页面中显示的评分优化器？

有关如何在不使用评分和评价解决方案的电子商务渠道中的搜索结果和类别页面中隐藏评分优化器的说明，请参阅[未启用评分和评价解决方案时搜索结果和类别页面中显示评分优化器](troubleshoot/hide-ratings-refiner.md)。

### <a name="how-do-i-enforce-the-manual-publishing-of-ratings-and-reviews"></a>如何强制手动发布评级和评论？

有关如何启用和强制手动发布评分和评论以避免自动发布的说明，请参阅[启用审查者手动发布评分和评价](manual-publish-rating-reviews.md)。
