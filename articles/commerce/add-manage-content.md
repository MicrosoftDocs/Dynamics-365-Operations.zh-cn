# <a name="ways-to-add-content"></a>添加内容的方法

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

本主题介绍如何在 Microsoft Dynamics 365 Commerce 站点中添加和管理内容。

## <a name="overview"></a>概览

可通过多种方法更改站点的外观和内容。 非开发人员可以实施这些更改中的许多，具体取决于所需自定义级别。 例如，不必编写代码即可生成模板，选择主题和选择并配置模板。 相反，需要开发技能才能新建主题或模块，因为必须使用电子商务软件开发套件 (SDK) 和 Microsoft Dynamics Lifecycle Services (LCS) 部署工作流。

以下主题提供了有关如何添加和管理站点内容的详细信息。 重点介绍了无需开发人员的站点区域。 按照要求，指出了需要 SDK 工作的任务。

- 若要更改现有站点页中的文本、图像或视频，请参阅[使用模块](work-with-modules.md)。
- 若要帮助确保 Web 内容作者的极简单品牌创作体验，请参阅[使用模板](work-with-templates.md)。
- 若要重新安排站点页中的分区，请参阅[使用布局](work-with-layouts.md)。
- 若要更改站点页的字体、颜色和一般外观，请参阅[选择站点主题](select-site-theme.md)。
