# <a name="create-new-address-book"></a>创建新的通讯簿


[!include [banner](includes/banner.md)]

此主题介绍如何在 Microsoft Dynamics 365 Commerce 中创建新通讯簿。

## <a name="overview"></a>概览

出于各种原因，会在 Commerce 中使用通讯簿，包括存储渠道的客户列表和员工列表。 通讯簿可以用于单个渠道，也可以在渠道之间共享。

## <a name="create-a-new-address-book"></a>创建新的通讯簿

要创建新通讯簿，请执行以下步骤。
 
1. 在导航窗格中，转到 **模块 \> 渠道设置 \> 通讯簿**。
1. 在操作栏上，选择 **新建**。
1. 输入名称和描述信息。
1. 在操作栏上，选择 **保存**。
