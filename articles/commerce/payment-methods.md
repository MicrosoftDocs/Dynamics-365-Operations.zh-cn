# <a name="payment-methods"></a><span data-ttu-id="615ae-104">付款方式</span><span class="sxs-lookup"><span data-stu-id="615ae-104">Payment methods</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="615ae-105">设置系统时，必须配置零售商接受的每种付款类型。</span><span class="sxs-lookup"><span data-stu-id="615ae-105">Each payment type that a retailer accepts must be configured when the system is set up.</span></span> <span data-ttu-id="615ae-106">本文介绍您可设置的付款类型并介绍设置它们的流程。</span><span class="sxs-lookup"><span data-stu-id="615ae-106">This article describes the payment types that you can set up and describes the process for setting them up.</span></span>

<span data-ttu-id="615ae-107">零售商可以接受各种类型的付款以交换其销售的产品和服务。</span><span class="sxs-lookup"><span data-stu-id="615ae-107">Retailers can accept various types of payment in exchange for the products and services that they sell.</span></span> <span data-ttu-id="615ae-108">尽管现金付款是最常用的形式，零售商也可以接收支票的形式、卡、付款，凭证等形式的付款。</span><span class="sxs-lookup"><span data-stu-id="615ae-108">Although cash is the most common form of payment, retailers can also receive payment in the form of checks, cards, vouchers, and so on.</span></span> <span data-ttu-id="615ae-109">设置系统时，用于接收的每种付款类型，零售商接受必须配置在 Dynamics 365 Commerce 中时。</span><span class="sxs-lookup"><span data-stu-id="615ae-109">Each payment type that the retailer accepts must be configured in Dynamics 365 Commerce when the system is set up.</span></span> <span data-ttu-id="615ae-110">下表描述设置每种付款类型：</span><span class="sxs-lookup"><span data-stu-id="615ae-110">The following list describes each payment type that can be set up:</span></span>

- <span data-ttu-id="615ae-111">**现金**– 以实际货币（例如纸币和硬币）形式。</span><span class="sxs-lookup"><span data-stu-id="615ae-111">**Cash** – Money in the physical form of currency, such as banknotes and coins.</span></span> <span data-ttu-id="615ae-112">此货币可以是公司币种或商店的本币。</span><span class="sxs-lookup"><span data-stu-id="615ae-112">This currency can be either the company currency or the store's local currency.</span></span>
- <span data-ttu-id="615ae-113">**支票**– 一种可转让票据，它指示从指定银行划走特定币种的特定金额进行付款。</span><span class="sxs-lookup"><span data-stu-id="615ae-113">**Check** – A negotiable instrument that instructs payment of a specific amount of a specific currency, and that is drawn on a specific bank.</span></span> <span data-ttu-id="615ae-114">除非指定了其他有效期，否则支票通常是永久有效或在签发之后的六个月内有效。</span><span class="sxs-lookup"><span data-stu-id="615ae-114">A check is typically valid either indefinitely or for six months after the date of issue, unless another period of validity is specified.</span></span> <span data-ttu-id="615ae-115">此期间因签发支票的银行而异。</span><span class="sxs-lookup"><span data-stu-id="615ae-115">This period varies, depending on the bank that the check is drawn on.</span></span> <span data-ttu-id="615ae-116">支票有各种类型，如记名支票、现金支票、不记名支票和转帐支票。</span><span class="sxs-lookup"><span data-stu-id="615ae-116">There are various kinds of checks, such as order checks, counter checks, bearer checks, and account payee checks.</span></span> <span data-ttu-id="615ae-117">您可以设置支票为每个商店的付款方式。</span><span class="sxs-lookup"><span data-stu-id="615ae-117">You can set up checks as a payment method for each store.</span></span> <span data-ttu-id="615ae-118">可接受采用在公司级别或商店级别定义的货币的支票。</span><span class="sxs-lookup"><span data-stu-id="615ae-118">Checks can be accepted in the currency that is defined at either the company level or the store level.</span></span> <span data-ttu-id="615ae-119">您可以接受支票作为商店的支付方式之前，必须设置支票作为付款方式。</span><span class="sxs-lookup"><span data-stu-id="615ae-119">You must set up checks as a payment method before you can accept a check as payment in a store.</span></span>
- <span data-ttu-id="615ae-120">**币种**– 除公司默认本币之外的最主要付款方式。</span><span class="sxs-lookup"><span data-stu-id="615ae-120">**Currency** – The primary form of payment other than the company's default currency.</span></span> <span data-ttu-id="615ae-121">硬币和纸币都是币种的形式。</span><span class="sxs-lookup"><span data-stu-id="615ae-121">Coins and paper money are both forms of currency.</span></span> <span data-ttu-id="615ae-122">币种付款方式表示使用的所有币种。</span><span class="sxs-lookup"><span data-stu-id="615ae-122">The currency payment method represents all currency that is used.</span></span> <span data-ttu-id="615ae-123">在使用此支付方式之前，必须设置币种并为币种指定兑换信息。</span><span class="sxs-lookup"><span data-stu-id="615ae-123">Before you can use this payment method, you must set up currencies and specify exchange information for the currencies.</span></span>
- <span data-ttu-id="615ae-124">**卡** – 使用的所有类型的卡，如借记卡和信用卡。</span><span class="sxs-lookup"><span data-stu-id="615ae-124">**Card** – All the kinds of cards that are used, such as debit cards and credit cards.</span></span> <span data-ttu-id="615ae-125">最好在组织级别设置一种卡付款方式以表示每种类型的卡。</span><span class="sxs-lookup"><span data-stu-id="615ae-125">It's a good idea to set up one card payment method at the organization level, to represent every kind of card.</span></span> <span data-ttu-id="615ae-126">然后，在商店级别，可以为每个卡或应使用相同设置处理的每组卡设置一种支付方式。</span><span class="sxs-lookup"><span data-stu-id="615ae-126">Then, at the store level, set up a payment method for each card or set of cards that is processed by using the same settings.</span></span> <span data-ttu-id="615ae-127">您必须设置可用于市场上的商用卡，如借记卡和信用卡，在此情况下，您可以接受在商店中卡作为支付方式之前。</span><span class="sxs-lookup"><span data-stu-id="615ae-127">You must set up the manufacturer cards that are available in the market, such as debit cards and credit cards, before you can accept the cards as payment in a store.</span></span>

<span data-ttu-id="615ae-143">若要设置付款方式，必须完成以下任务。</span><span class="sxs-lookup"><span data-stu-id="615ae-143">To set up payment methods, you must complete the following tasks.</span></span>
