# <a name="upload-files-other-than-images-and-videos"></a>上传图像和视频以外的文件

[!include [banner](includes/banner.md)]

本文介绍如何在 Microsoft Dynamics 365 Commerce 站点构建器中上传非图像和视频的文件。

Commerce 站点构建器的媒体库支持上传非图像或视频的二进制资产。 例如，可以上传 Microsoft Excel、Microsoft Word、Microsoft PowerPoint 或 PDF 文件。

支持以下文档类型：
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>上载文件

要将文件上传到 Commerce 站点构建器，请执行以下步骤。

1. 在左侧导航窗格中，选择 **媒体库**。
1. 在命令栏上，选择 **上传 \> 上传媒体项**。
1. 在文件资源管理器中，选择一个或多个文件，然后选择 **打开**。
1. 根据需要在 **上传媒体项** 对话框中输入标题、描述和关键字元数据。
1. 若要在上传后立即发布文件，请选中 **上传后发布媒体项** 复选框。
1. 选择 **确定**。
