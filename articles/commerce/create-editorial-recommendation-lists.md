## <a name="create-a-new-list"></a><span data-ttu-id="718ed-106">创建新列表</span><span class="sxs-lookup"><span data-stu-id="718ed-106">Create a new list</span></span>

<span data-ttu-id="718ed-107">若要创建精选产品建议列表，请执行以下步骤。</span><span class="sxs-lookup"><span data-stu-id="718ed-107">To create a curated product recommendation list, follow these steps.</span></span>
