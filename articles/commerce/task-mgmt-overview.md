# <a name="task-management-overview"></a><span data-ttu-id="f037d-103">任务管理概览</span><span class="sxs-lookup"><span data-stu-id="f037d-103">Task management overview</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="f037d-104">此主题概述 Microsoft Dynamics 365 Commerce 中适用于经理和工作人员的任务管理。</span><span class="sxs-lookup"><span data-stu-id="f037d-104">This topic provides an overview of task management for managers and workers in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="f037d-105">在零售环境中，始终很难确保正确的人在正确的时间执行任务。</span><span class="sxs-lookup"><span data-stu-id="f037d-105">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="f037d-106">零售商必须可以通知工作人员很快应该执行的任务，并提供相关业务上下文，以便正确、及时完成任务。</span><span class="sxs-lookup"><span data-stu-id="f037d-106">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="f037d-107">任务管理是 Dynamics 365 Commerce 中的一项生产力功能，用于供经理和工作人员创建任务列表，管理分配条件，跟踪任务状态，以及在 Commerce 后端与销售点 (POS) 应用程序之间集成这些操作。</span><span class="sxs-lookup"><span data-stu-id="f037d-107">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="f037d-108">总部人员可使用任务管理为零售商店创建任务列表和按商店或工作人员跟踪状态。</span><span class="sxs-lookup"><span data-stu-id="f037d-108">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="f037d-109">还可以创建重复执行任务（例如，“星期四晚上结算核对清单”）。</span><span class="sxs-lookup"><span data-stu-id="f037d-109">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="f037d-110">商店经理可在 POS 应用程序中使用任务管理将任务分配给工作人员个人，发送有关即将执行的任务或已到期任务的通知，更新任务状态，以及创建单一用途任务。</span><span class="sxs-lookup"><span data-stu-id="f037d-110">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="f037d-111">然后，工作人员可以在 POS 中查看通知，查看任务详细信息，以及更新任务状态。</span><span class="sxs-lookup"><span data-stu-id="f037d-111">Workers can then see notifications, view task details, and update task status at the POS.</span></span>
