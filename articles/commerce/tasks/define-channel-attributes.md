# <a name="define-channel-attributes"></a>定义渠道属性

[!include [banner](../includes/banner.md)]

通过添加渠道与产品属性充实渠道。 在定义属性前，必须把该渠道添加到默认的组织层次结构中。


## <a name="define-channel-product-attribute"></a>定义渠道产品属性。
1. 转到“Retail 和 Commerce”>“渠道设置”>“渠道类别和产品属性”。
2. 在树形图中，选择“绝对零售\时尚\休斯敦”。
3. 单击“编辑”。
4. 在“类别层次结构”字段中，输入或选择一个值。
    * 导航层次结构用于内部渠道导航产品。  
5. 展开“属性组”部分。
6. 单击“添加”。
    * 属性组显示产品的详细信息，同时在使用产品筛选器时提供帮助。  
7. 在“名称”字段中，输入或选择一个值。
8. 单击“查看属性”。
9. 展开“属性”部分。
10. 单击“添加”。
11. 在列表中，标记所选的行。
12. 单击 AddBtn。
13. 单击“确定”。
14. 单击“保存”。
15. 关闭该页面。
16. 展开“产品”部分。
17. 单击“添加”。
18. 在列表中，标记所选的行。
19. 单击“添加”。
    * 同样可以添加多个产品。  
20. 单击“确定”。
21. 单击“属性”。
22. 展开“渠道媒介”部分。
23. 单击“编辑”。
24. 在“渠道”字段中，选择一个选项。
25. 选择“默认”复选框。
26. 单击“确定”。
27. 展开“渠道产品属性”部分。
28. 在“品牌”字段中，键入一个值。
    * 同样可以设置其他属性。  
29. 单击“保存”。
30. 关闭该页面。
31. 单击“保存”。
32. 单击“发布渠道更新”。
    * 数据发布后将与渠道表同步。 如果没有发布已修改数据，该数据与渠道表则不同步。  
33. 单击“确定”。
34. 单击“保存”。
