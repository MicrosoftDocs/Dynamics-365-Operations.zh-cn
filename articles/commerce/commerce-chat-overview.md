# <a name="commerce-chat-features-overview"></a>商业聊天功能概述

[!include [banner](includes/banner.md)]

通过将 Microsoft Dynamics 365 Customer Service 全渠道和 Power Virtual Agents 应用程序与 Dynamics 365 Commerce 电子商务网站集成，企业可以通过聊天功能为在线购物者提供支持。 本文概述了当 Customer Service 全渠道和 Power Virtual Agents 与 Dynamics 365 Commerce 电子商务网站集成时，在线购物者可以使用的商业聊天功能。

## <a name="omnichannel-for-customer-service"></a>Customer Service 全渠道

Customer Service 全渠道是一个强大的应用程序，可扩展 Dynamics 365 Customer Service 的功能。 它让组织能够通过实时聊天、语音和短消息服务 (SMS) 等渠道即时联系并与客户互动。 有关 Customer Service 全渠道的更多信息，请参阅 [Customer Service 全渠道简介](/dynamics365/customer-service/introduction-omnichannel)。 有关如何将 Customer Service 全渠道与 Dynamics 365 Commerce 电子商务网站集成的信息，请参阅 [Customer Service 全渠道商业聊天模块](commerce-chat-module.md)。

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents 让您可以针对一系列请求创建强大的 AI 驱动的聊天机器人。 有关 Power Virtual Agents 的详细信息，请参阅 [Power Virtual Agents 概述](/power-virtual-agents/fundamentals-what-is-power-virtual-agents)。 有关如何将 Power Virtual Agents 与 Dynamics 365 Commerce 电子商务网站集成的信息，请参阅 [Microsoft Power Virtual Agents 商业聊天](chat-module-pva.md)。

> [!NOTE]
> Power Virtual Agents 已包含在 Customer Service 全渠道中。 如果您同时需要虚拟代理（聊天机器人）和实时代理功能，请使用 Customer Service 全渠道。
