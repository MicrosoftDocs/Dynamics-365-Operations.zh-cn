# <a name="online-store-overview"></a>在线商店概览
[!include [banner](includes/banner.md)]
[!include [banner](includes/preview-banner.md)]

本主题介绍在线商店的概念，说明在线商店在 Microsoft Dynamics 365 Commerce 中如何使用。 另外还提供了有关在线商店的更多信息以及有关如何设置在线商店的信息的链接。

您必须至少先设置一个在线商店，才能在 Dynamics 365 Commerce 中建立站点。 在 Dynamics 365 Commerce 中，使用在线商店建立产品、定价、语言、付款方式、交货方式、履行中心，以及其他应为客户提供的在线体验的各个方面。

只需设置一个在线商店，便可以开始使用 Dynamics 365 Commerce。 但是，一个 Dynamics 365 Commerce 站点可以为多个在线商店提供在线体验。 例如，如果设置了多个在线商店来支持不同的地理区域，则可以使用一组页面来提供每个商店定义的独特体验。 有关如何配置站点以支持多个在线商店的详细信息，请参阅[将在线站点与渠道关联](associate-site-online-store.md)。

设置在线商店后，可以将其与将用作您的在线店面的 Dynamics 365 Commerce 站点关联。 有关在线商店以及如何设置的详细信息，请参阅[设置在线商店](https://docs.microsoft.com/dynamics365/unified-operations/retail/online-stores)。
