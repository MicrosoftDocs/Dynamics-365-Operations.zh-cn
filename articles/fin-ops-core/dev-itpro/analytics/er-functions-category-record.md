# <a name="list-of-er-functions-in-the-record-category"></a>记录类别的 ER 函数列表

[!include [banner](../includes/banner.md)]

电子申报 (ER) 记录函数可用于从 *容器（记录）* 数据类型的数据源提取信息，并对数据源执行操作。 本主题提供这些函数的概要。

## <a name="list-of-supported-functions"></a>支持的函数列表

| 职能 | 说明 |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | 此函数返回与指定记录列表或记录具有相同结构的空 *容器（记录）* 值。 |
| [EmptyRecord](er-functions-record-emptyrecord.md) | 此函数返回与指定记录列表或记录具有相同结构的空 *容器（记录）* 值。 |
