# <a name="len-er-function"></a>LEN ER 函数

[!include [banner](../includes/banner.md)]

`LEN` 函数作为 *整数* 值在指定的字符串中返回字符数量。

## <a name="syntax"></a>语法

```vb
LEN (text)
```

## <a name="arguments"></a>参数

`text`：*字符串*

指定文本的 *字符串* 值。

## <a name="return-values"></a>返回值

*整数*

生成的数字值。

## <a name="example"></a>示例

`LEN ("Sample")` 将返回 **6**。
