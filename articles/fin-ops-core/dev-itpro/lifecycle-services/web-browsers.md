# <a name="supported-web-browsers-for-lifecycle-services"></a>Lifecycle Services 支持的 Web 浏览器 

[!include[banner](../includes/banner.md)]

用户可以使用这些热门浏览器的最新版本访问 Microsoft Dynamics Lifecyle Services (LCS)： 

- Microsoft Edge（推荐：[基于 Chromium 的 Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf)）
- Google Chrome
- Apple Safari

要查找每个 Web 浏览器的最新版本，请转至软件制造商的网站。 为了获得最佳性能和最佳体验，我们建议您使用最新版本的现代浏览器，尤其是 Microsoft Edge。
