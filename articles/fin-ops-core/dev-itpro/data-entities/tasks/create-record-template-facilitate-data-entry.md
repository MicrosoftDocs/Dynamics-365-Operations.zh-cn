[!include [banner](../../includes/banner.md)]

<span data-ttu-id="eeeb8-104">此主题演示如何创建报表模板，以便不必明确为每条记录输入常用字段值。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-104">This topic demonstrates how to create a record template so that field values that are used often do not have to be entered explicitly for each new record.</span></span> <span data-ttu-id="eeeb8-105">在此过程中，您将为应添加到固定资产的新笔记本电脑创建一条新记录。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-105">In this procedure, you'll create a new record for new laptops that should be added to your fixed assets.</span></span> <span data-ttu-id="eeeb8-106">本流程使用了 USMF 示例公司。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-106">This procedure uses the USMF sample company.</span></span>

1. <span data-ttu-id="eeeb8-107">在导航窗格中，转到 **模块 > 固定资产 > 固定资产 > 固定资产**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-107">In the navigation pane, go to **Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="eeeb8-108">选择 **新建**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-108">Select **New**.</span></span>
3. <span data-ttu-id="eeeb8-109">在 **固定资产组** 字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-109">In the **Fixed asset group** field, enter or select a value.</span></span>
4. <span data-ttu-id="eeeb8-110">在 **名称** 字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-110">In the **Name** field, type a value.</span></span> <span data-ttu-id="eeeb8-111">例如，输入 **公司潜在客户笔记本电脑**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-111">For example, enter **Corporate lead laptop**.</span></span>  
5. <span data-ttu-id="eeeb8-112">在 **搜索名称** 字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-112">In the **Search name** field, type a value.</span></span> <span data-ttu-id="eeeb8-113">例如，输入 **笔记本电脑**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-113">For example, enter **laptop**.</span></span>  
6. <span data-ttu-id="eeeb8-114">展开 **技术信息** 部分。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-114">Expand the **Technical information** section.</span></span>

9. <span data-ttu-id="eeeb8-117">选择 **记录信息**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-117">Select **Record info**.</span></span>
10. <span data-ttu-id="eeeb8-118">选择 **用户模板**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-118">Select **User template**.</span></span>
11. <span data-ttu-id="eeeb8-119">在 **名称** 字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-119">In the **Name** field, type a value.</span></span>
12. <span data-ttu-id="eeeb8-120">在 **描述** 字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-120">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="eeeb8-121">选择 **确定**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-121">Select **OK**.</span></span>
14. <span data-ttu-id="eeeb8-122">选择 **关闭**。</span><span class="sxs-lookup"><span data-stu-id="eeeb8-122">Select **Close**.</span></span>
