# <a name="use-record-template-to-create-a-new-record"></a>使用记录模板创建新记录

[!include [banner](../../includes/banner.md)]

此过程显示如何使用以前定义的记录模板创建新记录。 要完成此过程，您必须首先完成“创建记录模板以简化数据录入”过程。

本过程使用 USMF 公司。

1. 在**导航窗格**中，转到**模块 > 固定资产 > 固定资产 > 固定资产**。
2. 单击**新建**。 系统将提示您选择模板。 选择与您的企业需求对应的模板。  
3. 在列表中，找到并选择所需记录。
4. 单击 **确定**。
