## <a name="cds-exchange-rates-to-msdyn_currencyexchangerates"></a>CDS 汇率到 msdyn_currencyexchangerates

此模板在 Finance and Operations 应用与 Common Data Service 之间同步数据。

Finance and Operations 字段 | 映射类型 | 其他 Dynamics 365 字段 | 默认值
---|---|---|---
EXCHANGERATETYPENAME | >> | msdyn_exchangeratetypename | 
FROMCURRENCYCODE | >> | msdyn_fromcurrencycode | 
TOCURRENCYCODE | >> | msdyn_tocurrencycode | 
RATE | >> | msdyn_exchangerate | 
VALIDFROM | >> | msdyn_validfrom | 
VALIDTO | >> | msdyn_validto |
