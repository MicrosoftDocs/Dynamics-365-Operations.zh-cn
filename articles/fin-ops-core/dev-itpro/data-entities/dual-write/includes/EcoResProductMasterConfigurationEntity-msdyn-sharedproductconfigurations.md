## <a name="product-master-configurations-to-msdyn_sharedproductconfigurations"></a>基础产品配置到 msdyn_sharedproductconfigurations

此模板在 Finance and Operations 应用与 Common Data Service 之间同步数据。

Finance and Operations 字段 | 映射类型 | 其他 Dynamics 365 字段 | 默认值
---|---|---|---
CONTAINERUNITSYMBOL | >> | msdyn_containerunit.msdyn_symbol | 
PRODUCTCONFIGURATIONID | >> | msdyn_productconfiguration.msdyn_productconfiguration | 
PRODUCTMASTERNUMBER | >> | msdyn_globalproduct.msdyn_productnumber | 
REPLENISHMENTWEIGHT | >> | msdyn_replenishmentweight | 
DISPLAYSEQUENCENUMBER | >> | msdyn_displaysequencenumber |
