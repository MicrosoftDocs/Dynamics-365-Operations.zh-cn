# <a name="configure-address-books"></a>配置通讯簿

[!include [banner](../../includes/banner.md)]

使用该过程以及您在规划中作出的决定、全球通讯簿的配置和其他通讯簿文章，为您的组织设置其他通讯簿。

创建此任务的演示数据公司是 USMF。


## <a name="configure-address-books"></a>配置通讯簿
1. 在 **导航窗格** 中，转到 **模块 > 组织管理 > 全球通讯簿 > 通讯簿**。
2. 单击 **新建**。
3. 在 **名称** 字段中，键入一个值。
4. 在 **描述** 字段中，键入一个值。
5. 单击 **保存**。
6. 在列表中，找到并选择所需记录。
7. 单击箭头以添加选定的可用团队到通讯簿。
8. 单击 **保存**。
