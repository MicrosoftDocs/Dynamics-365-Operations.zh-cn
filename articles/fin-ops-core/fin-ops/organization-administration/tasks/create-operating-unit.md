# <a name="create-an-operating-unit"></a><span data-ttu-id="e800b-103">创建运营单位</span><span class="sxs-lookup"><span data-stu-id="e800b-103">Create an operating unit</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="e800b-104">运营单位是用来拆分企业中的经济资源和运营流程的控件的组织。</span><span class="sxs-lookup"><span data-stu-id="e800b-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="e800b-105">运营单位中的人有义务最大化对稀有资源的使用、改进流程和记录他们的业绩。</span><span class="sxs-lookup"><span data-stu-id="e800b-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="e800b-106">运营单位的类型包括成本中心、业务单位、部门以及价值流。</span><span class="sxs-lookup"><span data-stu-id="e800b-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="e800b-107">使用以下过程创建运营单位。</span><span class="sxs-lookup"><span data-stu-id="e800b-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="e800b-108">创建此程序的演示数据公司是 USMF。</span><span class="sxs-lookup"><span data-stu-id="e800b-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="e800b-109">转到**导航窗格 > 模块 > 组织管理 > 组织 > 运营单位**。</span><span class="sxs-lookup"><span data-stu-id="e800b-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="e800b-110">单击**新建**，以打开对话框。</span><span class="sxs-lookup"><span data-stu-id="e800b-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="e800b-111">在列表中，找到并选择所需记录。</span><span class="sxs-lookup"><span data-stu-id="e800b-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="e800b-112">选择您想要创建的运营单位。</span><span class="sxs-lookup"><span data-stu-id="e800b-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="e800b-113">在列表中，单击所选行中的链接。</span><span class="sxs-lookup"><span data-stu-id="e800b-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="e800b-114">在**名称**字段中，键入一个值。</span><span class="sxs-lookup"><span data-stu-id="e800b-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="e800b-115">如果需要，展开**常规**部分。</span><span class="sxs-lookup"><span data-stu-id="e800b-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="e800b-116">提供关于运营单位的一般信息，例如识别号码、DUNS 编号和管理器。</span><span class="sxs-lookup"><span data-stu-id="e800b-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="e800b-117">如果需要，展开**地址**部分。</span><span class="sxs-lookup"><span data-stu-id="e800b-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="e800b-118">输入街道名称和门牌号、邮政编码和城市等地址信息。</span><span class="sxs-lookup"><span data-stu-id="e800b-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="e800b-119">单击**添加**以输入新的地址记录，或单击“编辑”以修改现有地址记录。</span><span class="sxs-lookup"><span data-stu-id="e800b-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="e800b-120">如果需要，展开**联系信息**部分。</span><span class="sxs-lookup"><span data-stu-id="e800b-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="e800b-121">输入关于通信方法的信息，例如电子邮件地址、URL 和电话号码。</span><span class="sxs-lookup"><span data-stu-id="e800b-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="e800b-122">要输入新的通信记录，请单击“新建”。</span><span class="sxs-lookup"><span data-stu-id="e800b-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="e800b-123">要修改现有通信记录，请单击**更多选项 > 高级**。</span><span class="sxs-lookup"><span data-stu-id="e800b-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="e800b-124">（可选）根据需要更改**运营单位编号**。</span><span class="sxs-lookup"><span data-stu-id="e800b-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="e800b-125">请注意，此编号是相应**当事方**记录的唯一标识符，不能与其他任何运营单位相同。</span><span class="sxs-lookup"><span data-stu-id="e800b-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="e800b-126">选择**保存**。</span><span class="sxs-lookup"><span data-stu-id="e800b-126">Select **Save**.</span></span>
