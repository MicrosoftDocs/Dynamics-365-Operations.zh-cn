# <a name="set-a-users-preferred-time-zone"></a>设置用户的首选时区

[!include [banner](../../includes/banner.md)]

以下主题说明系统管理员角色的用户可以如何为用户设置时区。

1. 转到**导航窗格 > 模块 > 系统管理 > 用户 > 用户**。
2. 在列表中，找到并选择所需记录。
3. 选择**用户选项**。
4. 选择**首选项**选项卡。
5. 在**时区**字段中，从下拉列表中选择一个选项。
6. 选择**保存**。
