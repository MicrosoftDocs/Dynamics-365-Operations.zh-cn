# <a name="enter-applicant-and-application-data-manually"></a>手动输入申请人和申请表数据

[!include [banner](../../includes/banner.md)]

此过程显示如何手动维护申请人及其申请表的信息。 您可以输入和维护个人信息、面试日期和时间、推荐人、能力和申请人的住宿请求。 您还可以更新申请人的工作申请状态，以及写信或发送电子邮件与申请人沟通。 在您创建申请人记录时，在全球通讯簿中创建该申请人的人员记录。 创建此过程时使用的是 **USMF** 演示数据格式。

## <a name="create-a-new-applicant-record"></a>创建新的申请人记录

1. 转到 **人力资源 \> 招聘 \> 申请人 \> 申请人**。
2. 选择 **新建**。
3. 在 **名字** 字段中，输入一个值。
4. 在 **姓氏** 字段中，输入一个值。

    可以输入申请人的其他信息（如有）。 例如，此信息可包含申请人的最高学历、当前职务或前雇主。

5. 展开 **联系信息** 部分。
6. 选择 **添加**。
7. 在 **描述** 字段中，输入 **通信电子邮件**。
8. 在 **类型** 字段中，选择一个选项。
9. 在 **联系电话/地址** 字段中，输入一个值。

    此电子邮件地址将用于生成与申请人通信的电子邮件。

10. 选择 **添加**。
11. 在 **描述** 字段中，输入一个值。
12. 在 **联系电话/地址** 字段中，输入一个值。

    根据需要使用此字段输入有关申请人的其他个人信息。 例如，此信息可能包括申请人的出生日期、种族血统、性别或婚姻状况。

13. 在操作窗格上，选择 **能力**。

    您可以输入申请人的能力资料，包括其技能、专业经验、教育程度、测试或证书。 此信息可用于将申请人的技能映射到与贵公司数据中定义的工作相关的技能。

## <a name="create-an-application-for-the-applicant"></a>为申请人创建申请表。

1. 选择 **申请**。
2. 选择 **新建**。
3. 在 **招聘项目** 字段中，选择下拉箭头打开查找。

    通过选择招聘项目，您可以确保申请人与该招聘项目中包含的特定职位空缺相关联。

4. 在列表中，找到并选择所需记录。
5. 在列表中，选择所选择行中的链接。

    默认情况下，工作和部门基于所选的招聘项目。

6. 选择 **保存**。

    保存申请后，您可以将文件附加到该申请。 这些文件可能包括申请人的经验、奖励和求职信。
