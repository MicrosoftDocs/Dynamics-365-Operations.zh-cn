# <a name="enter-applicant-and-application-data-manually"></a>手动输入申请人和申请表数据

[!include [banner](../../includes/banner.md)]

此过程显示如何手动维护申请人及其申请表的信息。   您可以输入和维护个人信息、面试日期和时间、推荐人、能力和申请人的住宿请求。 您还可以更新申请人的工作申请状态，以及写信或发送电子邮件与申请人沟通。 在您创建申请人记录时，在全球通讯簿中创建该申请人的人员记录。       创建此程序的演示数据公司是 USMF。


## <a name="create-a-new-applicant-record"></a>创建新的申请人记录
1. 转到“人力资源”>“招聘”>“申请人”>“申请人”。
2. 单击“新建”。
3. 在“名字”字段中，键入一个值。
4. 在“姓氏”字段中，键入一个值。
    * 可以输入申请人的其他信息（如有）。 例如，可包含申请人的最高学历、当前职务或前雇主等信息。  
5. 切换“联系信息”部分的展开项。
6. 单击“添加”。
7. 在“描述”字段中，键入“通信电子邮件”。
8. 在“类型”字段中，选择一个选项。
9. 在“联系电话/地址”字段中，键入一个值。
    * 此电子邮件地址将用于生成与申请人通信的电子邮件。  
10. 单击“添加”。
11. 在“描述”字段中，键入一个值。
12. 在“联系电话/地址”字段中，键入一个值。
    * 申请人个人信息。  
    * 如果需要，可以输入申请人的其他个人信息。 例如，这可能包括出生日期、种族血统、性别或婚姻状况。  
13. 在操作窗格上，单击“能力”。
    * 您可以输入申请人的能力资料，包括其技能、专业经验、教育程度、测试或证书。  
    * 此信息可用于反应申请人是否具有与公司数据定义的工作相关的技能。   

## <a name="create-an-application-for-the-applicant"></a>为申请人创建申请表。
1. 单击“申请表”。
2. 单击“新建”。
3. 在“招聘项目”字段中，单击下拉按钮以打开查找。
    * 选择招聘项目，将申请人与该招聘项目中包含的特定开放职位关联起来。  
4. 在列表中，找到并选择所需记录。
5. 在列表中，单击所选行中的链接。
    * 默认情况下，工作和部门基于所选的招聘项目。  
6. 单击“保存”。
    * 在保存申请表后，您可以添加文档附件，包括申请人的经验、所获奖项和求职信。
