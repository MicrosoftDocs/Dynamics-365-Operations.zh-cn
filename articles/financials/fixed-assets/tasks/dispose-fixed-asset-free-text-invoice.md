<span data-ttu-id="1099c-104">此过程显示如何使用普通发票处置固定资产。</span><span class="sxs-lookup"><span data-stu-id="1099c-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="1099c-105">转到“应收账款”>“发票”>“所有普通发票”。</span><span class="sxs-lookup"><span data-stu-id="1099c-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="1099c-106">单击“新建”。</span><span class="sxs-lookup"><span data-stu-id="1099c-106">Click New.</span></span>
3. <span data-ttu-id="1099c-107">在“客户帐户”字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="1099c-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="1099c-108">验证默认发票日期并编辑（如果适用）。</span><span class="sxs-lookup"><span data-stu-id="1099c-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="1099c-109">验证其余默认标题字段（如“币种”）并编辑（如果适用）。</span><span class="sxs-lookup"><span data-stu-id="1099c-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="1099c-110">在发票行中输入描述。</span><span class="sxs-lookup"><span data-stu-id="1099c-110">Enter a Description into the invoice line.</span></span>

10. <span data-ttu-id="1099c-114">单击“行详细信息”。</span><span class="sxs-lookup"><span data-stu-id="1099c-114">Click Line details.</span></span>  
11. <span data-ttu-id="1099c-115">选择要出售的固定资产编号。</span><span class="sxs-lookup"><span data-stu-id="1099c-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="1099c-116">单击“过帐”。</span><span class="sxs-lookup"><span data-stu-id="1099c-116">Click Post.</span></span>
