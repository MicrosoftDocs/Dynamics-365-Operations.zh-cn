# <a name="propose-fixed-asset-acquisitions"></a>建议固定资产购置

[!include [task guide banner](../../includes/task-guide-banner.md)]

此过程显示如何使用固定资产日记帐中的购置方案购置固定资产。 它为 USMF 法人实体使用会计角色和演示数据。

1. 转到固定资产>流水输入项>固定资产流水。
2. 单击“新建”。
3. 在“名称”字段中，输入或选择一个值。
4. 单击“行”。
5. 单击“方案”。
6. 单击“购置方案”。
7. 单击“筛选器”。
8. 单击“重置”以清除先前值。
9. 选择固定资产编号行。
10. 在“标准”字段中，输入或选择一个值。
    * 为您想要以此方案购置的固定资产设置其他条件。  
11. 单击“确定”。
12. 单击“确定”。
    * 验证已创建的交易记录行。  
    * 只有在帐簿中设置了购置日期和购置价格的固定资产才会包含在购置方案中。  
13. 单击“帐簿”选项卡。
14. 单击“过帐”。
