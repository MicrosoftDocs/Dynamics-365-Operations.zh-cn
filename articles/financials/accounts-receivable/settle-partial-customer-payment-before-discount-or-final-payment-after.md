# <a name="settle-a-partial-customer-payment-before-the-discount-date-with-a-final-payment-after-the-discount-date"></a>在使用最后付款的折扣日期之前的折扣日期之后结算部分客户付款

[!include [banner](../includes/banner.md)]

文本讨论结算客户发票付款的影响。 此情况关注对子分类帐而不是总帐的影响。

Fabrikam 向客户 4027 销售货物， 如果发票在 14 天内支付，则 Fabrikam 提供 1% 的现金折扣。 发票必须在 30 天时支付。 Fabrikam 还为部分付款提供现金折扣。 结算参数位于**应付账款参数**页上。

## <a name="invoice"></a>开票
6 月 25 日，Arnie 为客户 4027 输入并过帐 1,000.00 的发票。 Arnie 可以使用**客户**页上的**交易记录**按钮查看此发票。

| 凭证   | 交易记录类型 | 日期      | 开票 | 交易币种借方金额 | 交易币种贷方金额 | 余额  | 货币 |
|-----------|------------------|-----------|---------|--------------------------------------|---------------------------------------|----------|----------|
| FTI-10020 | 开票          | 6/25/2015 | 10020   | 1,000.00                             |                                       | 1,000.00 | 美元      |

## <a name="partial-payment-before-the-cash-discount-date"></a>现金折扣日期之前的部分付款
7 月 2 日，客户 4027 为发票进行部分付款 297.00。 付款符合现金折扣的条件，因为 Fabrikam 提供部分付款的现金折扣，并且，部分付款是在现金折扣日期之前进行的。 因此，客户 4027 获得 3.00 现金折扣。 Arnie 使用付款日记帐记录客户 4027 的付款。 然后，Arnie 打开**结算交易记录**页，以便他可以标记要结算的发票。

| 标记     | 使用现金折扣 | 凭证   | 帐户 | 日期      | 到期日期  | 开票 | 交易币种借方金额 | 货币 | 要结算的金额 |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------------|----------|------------------|
| 已选择 | 标准            | FTI-10020 | 4027    | 6/25/2015 | 7/25/2015 | 10020   | 1,000.00                             | 美元      | 297.00           |

折扣信息显示在**结算未结交易记录**页的底部。 如果您不将**要结算的金额**值更改为 297.00，则显示的**现金折扣金额**值将不同。 但是，在过帐付款时，将提取 3.00 的现金折扣，因为结算会自动调整 **要结算的金额** 值。

|                              |           |
|------------------------------|-----------|
| 现金折扣日期           | 7/09/2015 |
| 现金折扣金额         | 10.00     |
| 使用现金折扣            | 标准    |
| 提取了现金折扣          | 0.00      |
| 要提取的现金折扣金额 | 3.00      |

Arnie 过帐此付款。 现在发票具有 700.00 余额。 以下交易记录对客户来说是可以查看的。

| 凭证    | 交易记录类型 | 日期      | 开票 | 交易币种借方金额 | 交易币种贷方金额 | 余额 | 货币 |
|------------|------------------|-----------|---------|--------------------------------------|---------------------------------------|---------|----------|
| FTI-10020  | 开票          | 6/25/2015 | 10020   | 1,000.00                             |                                       | 700.00  | 美元      |
| ARP-10020  |  付款         | 7/1/2015  |         |                                      | 297.00                                | 0.00    | 美元      |
| DISC-10020 |  现金折扣   | 7/1/2015  |         |                                      | 3.00                                  | 0.00    | 美元      |

## <a name="remaining-payment-after-the-cash-discount-date"></a>在现金折扣日期之后的其余付款
7 月 11 日 （折扣期间后），客户 4027 支付此发票的剩余金额。 在**结算交易记录**页上，**估计的现金折扣**字段中不显示折扣金额，并且**现金折扣金额**字段中的值为**0.00**。 在客户 4027 支付剩余的 700.00 时，将不会获得其他折扣。

| 标记     | 使用现金折扣 | 凭证   | 帐户 | 日期      | 到期日期  | 开票 | 交易币种借方金额 | 货币 | 要结算的金额 |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------------|----------|------------------|
| 已选择 | 标准            | FTI-10020 | 4027    | 6/25/2015 | 7/25/2015 | 10020   | 700.00                               | 美元      | 700.00           |

折扣信息显示在**结算未结交易记录**页的底部。

|                              |           |
|------------------------------|-----------|
| 现金折扣日期           | 7/09/2015 |
| 现金折扣金额         | 0.00      |
| 使用现金折扣            | 标准    |
| 提取了现金折扣          | 3.00      |
| 要提取的现金折扣金额 | 0.00      |

如果 Arnie 将**使用现金折扣**字段中的值更改为**始终**，则**计算部分付款的现金折扣**设置将被覆盖，现金折扣将被获取。 付款金额更改为 693.00，并且，现金折扣为剩余 7.00。

| 标记     | 使用现金折扣 | 凭证   | 帐户 | 日期      | 到期日期  | 开票 | 交易币种借方金额 | 交易币种贷方金额 | 货币 | 要结算的金额 |
|----------|-------------------|-----------|---------|-----------|-----------|---------|--------------------------------------|---------------------------------------|----------|------------------|
| 已选择 | 始终            | FTI-10020 | 4027    | 6/25/2015 | 7/25/2015 | 10020   | 700.00                               |                                       | 美元      | 693.00           |

折扣信息显示在**结算未结交易记录**页的底部。

|                              |           |
|------------------------------|-----------|
| 现金折扣日期           | 7/09/2015 |
| 现金折扣金额         | 7.00      |
| 使用现金折扣            | 始终    |
| 提取了现金折扣          | 3.00      |
| 要提取的现金折扣金额 | 7.00      |

Arnie 将**使用现金折扣**字段中的值改回**正常**，因为他不让该客户拿走 7.00 的剩余现金折扣。 然后，Arnie 过帐付款。 在 Arnie 打开**客户交易记录**页时，他将看到发票的余额为 0.00。 他还将具有两个付款。 一个付款针对 297.00 和 3.00 现金折扣；并且，另一个付款针对 700.00。

| 凭证    | 交易记录类型 | 日期      | 开票 | 交易币种借方金额 | 交易币种贷方金额 | 余额 | 货币 |
|------------|------------------|-----------|---------|--------------------------------------|---------------------------------------|---------|----------|
| FTI-10020  | 开票          | 6/25/2015 | 10020   | 1,000.00                             |                                       | 0.00    | 美元      |
| ARP-10020  |                  | 7/1/2015  |         |                                      | 297.00                                | 0.00    | 美元      |
| DISC-10020 |                  | 7/1/2015  |         |                                      | 3.00                                  | 0.00    | 美元      |
| ARP-10021  |                  | 7/11/2015 |         |                                      | 700.00                                | 0.00    | 美元      |
