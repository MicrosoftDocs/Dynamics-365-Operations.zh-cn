# <a name="set-up-workflows-for-expense"></a>设置费用的工作流

[!include[banner](../includes/banner.md)]您可以设置用于查看和审核出差和支出单据的工作流流程。 工作流可以定义的单据包括费用报表、出差申请和预付现金请求。

“工作流”代表业务流程。 它定义单据如何在系统中流动并指示必须由谁完成任务或审核单据。 在组织内使用工作流系统具有若干优点：

-   “**一致的流程**”— 您可定义特定单据（如采购申请和支出报表）的审核流程。 使用工作流系统有助于确保以一致且有效的方式来处理和审核单据。

-   “流程可见性”— 您可跟踪特定工作流实例的状态、历史记录和绩效指标。 这帮助您确定是否需要对该工作流做出更改，以提高效率。

-   “集中的工作列表”— 用户可以查看集中化的工作列表，以查看分配给他们的工作流任务和审核任务。 

**您可以创建工作流类型**

下表列出了您可以在**费用**过程中创建的工作流类型。

| **类型**                           | **使用此类型可以**                                                 |     
|------------------------------------|----------------------------------------------------------------------|
| **支出报表**                 | 为支出报表创建审核工作流。                       |      
| **支出报表自动过帐**    | 为支出报表创建自动过账工作流。              |     
| **支出行项**              | 为支出报表行项创建审核工作流。         |     
| **支出行项自动过帐** | 为支出报表行项创建自动过账工作流。|
| **出差申请**             | 为出差申请创建审核工作流。                   |    
| **预付现金请求**           | 创建预付现金请求的审核工作流。                 |     
| **增值税退税**               | 创建增值税 (VAT) 退税审核工作流。 |
