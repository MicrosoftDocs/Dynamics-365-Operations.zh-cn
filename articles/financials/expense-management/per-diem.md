# <a name="per-diem-rules"></a>出差津贴规则

[!INCLUDE [banner](../includes/banner.md)]

*出差津贴*是支付给出差员工的津贴。 在费用管理中，可为各种出差情况创建出差津贴规则。 出差津贴比率可以基于出差时间和/或出差地点。 在创建某一出差津贴规则时，您可以指定将预扣的出差津贴费用百分比（如果工作人员获得了免费提供的餐饮或服务）。 您还可以设置某一出差津贴比率可以应用于员工差旅的最低和最高小时数。

出差津贴总额的计算基于您可以在**出差津贴**页面上设置的开始日期和结束日期。
