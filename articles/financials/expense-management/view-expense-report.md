# <a name="view-an-expense-report"></a><span data-ttu-id="1b8ed-103">查看支出报表</span><span class="sxs-lookup"><span data-stu-id="1b8ed-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1b8ed-104">在总帐或采购中运行查询以查看交易记录时，查询可能会返回过多信息。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="1b8ed-105">因此，可能需要缩小搜索范围。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="1b8ed-106">若要获取所需的信息，您可以深化到交易记录以查看它将出现的支出报表。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="1b8ed-107">对于支出报告，您可以在已存储的出差费用管理中从某一交易记录深化到原始报表。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="1b8ed-108">除了支出总额，还可以查看逐项交易、适用于增值税 (VAT) 回收的交易记录，以及附加到支出的收货。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="1b8ed-109">若要向下钻取到交易记录所在的原始支出报表，请选择交易记录，然后选择**原始文档**。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="1b8ed-110">在查看各个交易记录时，您还可以查看任何附加到交易记录的附加数据。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="1b8ed-111">例如，对于列出的酒店支出明细，您可以查看支出的详细列表，如 Internet 访问权限、电话和房间服务。</span><span class="sxs-lookup"><span data-stu-id="1b8ed-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
