# <a name="configure-base-entities"></a>配置基本实体 

[!include[banner](../includes/banner.md)]


成本核算是 Dynamics 365 for Finance and Operations Enterprise Edition 中的一个独立模块，要求正确配置币种、汇率和会计日历实体。

-  货币
-  汇率
-  会计日历  

在您使用成本核算模块前，需要为您的公司定义币种、汇率和会计日历。 

若要了解有关这些实体的详细信息，请在网络培训课程[模块 2 币种和汇率设置](https://mbspartner.microsoft.com/AX/CourseModules/1215)（需要 CustomerSource 帐户）。
