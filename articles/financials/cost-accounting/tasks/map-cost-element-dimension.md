# <a name="map-a-cost-element-dimension"></a><span data-ttu-id="08e7b-103">映射成本元素维度</span><span class="sxs-lookup"><span data-stu-id="08e7b-103">Map a cost element dimension</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="08e7b-104">成本总监可使用此过程将成本元素维度映射到 MXMF 法人中的成本元素维度。</span><span class="sxs-lookup"><span data-stu-id="08e7b-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="08e7b-105">此录制使用 USP2 演示数据公司。</span><span class="sxs-lookup"><span data-stu-id="08e7b-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="08e7b-106">转到“成本核算”>“维度”>“成本要素维度”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="08e7b-107">在列表中，找到并选择所需记录。</span><span class="sxs-lookup"><span data-stu-id="08e7b-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="08e7b-108">在此示例中，选择“成本元素”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="08e7b-109">单击“维度映射”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="08e7b-110">单击“从该维度配置映射”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="08e7b-111">单击“新建”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-111">Click New.</span></span>
6. <span data-ttu-id="08e7b-112">在“截止维度”字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="08e7b-112">In the To dimension field, enter or select a value.</span></span>
    * <span data-ttu-id="08e7b-113">在此示例中，选择“MXMF 成本元素”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-113">For this example, select MXMF Cost elements.</span></span>  
7. <span data-ttu-id="08e7b-114">单击“新建”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-114">Click New.</span></span>
8. <span data-ttu-id="08e7b-115">在列表中，标记所选的行。</span><span class="sxs-lookup"><span data-stu-id="08e7b-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="08e7b-116">在“起始维度成员”字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="08e7b-116">In the From dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="08e7b-117">对于此示例，选择维度成员“606400 电话和传真费用”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-117">For this example, select dimension member 606400 Telephone & Fax Expense.</span></span>  
10. <span data-ttu-id="08e7b-118">在“截止维度成员”字段中，输入或选择一个值。</span><span class="sxs-lookup"><span data-stu-id="08e7b-118">In the To dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="08e7b-119">对于此示例，选择维度成员“6001004 电话”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-119">For this example, select dimension member 6001004 Telefono.</span></span>  
11. <span data-ttu-id="08e7b-120">单击“保存”。</span><span class="sxs-lookup"><span data-stu-id="08e7b-120">Click Save.</span></span>
