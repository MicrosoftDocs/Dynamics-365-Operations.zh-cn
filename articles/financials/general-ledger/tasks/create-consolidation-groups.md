# <a name="create-consolidation-groups-and-additional-consolidation-accounts"></a>创建合并组和其他合并帐户

[!include [task guide banner](../../includes/task-guide-banner.md)]

该过程显示如何创建合并帐户组，然后添加帐户到组中。 此过程使用了演示数据公司 USMF。


## <a name="create-a-consolidation-account-group"></a>创建合并帐户组
1. 转到“总帐”>“会计科目表”>“帐户”>“合并帐户组”。
2. 单击“新建”。
3. 在“合并帐户组”字段中，输入合并帐户组的唯一标识符。
4. 在“名称”字段中，键入一个值。

## <a name="add-accounts-to-consolidation-account-group"></a>添加帐户到合并帐户组中
1. 关闭该页面。
2. 转到“总帐”>“会计科目表”>“帐户”>“其他合并帐户”。
3. 单击“新建”。
4. 在“主科目”字段中，单击下拉按钮以打开查找。
5. 在列表中，单击要映射的主科目。
6. 在“合并帐户组”字段中，单击下拉按钮以打开查找。
7. 在列表中，单击相应的合并帐户组。
8. 在“合并帐户”字段中，键入一个值。
9. 在“合并帐户名称”字段中，键入一个值。
