2. <span data-ttu-id="100a1-106">选择要为其定义筛选条件的字段。</span><span class="sxs-lookup"><span data-stu-id="100a1-106">Select the field for which you want to define a filter criteria.</span></span>

    * <span data-ttu-id="100a1-108">您可以对单个值或范围进行筛选。</span><span class="sxs-lookup"><span data-stu-id="100a1-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="100a1-109">在定义范围时，确保使用正确的语法。</span><span class="sxs-lookup"><span data-stu-id="100a1-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="100a1-110">值应该使用双句点 (..) 分隔。</span><span class="sxs-lookup"><span data-stu-id="100a1-110">The values should be separated by a double period (..).</span></span>
