[!include[banner](../includes/banner.md)]

## <a name="voucher-transactions"></a><span data-ttu-id="95ca2-107">凭证交易记录</span><span class="sxs-lookup"><span data-stu-id="95ca2-107">Voucher transactions</span></span>
<span data-ttu-id="95ca2-108">**凭证交易记录**是一个查询页，您可以从中选择各个表和字段以指定您要搜索的余额或交易记录的条件。</span><span class="sxs-lookup"><span data-stu-id="95ca2-108">**Voucher transactions** is an inquiry page where you can select from various tables and fields to specify criteria for the balance or transaction that you're searching for.</span></span> <span data-ttu-id="95ca2-109">例如，您可以查看特定日期或帐户的所有交易记录，或特定过帐层中的**运营**类型的所有交易记录。</span><span class="sxs-lookup"><span data-stu-id="95ca2-109">For example, you can view all transactions for a specific date or account, or all transactions of the **Operating** type that are in a specific posting layer.</span></span> <span data-ttu-id="95ca2-110">默认情况下，此页显示日记帐编号、凭证、日期和主科目，不过，您可以添加其他表、字段和条件来缩小搜索范围。</span><span class="sxs-lookup"><span data-stu-id="95ca2-110">By default, the page shows the journal number, voucher, date, and main account, but you can add additional tables, fields, and criteria to narrow down your search.</span></span>

## <a name="audit-trail"></a><span data-ttu-id="95ca2-111">审计线索</span><span class="sxs-lookup"><span data-stu-id="95ca2-111">Audit trail</span></span>
<span data-ttu-id="95ca2-112">**审计线索**是显示交易记录类型、描述、交易记录创建人和创建日期的查询页。</span><span class="sxs-lookup"><span data-stu-id="95ca2-112">**Audit trail** is an inquiry page that shows the types of transactions, descriptions, who the transactions were created by, and when they were created.</span></span> <span data-ttu-id="95ca2-113">从**审计线索**查询页，可以查看凭证交易记录。</span><span class="sxs-lookup"><span data-stu-id="95ca2-113">From the **Audit trail** inquiry page, you can view the voucher transactions.</span></span>

## <a name="financial-reports"></a><span data-ttu-id="95ca2-114">财务报表</span><span class="sxs-lookup"><span data-stu-id="95ca2-114">Financial reports</span></span>
<span data-ttu-id="95ca2-115">您还可以通过运行财务报表探索和分析总帐交易记录。</span><span class="sxs-lookup"><span data-stu-id="95ca2-115">You can also explore and analyze general ledger transactions by running financial reports.</span></span> <span data-ttu-id="95ca2-116">因为财务报表的设计可以基于帐户、维度、科目类别或三者组合，您可以通过不同方式深化来查看交易记录。</span><span class="sxs-lookup"><span data-stu-id="95ca2-116">Because the design of financial reports can be based on accounts, dimensions, account categories, or a combination of the three, you can view the transactions by drilling down in various ways.</span></span> <span data-ttu-id="95ca2-117">如果您需要总帐交易记录的更多信息，还可以包括多个交易记录属性作为报表设计的一部分。</span><span class="sxs-lookup"><span data-stu-id="95ca2-117">If you require more information for general ledger transactions, you can also include multiple transaction properties as part of the report design.</span></span> <span data-ttu-id="95ca2-118">此外，如果您要查看构成总帐余额的交易记录，可以深化到核算交易记录，就像您可以从列 **试算平衡表**列表页执行的操作。</span><span class="sxs-lookup"><span data-stu-id="95ca2-118">Additionally, if you want to see the transactions that make up a general ledger balance, you can drill down to account transactions, just as you can from the **Trial balance** list page.</span></span>

-   <span data-ttu-id="95ca2-122">**分类帐交易记录列表** – 此报表可以在交易记录、核算和帐户的申报币种中显示交易记录。</span><span class="sxs-lookup"><span data-stu-id="95ca2-122">**Ledger transaction list** – This report shows transactions in transaction, accounting, and reporting currencies for an account.</span></span>

-   <span data-ttu-id="95ca2-125">**已过帐的日记帐交易记录** – 此报表显示已过帐到按凭证分组的日记帐的交易记录。</span><span class="sxs-lookup"><span data-stu-id="95ca2-125">**Posted transactions by journal** – This report shows the transactions that have been posted to a journal, grouped by voucher.</span></span>
