# <a name="create-a-bank-facility-agreement-for-the-letter-of-guarantee"></a>创建保函的银行信贷协议

[!include [task guide banner](../../includes/task-guide-banner.md)]

此任务可以创建一个银行融资协议，以便处理保函。 本任务使用 USMF 公司进行演示。 


## <a name="create-bank-facility-agreement"></a>创建银行信贷协议
1. 转到“现金和银行管理”>“保函”>“银行融资协议”。
2. 单击“新建”。
3. 在“协议编号”字段中，输入交易的银行协议编号。
4. 在“银行帐户”字段中，选择开放保函适用的银行帐号。 
5. 在列表中，单击所选行中的链接。
6. 在“开始日期”字段中，输入日期和时间。
7. 在“结束日期”字段中，输入日期和时间。
8. 切换“概况”部分的扩展。
9. 单击“添加行”。
10. 在“融资类型”字段中，单击下拉键打开查询。
11. 在列表中，找到并选择所需记录。
12. 在列表中，单击所选行中的链接。
13. 在“限值”字段中，输入与银行协商的金额。
14. 单击“保存”。
15. 切换“保函”部分的展开项。
16. 在“计算方法”字段中，选择一个选项。
    * 根据需要输入“现金保证金”、“发货佣金”、“展开佣金”、“增值佣金”或“降值佣金”的计算方法和百分比详情。   
17. 单击“保存”。

## <a name="extend-bank-facility-agreement"></a>展开银行融资协议
1. 单击“扩展”打开垂直对话框。
2. 在“新协议数目”字段中，键入一个值。
3. 在“结束日期”字段中，输入日期和时间。
4. 单击“扩展”。
5. 单击“保存”。
6. 关闭该页面。
