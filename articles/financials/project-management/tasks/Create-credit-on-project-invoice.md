# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="01bd8-103">创建项目发票上的贷方通知单</span><span class="sxs-lookup"><span data-stu-id="01bd8-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="01bd8-104">转到项目管理和记账>项目>全部项目。</span><span class="sxs-lookup"><span data-stu-id="01bd8-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="01bd8-105">在列表中，找到并选择所需记录。</span><span class="sxs-lookup"><span data-stu-id="01bd8-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="01bd8-106">选择用于贷方通知单的项目 ID。</span><span class="sxs-lookup"><span data-stu-id="01bd8-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="01bd8-107">在列表中，单击所选行中的链接。</span><span class="sxs-lookup"><span data-stu-id="01bd8-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="01bd8-108">在“操作窗格”上单击“管理”。</span><span class="sxs-lookup"><span data-stu-id="01bd8-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="01bd8-109">单击“发票日记帐”。</span><span class="sxs-lookup"><span data-stu-id="01bd8-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="01bd8-110">为贷方通知单选择交易记录</span><span class="sxs-lookup"><span data-stu-id="01bd8-110">Select the transaction for credit note.</span></span> <span data-ttu-id="01bd8-111">在列表中，标记所选的行。</span><span class="sxs-lookup"><span data-stu-id="01bd8-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="01bd8-112">单击“功能”。</span><span class="sxs-lookup"><span data-stu-id="01bd8-112">Click Functions.</span></span> 
8. <span data-ttu-id="01bd8-113">单击“为贷方通知单选择”。</span><span class="sxs-lookup"><span data-stu-id="01bd8-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="01bd8-114">通过选中复选框来选择用于贷方通知单的交易记录。</span><span class="sxs-lookup"><span data-stu-id="01bd8-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="01bd8-115">单击“确定”。</span><span class="sxs-lookup"><span data-stu-id="01bd8-115">Click OK.</span></span> 

<span data-ttu-id="01bd8-116">在此步骤之后，贷方通知单将可用于发票方案。</span><span class="sxs-lookup"><span data-stu-id="01bd8-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
