# <a name="report-designer-interface"></a>报表设计器界面

[!include [banner](../includes/banner.md)]

本文介绍如何在报表设计器内导航，以及如何使用各种选项以满足您的特定要求。

## <a name="report-designer-menu-commands"></a>报表设计器菜单命令

以下表格描述了您在设计财务报表时可使用的菜单命令和选项。 一些菜单命令和选项仅在特定情况下才可用。 例如，用于提升和降低报告单位级别的命令只有在修改报告树定义时才可用。

### <a name="file-menu"></a>文件菜单

**文件**菜单适用于所有用户并包含以下命令。

| 命令                           | 说明 |
|-----------------------------------|-------------|
| 新建                               | 创建新的报表定义、行定义、列定义、报告树定义、报表组定义或文件夹。 另外还提供其他选项，这取决于您的用户角色。 |
| 未完成                              | 打开现有行定义、列定义、报告树定义或报表定义。 |
| 关闭                             | 关闭当前构建基块。 |
| 全部关闭                         | 关闭所有构建基块。 |
| 保存                              | 保存当前的行定义、列定义、报告树定义或报表定义。 |
| 另存为                           | 用新名称保存当前的行定义、列定义、报告树定义或报表定义。 |
| 属性                        | 打开**属性**对话框，您可以在此对话框中更改报表的名称和描述。 |
| 生成                          | 生成当前报表。 可从报表定义中获取此命令。 |
| 查看报表                       | 在 Finance and Operations 中打开生成的最新版本报表。 如果您已生成至少一个报表，则报表定义中将提供此命令。 |
| 最新报表定义         | 显示最近创建或修改的报表的列表。 您可以随后选择该列表中的报表。 |
| 最新行定义            | 显示最近创建或修改的行定义的列表。 您可以随后选择该列表中的行定义。 |
| 最新列定义         | 显示最近创建或修改的列定义的列表。 您可以随后选择该列表中的列定义。 |
| 最新报告树定义 | 显示最近创建或修改的报告树定义的列表。 您可以随后选择该列表中的报告树定义。 |
| 退出                              | 退出报表设计器。 |

### <a name="edit-menu"></a>“编辑”菜单

**编辑**菜单适用于具有**设计人员**或**管理员**角色的用户。 此菜单包括以下命令。

| 命令                                | 说明 |
|----------------------------------------|-------------|
| 撤消                                   | 撤消上一操作。 |
| 重做                                   | 反向执行上次撤消的操作。 |
| 剪切                                    | 删除选定文本并将其复制到剪贴板。 |
| 复制                                   | 将选定文本复制到剪贴板。 |
| 粘贴                                  | 插入剪贴板中最近剪切或复制的文本。 |
| 清除                                  | 删除选定构建基块单元格的内容。 |
| 查找                                   | 打开**查找和替换**对话框，您可以在此对话框中搜索视图窗格中的文本。 |
| 替换                                | 打开**查找和替换**对话框，您可以在此对话框中搜索和替换视图窗格中的文本。 |
| 插入维度中的行            | 打开**从维度插入行**对话框，您可以在此对话框中选择要包含在行定义中的维度值。 可从行定义中获取此命令。 |
| 对行重新编号                          | 对所有数字行代码重新进行编号。 可从行定义中获取此命令。 |
| 行链接                              | 打开**行链接**对话框，您可以在此对话框中指定行定义和报告树定义中数据链接的源。 可从行定义中获取此命令。 |
| 舍入调整                    | 打开**舍入调整**对话框，您可以在此对话框中指定舍入的参数。 可从行定义中获取此命令。 |
| 管理维度集                  | 打开**维度集**对话框，您可以在此对话框中创建和修改维度集。 可从行定义或报告树定义获取此命令。 |
| 插入行                             | 在行定义中插入空白行或在列定义中插入空白标题行。 可从行定义或列定义中获取此命令。 |
| 删除行                             | 从行定义中删除选定的行或从列定义中删除选定的标题行。 可从行定义或列定义中获取此命令。 |
| 插入列                          | 在列定义中插入空白列。 可从列定义中获取此命令。 |
| 删除列                          | 从列定义中删除选定的列。 可从列定义中获取此命令。 |
| 插入维度中的报告单位 | 打开**从维度插入报告单元**对话框，您可以在此对话框中选择要包含在报告树定义中的维度值。 可从报告树定义中获取此命令。 |
| 导入维度集层次结构         | 打开**维度集层次结构**对话框，您可以在此对话框中导入财务数据中的维度集层次结构。 ...\\财务维度\\基于维度的系统的报告树定义中提供此命令。 |
| 插入报告单位                  | 将空行插入报告树定义中。 可从报告树定义中获取此命令。 |
| 删除报告单位                  | 从报告树定义中删除选定的报告单位行。 可从报告树定义中获取此命令。 |

### <a name="view-menu"></a>“视图”菜单

**视图**菜单适用于所有用户并包含以下命令。

| 命令         | 说明                                                            |
|-----------------|------------------------------------------------------------------------|
| 导航窗格 | 显示或隐藏导航窗格。                                      |
| 工具栏        | 选择要显示的工具栏。                                  |
| 状态栏      | 在**报表设计器**窗口中显示或隐藏状态信息。 |
| “欢迎使用”页面    | 打开**欢迎使用**页面。                                             |

### <a name="format-menu"></a>“格式”菜单

**格式**菜单适用于具有**设计人员**或**管理员**角色的用户。 此菜单包括以下命令。

| 命令               | 说明 |
|-----------------------|-------------|
| 样式和格式 | 打开**样式和格式设置**对话框，您可以在此对话框中创建和修改行定义和列定义中文本的样式。 可从行定义或列定义中获取此命令。 |
| 列宽          | 打开**列宽度**对话框，您可以在此对话框中设置所选列的宽度。 可从行定义、列定义或报告树定义获取此命令。 |
| 隐藏                  | 隐藏选定的列。 可从行定义、列定义或报告树定义获取此命令。 |
| 取消隐藏                | 使选定列之间的隐藏列可见。 可从行定义、列定义或报告树定义获取此命令。 |

### <a name="company-menu"></a>“公司”菜单

**公司**菜单适用于具有**设计人员**或**管理员**角色的用户。 此菜单包括以下命令。

| 命令               | 说明                                                                                                            |
|-----------------------|------------------------------------------------------------------------------------------------------------------------|
| 公司             | 打开**公司**对话框，您可以在此对话框中创建和修改公司。                                          |
| 构建基块组 | 打开**构建基块组**对话框，您可以在此对话框中创建、修改、导入和导出构建基块组。 |

### <a name="go-menu"></a>“开始”菜单

**开始**菜单对所有用户均可用，它包括以下命令。

> [!NOTE]
> 除非显示导航窗格，否则这些命令不会显示。

| 命令                   | 说明                                                                        |
|----------------------------|------------------------------------------------------------------------------------|
| 报表定义         | 在导航窗格中显示报表定义。                                    |
| 行定义            | 在导航窗格中显示行定义。                                       |
| 列定义         | 在导航窗格中显示列定义。                                    |
| 报表树定义 | 在导航窗格中显示报告树定义。                            |
| 安全性                   | 在导航窗格中显示用户、组和公司的安全信息。 |

### <a name="tools-menu"></a>“工具”菜单

**工具**菜单适用于所有用户但某些命令具有有限的可用性。 此菜单包括以下命令。

| 命令                       | 说明 |
|-------------------------------|-------------|
| 保护                       | 对当前构建基块应用密码。 此命令适用于具有**设计人员**或**管理员**角色的用户。 |
| 报表队列状态           | 打开**报表队列状态**对话框，您可以在此对话框中查看最近生成的所有报表以及每个报表的详细信息。 |
| 源系统信息     | 显示您的 Microsoft Dynamics ERP 系统的设置。 此命令适用于具有**设计人员**或**管理员**角色的用户。 |
| 已签出的项目             | 显示当前打开的行定义、列定义、报告树定义和报表定义。 此命令适用于具有**设计人员**或**管理员**角色的用户。 |
| 刷新缓存的财务数据 | 更新财务维度列中的数据。 |
| 选项                       | 打开**选项**对话框，您可以在此对话框中修改 Report Designer 的用户首选项。 |

### <a name="window-menu"></a>“窗口”菜单

**窗口**菜单适用于所有用户并包含以下命令。

| 命令              | 说明 |
|----------------------|-------------|
| 横向平铺    | 以左右相邻的方式显示所有打开的窗口。 |
| 纵向平铺      | 以上下相邻的方式显示所有打开的窗口。 |
| 层叠              | 分层显示所有打开的窗口，以便每个窗口的标题栏可见。 |
| 水平冻结    | 冻结选定的行，以便在滚动时该行继续显示在窗口中。 此命令适用于具有**设计人员**或**管理员**角色的用户。 |
| 垂直冻结      | 冻结选定的列，以便在滚动时该列继续显示在窗口中。 此命令适用于具有**设计人员**或**管理员**角色的用户。 |
| 已打开窗口的列表 | 显示打开的窗口列表。 选择某个窗口以将其显示在最前。 |

### <a name="help-menu"></a>“帮助”菜单

**帮助**菜单适用于所有用户并包含以下命令。

| 命令 | 说明                                                              |
